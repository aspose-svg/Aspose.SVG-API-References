## Color.ToNaturalColorString method

Renvoie une couleur spécifiée par les couleurs naturelles (NCol) en utilisant une lettre de couleur avec un nombre pour spécifier la distance (en pourcentage) de la couleur.

```csharp
public string ToNaturalColorString(int digits)
```

| Paramètre | Taper | La description |
| --- | --- | --- |
| digits | Int32 | Définit la précision d'arrondi des composants de couleur. |
