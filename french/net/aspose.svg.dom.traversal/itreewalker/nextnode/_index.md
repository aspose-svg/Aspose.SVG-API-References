## ITreeWalker.NextNode method

Déplace le TreeWalker vers le nœud visible suivant dans l'ordre document par rapport au nœud actuel et renvoie le nouveau nœud. Si le nœud actuel n'a pas de nœud suivant, ou si la recherche de nextNode tente de remonter à partir du nœud root du TreeWalker, renvoie null et conserve le nœud actuel.

```csharp
public Node NextNode()
```

### Return_Value

Le nouveau nœud, ou null si le nœud actuel n'a pas de nœud suivant dans la vue logique du TreeWalker.
