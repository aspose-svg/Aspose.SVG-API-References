## ICSSStyleDeclaration indexer

Utilisé pour récupérer les propriétés qui ont été explicitement définies dans ce bloc de déclaration. L'ordre des propriétés récupérées à l'aide de cette méthode ne doit pas nécessairement correspondre à l'ordre dans lequel elles ont été définies. Cette méthode peut être utilisée pour parcourir toutes les propriétés de ce bloc de déclaration.

```csharp
public string this[int index] { get; }
```

| Paramètre | La description |
| --- | --- |
| index | L'indice immobilier |

### Return_Value

La valeur CSS.
