## CSSPrimitiveValue.GetRectValue method

Cette méthode est utilisée pour obtenir la valeur Rect. Si cette valeur CSS ne contient pas de valeur rect, une exception DOMException est levée. La modification de la propriété de style correspondante peut être réalisée à l'aide de l'interface Rect.

```csharp
public abstract Rect GetRectValue()
```

### Return_Value

Renvoie la valeur Rect
