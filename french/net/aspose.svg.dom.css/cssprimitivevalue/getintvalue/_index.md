## CSSPrimitiveValue.GetIntValue method

Cette méthode est utilisée pour obtenir une valeur int dans une unité spécifiée. Si cette valeur CSS ne contient pas de valeur int ou ne peut pas être convertie dans l'unité spécifiée, une DOMException est levée.
