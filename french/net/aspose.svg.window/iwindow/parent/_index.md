## IWindow.Parent property

L'attribut IDL parent sur l'objet Window d'un Document dans un contexte de navigation b doit retourner l'objet WindowProxy du contexte de navigation parent, s'il y en a un (c'est-à-dire si b est un contexte de navigation enfant), ou l'objet WindowProxy du contexte de navigation contexte b lui-même, sinon (c'est-à-dire s'il s'agit d'un contexte de navigation de niveau supérieur ou d'un contexte de navigation imbriqué détaché).

```csharp
public IWindow Parent { get; }
```

### Valeur de la propriété

Le parent.
