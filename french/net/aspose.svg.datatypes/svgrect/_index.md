## SVGRect class

Représente la géométrie rectangulaire. Les rectangles sont définis comme étant constitués d'une paire de coordonnées (x, y) identifiant une valeur X minimale, une valeur Y minimale et une largeur et une hauteur, qui sont généralement contraintes d'être non négatives.

```csharp
public class SVGRect : SVGValueType
```
