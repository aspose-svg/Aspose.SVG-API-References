## SVGAngle.Value property

La valeur de l'angle sous forme de valeur à virgule flottante, en degrés. La définition de cet attribut entraînera la mise à jour automatique de valueInSpecifiedUnits et valueAsString pour refléter ce paramètre.

```csharp
public float Value { get; set; }
```

### Valeur de la propriété

La valeur.
