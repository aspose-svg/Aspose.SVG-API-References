## SVGLength.ValueInSpecifiedUnits property

La valeur sous forme de valeur à virgule flottante, dans les unités exprimées par unitType. La définition de cet attribut entraînera la mise à jour automatique de value et valueAsString pour refléter ce paramètre.

```csharp
public float ValueInSpecifiedUnits { get; set; }
```

### Valeur de la propriété

La valeur dans les unités spécifiées.
