## Node.ReplaceChild method

Remplace le nœud enfant oldChild par newChild dans la liste des enfants et renvoie le nœud oldChild. Si newChild est un objet DocumentFragment, oldChild est remplacé par tous les enfants DocumentFragment, qui sont insérés dans le même ordre. Si le newChild est déjà dans l'arborescence, il est d'abord supprimé.

```csharp
public Node ReplaceChild(Node node, Node child)
```

| Paramètre | Taper | La description |
| --- | --- | --- |
| node | Node | Le nouveau nœud. |
| child | Node | Le vieil enfant. |
