## Entity.InputEncoding property

Un attribut spécifiant l'encodage utilisé pour cette entité au moment de l'analyse, lorsqu'il s'agit d'une entité externe analysée. Ceci est nul s'il s'agit d'une entité du sous-ensemble interne ou s'il n'est pas connu.

```csharp
public string InputEncoding { get; }
```

### Valeur de la propriété

L'encodage d'entrée.
