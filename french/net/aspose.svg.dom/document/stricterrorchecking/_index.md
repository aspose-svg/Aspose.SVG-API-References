## Document.StrictErrorChecking property

Un attribut spécifiant si la vérification des erreurs est appliquée ou non. Lorsqu'il est défini sur false, l'implémentation est libre de ne pas tester tous les cas d'erreur possibles normalement définis sur les opérations DOM, et de ne pas déclencher d'exception DOMException sur les opérations DOM ou de signaler des erreurs lors de l'utilisation de Document.normalizeDocument(). En cas d'erreur, le comportement est indéfini. Cet attribut est vrai par défaut.

```csharp
public bool StrictErrorChecking { get; set; }
```

### Valeur de la propriété

`vrai` si vérification stricte des erreurs ; sinon,`FAUX` .
