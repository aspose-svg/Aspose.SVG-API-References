## Document.XmlVersion property

Un attribut spécifiant, dans le cadre de la déclaration XML, le numéro de version de ce document. S'il n'y a pas de déclaration et si ce document prend en charge la fonctionnalité "XML", la valeur est "1.0". Si ce document ne prend pas en charge la fonctionnalité "XML", la valeur est toujours nulle.

```csharp
public string XmlVersion { get; set; }
```

### Valeur de la propriété

La version XML.
