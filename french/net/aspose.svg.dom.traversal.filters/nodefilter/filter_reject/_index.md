## NodeFilter.FILTER_REJECT field

Rejeter le nœud. Les méthodes de navigation définies pour NodeIterator ou TreeWalker ne renverront pas ce nœud. Pour TreeWalker, les enfants de ce nœud seront également rejetés. Les NodeIterators traitent cela comme un synonyme de FILTER_SKIP.

```csharp
public const short FILTER_REJECT;
```
