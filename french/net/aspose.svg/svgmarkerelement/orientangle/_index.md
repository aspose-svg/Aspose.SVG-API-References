## SVGMarkerElement.OrientAngle property

Correspond à l'attribut 'orient' sur l'élément 'marker' donné. Si markerUnits est SVG_MARKER_ORIENT_ANGLE, la valeur d'angle pour l'attribut 'orient' ; sinon, il sera mis à zéro.

```csharp
public SVGAnimatedAngle OrientAngle { get; }
```

### Valeur de la propriété

L'attribut 'orient' de l'élément 'marker' donné.
