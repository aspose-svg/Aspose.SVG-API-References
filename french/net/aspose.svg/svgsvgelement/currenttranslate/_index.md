## SVGSVGElement.CurrentTranslate property

Sur un élément svg le plus à l'extérieur, le facteur de traduction correspondant qui prend en compte le "grossissement" de l'utilisateur. Lorsqu'il est accédé sur un élément 'svg' qui n'est pas un élément svg le plus à l'extérieur, le comportement de cet attribut n'est pas défini.

```csharp
public SVGPoint CurrentTranslate { get; }
```

### Valeur de la propriété

La traduction actuelle.
