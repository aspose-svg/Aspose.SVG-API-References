## SVGSVGElement.GetElementById method

Recherche dans ce fragment de document SVG (c'est-à-dire que la recherche est limitée à un sous-ensemble de l'arborescence du document) un élément dont l'identifiant est donné par elementId. Si un élément est trouvé, cet élément est renvoyé. Si aucun élément de ce type n'existe, renvoie null. Le comportement n'est pas défini si plus d'un élément a cet identifiant.

```csharp
public Element GetElementById(string elementId)
```

| Paramètre | Taper | La description |
| --- | --- | --- |
| elementId | String | La valeur d'identifiant unique pour un élément. |

### Return_Value

L'élément correspondant.
