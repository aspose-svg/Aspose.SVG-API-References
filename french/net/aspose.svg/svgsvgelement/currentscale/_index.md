## SVGSVGElement.CurrentScale property

Sur un élément svg le plus à l'extérieur, cet attribut indique le facteur d'échelle actuel par rapport à la vue initiale pour prendre en compte les opérations d'agrandissement et de panoramique de l'utilisateur, comme décrit dans Agrandissement et panoramique. Les attributs DOM currentScale et currentTranslate sont équivalents à la matrice 2x3 [abcdef] = [currentScale 0 0 currentScale currentTranslate.x currentTranslate.y]. Si le "grossissement" est activé (c'est-à-dire zoomAndPan="magnify"), l'effet est comme si une transformation supplémentaire était placée au niveau le plus externe sur le fragment de document SVG (c'est-à-dire à l'extérieur de l'élément svg le plus externe). un élément 'svg' qui n'est pas un élément svg le plus externe, le comportement de cet attribut n'est pas défini.

```csharp
public float CurrentScale { get; set; }
```

### Valeur de la propriété

L'échelle actuelle.
