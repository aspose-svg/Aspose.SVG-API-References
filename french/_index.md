## Aspose.SVG pour .NET

{{% alert color="primary" %}} 

![Aspose.SVG pour le logo du produit .NET](../home_1.png)


Aspose.SVG pour l'API .NET est une bibliothèque multiplateforme qui fournit un large éventail de fonctionnalités pour le traitement et le rendu des documents SVG. L'API maintient le modèle d'objet de document (DOM) et est entièrement compatible avec les spécifications SVG officielles, vous permettant d'avoir un contrôle total sur les nœuds SVG et leurs propriétés. Vous pouvez facilement modifier l'arborescence du document, ajouter et supprimer des nœuds, modifier les propriétés des nœuds comme décrit dans les spécifications officielles. En utilisant l'API Aspose.SVG, vous pouvez rapidement charger, enregistrer et convertir des SVG ainsi que lire et parcourir les éléments du fichier. L'API est indépendante de tout autre logiciel. Il vous permet de travailler avec des fichiers SVG sur toutes les plates-formes prenant en charge .NET (Windows, Linux et macOS).
