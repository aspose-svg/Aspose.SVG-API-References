## SVGSVGElement.CurrentScale property

Σε ένα εξώτατο στοιχείο svg, αυτό το χαρακτηριστικό υποδεικνύει τον τρέχοντα συντελεστή κλίμακας σε σχέση με την αρχική προβολή για να ληφθούν υπόψη οι λειτουργίες μεγέθυνσης και μετατόπισης χρήστη, όπως περιγράφεται στην ενότητα Μεγέθυνση και μετατόπιση. Τα χαρακτηριστικά DOM currentScale και currentTranslate είναι ισοδύναμα με τον πίνακα 2x3 [abcdef] = [currentScale 0 0 currentScale currentTranslate.x currentTranslate.y]. Εάν η "μεγέθυνση" είναι ενεργοποιημένη (δηλαδή, zoomAndPan="μεγέθυνση"), τότε το εφέ είναι σαν να τοποθετήθηκε ένας επιπλέον μετασχηματισμός στο εξώτατο επίπεδο στο τμήμα του εγγράφου SVG (δηλαδή, έξω από το εξώτατο στοιχείο svg). Κατά την πρόσβαση στο ένα στοιχείο 'svg' που δεν είναι ένα εξώτατο στοιχείο svg, δεν έχει οριστεί ποια συμπεριφορά έχει αυτό το χαρακτηριστικό.

```csharp
public float CurrentScale { get; set; }
```
