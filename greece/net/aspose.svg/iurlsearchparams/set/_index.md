## IUrlSearchParams.Set method

Ορίζει την τιμή του πρώτου ζεύγους ονόματος-τιμής που βρέθηκε στην καθορισμένη τιμή και αφαιρεί τις άλλες. Εάν δεν βρεθούν ζεύγη ονόματος-τιμής με το καθορισμένο όνομα, θα προστεθεί νέο στη λίστα.

```csharp
public void Set(string name, string value)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| name | String | Κλειδί που θα χρησιμοποιηθεί για την εύρεση ζευγών ονόματος-τιμής. |
| value | String | Τιμή που θα οριστεί. |
