## Sandbox enumeration

Ένα σύνολο σημαιών sandboxing είναι ένα σύνολο μηδέν ή περισσότερων από τις ακόλουθες σημαίες, οι οποίες χρησιμοποιούνται για τον περιορισμό των ικανοτήτων που ενδεχομένως δεν εμπιστεύονται τους πόρους.

```csharp
[Flags]
public enum Sandbox
```

### Αξίες

| Ονομα | αξία | Περιγραφή |
| --- | --- | --- |
| None | `0` | Δεν έχει οριστεί σημαία, κάθε χαρακτηριστικό sandbox είναι αποδεκτό |
| Navigation | `1` | Αυτή η σημαία αποτρέπει το περιεχόμενο από την πλοήγηση σε περιβάλλοντα περιήγησης εκτός από το ίδιο το περιβάλλον περιήγησης σε περιβάλλον δοκιμών (ή τα περιβάλλοντα περιήγησης που είναι πιο ένθετα μέσα σε αυτό), τα βοηθητικά περιβάλλοντα περιήγησης (τα οποία προστατεύονται από τη σημαία περιβάλλοντος περιήγησης με βοηθητικό περιβάλλον περιήγησης που ορίζεται στη συνέχεια) και το ανώτερο επίπεδο περιβάλλον περιήγησης (το οποίο προστατεύεται από τη σημαία περιβάλλοντος περιήγησης ανώτατου επιπέδου πλοήγησης σε πλαίσιο sandbox που ορίζεται παρακάτω). Εάν δεν έχει οριστεί η σημαία περιβάλλοντος περιήγησης της βοηθητικής πλοήγησης σε περιβάλλον δοκιμών, τότε σε ορισμένες περιπτώσεις οι περιορισμοί επιτρέπουν ωστόσο το άνοιγμα αναδυόμενων παραθύρων (νέα περιβάλλοντα περιήγησης ανώτατου επιπέδου). Αυτά τα περιβάλλοντα περιήγησης έχουν πάντα έναν επιτρεπόμενο πλοηγό sandbox, που ορίζεται όταν δημιουργείται το περιβάλλον περιήγησης, το οποίο επιτρέπει στο περιβάλλον περιήγησης που τα δημιούργησε να πλοηγηθεί πραγματικά σε αυτά. (Διαφορετικά, η σημαία περιβάλλοντος περιήγησης περιήγησης σε περιβάλλον δοκιμών θα εμπόδιζε την πλοήγησή τους ακόμα κι αν ήταν ανοιχτά. |
| AuxiliaryNavigation | `2` | Αυτή η σημαία εμποδίζει το περιεχόμενο να δημιουργήσει νέα βοηθητικά περιβάλλοντα περιήγησης, π.χ. χρησιμοποιώντας το χαρακτηριστικό target ή τη μέθοδο window.open(). |
| TopLevelNavigation | `4` | Αυτή η σημαία εμποδίζει το περιεχόμενο να περιηγηθεί στο περιβάλλον περιήγησής του ανώτατου επιπέδου και εμποδίζει το περιεχόμενο να κλείσει το περιβάλλον περιήγησής του ανώτατου επιπέδου. Όταν η σημαία περιβάλλοντος περιήγησης ανώτατου επιπέδου περιήγησης σε περιβάλλον δοκιμών δεν έχει οριστεί, το περιεχόμενο μπορεί να πλοηγηθεί στο πλαίσιο περιήγησής του στο ανώτατο επίπεδο, αλλά άλλα περιβάλλοντα περιήγησης εξακολουθούν να προστατεύονται από τη σημαία περιβάλλοντος περιήγησης περιήγησης σε περιβάλλον δοκιμών δοκιμών και πιθανώς τη βοηθητική σημαία περιήγησης περιήγησης σε περιβάλλον δοκιμών._x0._x0. |
| Plugins | `8` | Αυτή η σημαία εμποδίζει το περιεχόμενο να δημιουργήσει προσθήκες, είτε χρησιμοποιεί το στοιχείο ενσωμάτωσης, το στοιχείο αντικειμένου, το στοιχείο μικροεφαρμογής είτε μέσω πλοήγησης σε ένα ένθετο περιβάλλον περιήγησης, εκτός εάν αυτές οι προσθήκες μπορούν να ασφαλιστούν. |
| Origin | `10` | Αυτή η σημαία αναγκάζει το περιεχόμενο σε μια μοναδική προέλευση, αποτρέποντάς του έτσι την πρόσβαση σε άλλο περιεχόμενο από την ίδια προέλευση. |
| Forms | `20` | Αυτή η σημαία αποκλείει την υποβολή φόρμας. |
| PointerLock | `40` | Αυτή η σημαία απενεργοποιεί το Pointer Lock API. |
| Scripts | `80` | Αυτή η σημαία αποκλείει την εκτέλεση σεναρίου. |

| Images | `800` | Αυτή η σημαία απενεργοποιεί τη φόρτωση εικόνας. |
