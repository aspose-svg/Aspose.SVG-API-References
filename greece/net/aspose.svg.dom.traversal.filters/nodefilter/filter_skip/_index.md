## NodeFilter.FILTER_SKIP field

Παράλειψη αυτού του μεμονωμένου κόμβου. Οι μέθοδοι πλοήγησης που ορίζονται για NodeIterator ή TreeWalker δεν θα επιστρέψουν αυτόν τον κόμβο. Και για το NodeIterator και για TreeWalker, τα παιδιά αυτού του κόμβου θα εξακολουθούν να θεωρούνται .

```csharp
public const short FILTER_SKIP;
```
