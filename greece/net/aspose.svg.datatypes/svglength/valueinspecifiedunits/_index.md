## SVGLength.ValueInSpecifiedUnits property

Η τιμή ως τιμή κινητής υποδιαστολής, στις μονάδες που εκφράζεται με unitType. Η ρύθμιση αυτού του χαρακτηριστικού θα προκαλέσει την αυτόματη ενημέρωση της τιμής και του valueAsString ώστε να αντικατοπτρίζει αυτήν τη ρύθμιση.

```csharp
public float ValueInSpecifiedUnits { get; set; }
```

### Αξία περιουσίας

Η τιμή σε καθορισμένες μονάδες.
