## SVGAngle.ValueInSpecifiedUnits property

Η τιμή της γωνίας ως τιμή κινητής υποδιαστολής, στις μονάδες που εκφράζονται με unitType. Η ρύθμιση αυτού του χαρακτηριστικού θα προκαλέσει την αυτόματη ενημέρωση της τιμής και του valueAsString ώστε να αντικατοπτρίζει αυτήν τη ρύθμιση.

```csharp
public float ValueInSpecifiedUnits { get; set; }
```

### Αξία περιουσίας

Η τιμή σε καθορισμένες μονάδες.
