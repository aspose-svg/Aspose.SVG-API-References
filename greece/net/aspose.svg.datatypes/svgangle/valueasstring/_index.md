## SVGAngle.ValueAsString property

Η τιμή της γωνίας ως τιμή συμβολοσειράς, στις μονάδες που εκφράζονται με unitType. Η ρύθμιση αυτού του χαρακτηριστικού θα προκαλέσει την αυτόματη ενημέρωση των τιμών, valueInSpecifiedUnits και unitType ώστε να αντικατοπτρίζει αυτήν τη ρύθμιση.

```csharp
public string ValueAsString { get; set; }
```

### Αξία περιουσίας

Η τιμή ως συμβολοσειρά.
