## ITrueTypeFont.SubFamilyName property

Το όνομα της υποοικογένειας γραμματοσειράς διακρίνει τη γραμματοσειρά σε μια ομάδα με το ίδιο όνομα οικογένειας γραμματοσειράς. Υποτίθεται ότι αφορά το στυλ (πλάγια, πλάγια) και το βάρος (ανοιχτό, τολμηρό, μαύρο, κ.λπ.). Μια γραμματοσειρά χωρίς ιδιαίτερες διαφορές στο βάρος ή το στυλ (π.χ. μεσαίο βάρος, όχι πλάγιο και σύνολο fsSelection bit 6) θα πρέπει να έχει αποθηκευμένη τη συμβολοσειρά "Regular" σε αυτή τη θέση.

```csharp
public string SubFamilyName { get; }
```
