## FromRgba(byte, byte, byte, byte) {#fromrgba}

Επιστρέφει ένα νέο χρώμα με τις ζητούμενες τιμές ged, πράσινο, μπλε, άλφα. Όλα τα στοιχεία χρώματος πρέπει να βρίσκονται στην περιοχή 0-255.

```csharp
public static Color FromRgba(byte red, byte green, byte blue, byte alpha)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| red | Byte | Ένα byte που αντιπροσωπεύει το κόκκινο συστατικό του χρώματος. |
| green | Byte | Ένα byte που αντιπροσωπεύει το πράσινο στοιχείο του χρώματος. |
| blue | Byte | Ένα byte που αντιπροσωπεύει το μπλε συστατικό του χρώματος. |
| alpha | Byte | Ένα byte που αντιπροσωπεύει το στοιχείο άλφα του χρώματος. |
