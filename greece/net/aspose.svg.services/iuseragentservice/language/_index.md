## IUserAgentService.Language property

Το`Language` καθορίζει την κύρια γλώσσα για τα περιεχόμενα του στοιχείου και για οποιοδήποτε από τα χαρακτηριστικά του στοιχείου που περιέχουν κείμενο. Η τιμή του πρέπει να είναι έγκυρο BCP 47 ( ετικέτα γλώσσας ή την κενή συμβολοσειρά. Η ρύθμιση του χαρακτηριστικού στην κενή συμβολοσειρά υποδηλώνει ότι η κύρια γλώσσα είναι άγνωστη.

```csharp
public string Language { get; set; }
```

### Αξία περιουσίας

Η γλώσσα του παράγοντα χρήστη.
