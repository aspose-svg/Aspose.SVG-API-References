## CSSPrimitiveValue.SetFloatValue method

Μια μέθοδος για τον ορισμό της τιμής float με μια καθορισμένη μονάδα. Εάν η ιδιότητα που επισυνάπτεται με αυτήν την τιμή δεν μπορεί να δεχτεί την καθορισμένη μονάδα ή την τιμή float, η τιμή θα παραμείνει αμετάβλητη και θα αυξηθεί μια εξαίρεση DOME.

```csharp
public abstract void SetFloatValue(ushort unitType, float floatValue)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| unitType | UInt16 | Τύπος μονάδας. |
| floatValue | Single | Η κινητή τιμή. |
