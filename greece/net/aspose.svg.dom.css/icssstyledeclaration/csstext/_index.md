## ICSSStyleDeclaration.CSSText property

Η αναλυόμενη κειμενική αναπαράσταση του μπλοκ δήλωσης (εξαιρουμένων των γύρω σγουρά άγκιστρα). Η ρύθμιση αυτού του χαρακτηριστικού θα έχει ως αποτέλεσμα την ανάλυση της νέας τιμής και την επαναφορά όλων των ιδιοτήτων στο μπλοκ δήλωσης, συμπεριλαμβανομένης της αφαίρεσης ή της προσθήκης ιδιοτήτων.

```csharp
public string CSSText { get; set; }
```

Το κείμενο CSS.
