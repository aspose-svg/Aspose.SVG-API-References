## ICSSStyleDeclaration indexer

Χρησιμοποιείται για την ανάκτηση των ιδιοτήτων που έχουν οριστεί ρητά σε αυτό το μπλοκ δήλωσης. Η σειρά των ιδιοτήτων που ανακτήθηκαν χρησιμοποιώντας αυτήν τη μέθοδο δεν χρειάζεται να είναι η σειρά με την οποία ορίστηκαν. Αυτή η μέθοδος μπορεί να χρησιμοποιηθεί για επανάληψη σε όλες τις ιδιότητες σε αυτό το μπλοκ δήλωσης.

```csharp
public string this[int index] { get; }
```

| Παράμετρος | Περιγραφή |
| --- | --- |
| index | Ο δείκτης ιδιοκτησίας |
