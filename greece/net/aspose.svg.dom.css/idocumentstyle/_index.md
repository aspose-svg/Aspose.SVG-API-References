## IDocumentStyle interface

Η διεπαφή DocumentStyle παρέχει έναν μηχανισμό με τον οποίο μπορούν να ανακτηθούν τα φύλλα στυλ που είναι ενσωματωμένα σε ένα έγγραφο. Η προσδοκία είναι ότι μια παρουσία της διεπαφής DocumentStyle μπορεί να ληφθεί με τη χρήση μεθόδων casting ειδικά για δέσμευση σε μια παρουσία της διεπαφής Document.

```csharp
public interface IDocumentStyle
```
