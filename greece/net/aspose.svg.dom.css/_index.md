Το **Aspose.Svg.Dom.Css** Ο χώρος ονομάτων είναι για όλους τους χειρισμούς που σχετίζονται με το CSS. Συγκεντρώνεται γύρω από το όνομα ιδιότητας CSS - ζεύγη τιμών που καθορίζονται από επίσημα έγγραφα CSS.

## Τάξεις

| Τάξη | Περιγραφή |
| --- | --- |
| [Counter](./counter/) | Η διεπαφή Counter χρησιμοποιείται για να αναπαραστήσει οποιαδήποτε τιμή συνάρτησης μετρητή ή μετρητών. Αυτή η διεπαφή αντικατοπτρίζει τις τιμές στην υποκείμενη ιδιότητα στυλ. |
| [CSSPrimitiveValue](./cssprimitivevalue/) | Η διεπαφή CSSPrimitiveValue αντιπροσωπεύει μια μεμονωμένη τιμή CSS. Αυτή η διεπαφή μπορεί να χρησιμοποιηθεί για τον προσδιορισμό της τιμής μιας συγκεκριμένης ιδιότητας στυλ που έχει οριστεί αυτήν τη στιγμή σε ένα μπλοκ ή για τον ορισμό μιας συγκεκριμένης ιδιότητας στυλ ρητά εντός του μπλοκ. Μια παρουσία αυτής της διεπαφής μπορεί να ληφθεί από τη μέθοδο getPropertyCSSValue της διεπαφής CSSStyleDeclaration. Ένα αντικείμενο CSSPrimitiveValue εμφανίζεται μόνο σε ένα περιβάλλον μιας ιδιότητας CSS. |
| [CSSValue](./cssvalue/) | Αντιπροσωπεύει μια απλή ή μια σύνθετη τιμή. Ένα αντικείμενο CSSValue εμφανίζεται μόνο σε ένα περιβάλλον μιας ιδιότητας CSS. |
| [CSSValueList](./cssvaluelist/) | Η διεπαφή CSSValueList παρέχει την αφαίρεση μιας διατεταγμένης συλλογής τιμών CSS. |
| [Rect](./rect/) | Η διεπαφή Rect χρησιμοποιείται για να αναπαραστήσει οποιαδήποτε ορθή τιμή. Αυτή η διεπαφή αντικατοπτρίζει τις τιμές στην υποκείμενη ιδιότητα στυλ. Ως εκ τούτου, οι τροποποιήσεις που γίνονται στα αντικείμενα CSSPrimitiveValue τροποποιούν την ιδιότητα στυλ. |
| [RGBColor](./rgbcolor/) | Η διεπαφή RGBColor χρησιμοποιείται για να αναπαραστήσει οποιαδήποτε τιμή χρώματος RGB. Αυτή η διεπαφή αντικατοπτρίζει τις τιμές στην υποκείμενη ιδιότητα στυλ. Ως εκ τούτου, οι τροποποιήσεις που γίνονται στα αντικείμενα CSSPrimitiveValue τροποποιούν την ιδιότητα στυλ. |

Η τιμή του χαρακτηριστικού κωδικοποίησης μπορεί επίσης να χρησιμοποιηθεί ως υπόδειξη για την κωδικοποίηση που χρησιμοποιείται στη σειριοποίηση του φύλλου στυλ. |
