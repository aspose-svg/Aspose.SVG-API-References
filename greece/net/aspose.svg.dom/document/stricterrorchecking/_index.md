## Document.StrictErrorChecking property

Χαρακτηριστικό που προσδιορίζει εάν επιβάλλεται έλεγχος σφαλμάτων ή όχι. Όταν οριστεί σε false, η υλοποίηση είναι ελεύθερη να μην δοκιμάζει κάθε πιθανή περίπτωση σφάλματος που κανονικά ορίζεται σε λειτουργίες DOM και να μην δημιουργεί καμία εξαίρεση DOME στις λειτουργίες DOM ή να αναφέρει σφάλματα κατά τη χρήση της Document.normalizeDocument(). Σε περίπτωση λάθους, η συμπεριφορά είναι απροσδιόριστη. Αυτό το χαρακτηριστικό είναι αληθές από προεπιλογή.

```csharp
public bool StrictErrorChecking { get; set; }
```
