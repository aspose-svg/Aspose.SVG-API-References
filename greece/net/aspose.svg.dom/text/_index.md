## Text class

Η διεπαφή κειμένου κληρονομεί από το CharacterData και αντιπροσωπεύει το περιεχόμενο κειμένου (ονομάζονται δεδομένα χαρακτήρων σε XML) ενός Στοιχείου ή Attr.

```csharp
public class Text : CharacterData
```
