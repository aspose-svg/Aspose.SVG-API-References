## IWindow.Confirm method

Εμφανίζει μια τροπική προτροπή OK/Cancel με το δεδομένο μήνυμα, περιμένει να το απορρίψει ο χρήστης και επιστρέφει true εάν ο χρήστης κάνει κλικ στο OK και false εάν ο χρήστης κάνει κλικ στο Cancel.

```csharp
public bool Confirm(string message)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| message | String | Το μήνυμα. |

### Επιστρεφόμενη Αξία

Επιστρέφει true εάν ο χρήστης κάνει κλικ στο OK και false αν ο χρήστης κάνει κλικ στο Cancel
