## IWindow.Top property

Το κορυφαίο χαρακτηριστικό IDL στο αντικείμενο Window ενός Εγγράφου σε περιβάλλον περιήγησης b πρέπει να επιστρέψει το αντικείμενο WindowProxy του περιβάλλοντος περιήγησής του ανώτατου επιπέδου (το οποίο θα ήταν το δικό του αντικείμενο WindowProxy αν ήταν ένα περιβάλλον περιήγησης ανώτατου επιπέδου), εάν έχει ένα ή το δικό του αντικείμενο WindowProxy διαφορετικά (π.χ. αν ήταν ένα αποκομμένο ένθετο περιβάλλον περιήγησης).

```csharp
public IWindow Top { get; }
```
