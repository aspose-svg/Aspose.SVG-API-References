## IWindow.Parent property

Το γονικό χαρακτηριστικό IDL στο αντικείμενο Παράθυρο ενός εγγράφου σε περιβάλλον περιήγησης b πρέπει να επιστρέψει το αντικείμενο WindowProxy του περιβάλλοντος γονικής περιήγησης, εάν υπάρχει (δηλ. εάν το b είναι θυγατρικό περιβάλλον περιήγησης) ή το αντικείμενο WindowProxy της περιήγησης το ίδιο το πλαίσιο b, διαφορετικά (δηλαδή, εάν πρόκειται για περιβάλλον περιήγησης ανώτατου επιπέδου ή για αποκομμένο ένθετο περιβάλλον περιήγησης).

```csharp
public IWindow Parent { get; }
```
