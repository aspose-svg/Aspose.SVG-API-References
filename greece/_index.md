## Aspose.SVG για .NET

{{% alert color="primary" %}} 

![Λογότυπο προϊόντος Aspose.SVG για .NET](../home_1.png)


Το Aspose.SVG for .NET API είναι μια βιβλιοθήκη πολλαπλών πλατφορμών που παρέχει ένα ευρύ φάσμα δυνατοτήτων για την επεξεργασία και την απόδοση εγγράφων SVG. Το API διατηρεί το Document Object Model (DOM) και είναι πλήρως συμβατό με τις επίσημες προδιαγραφές SVG, επιτρέποντάς σας να έχετε τον απόλυτο έλεγχο των κόμβων SVG και των ιδιοτήτων τους. Μπορείτε εύκολα να τροποποιήσετε το δέντρο εγγράφων, να προσθέσετε και να αφαιρέσετε κόμβους, να αλλάξετε τις ιδιότητες του κόμβου όπως περιγράφεται στις επίσημες προδιαγραφές. Χρησιμοποιώντας το Aspose.SVG API μπορείτε να φορτώσετε γρήγορα, να αποθηκεύσετε και να μετατρέψετε SVG, καθώς και να διαβάσετε και να διασχίσετε τα στοιχεία του αρχείου. Το API είναι ανεξάρτητο από οποιοδήποτε άλλο λογισμικό. Σας δίνει τη δυνατότητα να εργάζεστε με αρχεία SVG σε όλες τις πλατφόρμες που υποστηρίζουν .NET (Windows, Linux και macOS).
