## SVGListBase&lt;T&gt;.InsertItemBefore method

Inserta un nuevo elemento en la lista en la posición especificada. El primer elemento es el número 0.

```csharp
public T InsertItemBefore(T newItem, ulong index)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| newItem | T | El elemento que se va a insertar en la lista. |
| index | UInt64 | El índice del elemento antes del cual se insertará el nuevo elemento. El primer elemento es el número 0. Si el índice es igual a 0, el nuevo elemento se inserta al principio de la lista. Si el índice es mayor o igual que numberOfItems, el nuevo elemento se agrega al final de la lista. |

### Valor_devuelto

El elemento insertado.
