## NodeList indexer

El método devuelve el elemento índice de la colección. Si el índice es mayor o igual que el número de nodos en la lista, esto devuelve nulo.

```csharp
public abstract Node this[int index] { get; }
```

| Parámetro | Descripción |
| --- | --- |
| index | El índice. |

### Valor_devuelto

Devuelve nodo.
