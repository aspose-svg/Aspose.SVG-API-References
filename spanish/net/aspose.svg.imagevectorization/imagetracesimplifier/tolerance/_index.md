## ImageTraceSimplifier.Tolerance property

El valor de la tolerancia determina la tolerancia de error máxima permitida para que un punto sea eliminado de la traza. Debe estar en el rango de 0 a 4. Cualquier valor mayor o menor se alineará con los valores mínimo y máximo de este rango, según corresponda. El valor predeterminado es 0.3.

```csharp
public float Tolerance { get; set; }
```
