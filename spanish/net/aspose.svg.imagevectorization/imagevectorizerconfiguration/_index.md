## ImageVectorizerConfiguration class

El`ImageVectorizerConfiguration` class define una configuración de métodos y opciones de vectorización de imágenes. La configuración se utiliza para inicializar un ImageVectorizer y proporciona las opciones de configuración para vectorizar imágenes.
