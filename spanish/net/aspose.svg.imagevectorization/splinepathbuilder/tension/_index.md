## SplinePathBuilder.Tension property

El valor de las tensiones afecta la forma en que la curva se dobla en los puntos de control (interpolados). Debe estar en el rango de 0 a 1. Cualquier valor mayor o menor se alineará con los valores mínimo y máximo de este rango, en consecuencia.

```csharp
public float Tension { get; set; }
```
