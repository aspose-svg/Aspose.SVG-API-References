## IWindow.Confirm method

Muestra un indicador modal Aceptar/Cancelar con el mensaje dado, espera a que el usuario lo descarte y devuelve verdadero si el usuario hace clic en Aceptar y falso si el usuario hace clic en Cancelar.

```csharp
public bool Confirm(string message)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| message | String | El mensaje. |

### Valor_devuelto

Devuelve verdadero si el usuario hace clic en Aceptar y falso si el usuario hace clic en Cancelar
