## IWindow.Parent property

El atributo IDL principal en el objeto Ventana de un Documento en un contexto de navegación b debe devolver el objeto WindowProxy del contexto de navegación principal, si lo hay (es decir, si b es un contexto de navegación secundario), o el objeto WindowProxy del contexto de navegación contexto b en sí mismo, de lo contrario (es decir, si es un contexto de navegación de nivel superior o un contexto de navegación anidado separado).

```csharp
public IWindow Parent { get; }
```

### El valor de la propiedad

El padre.
