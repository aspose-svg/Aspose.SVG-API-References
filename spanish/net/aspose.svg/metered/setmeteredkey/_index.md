## Metered.SetMeteredKey method

Establece la clave pública y privada medidas. Si compra una licencia medida, cuando inicie la aplicación, esta API debe llamarse, normalmente, esto es suficiente. Sin embargo, si siempre falla al cargar los datos de consumo y excede las 24 horas, la licencia se establecerá en estado de evaluación, para evitar tal caso, debe verificar regularmente el estado de la licencia, si es el estado de evaluación, llame a esta API nuevamente.

public void SetMeteredKey(string publicKey, string privateKey)
