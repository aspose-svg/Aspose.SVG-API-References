## SVGSVGElement.CurrentTranslate property

En un elemento svg más externo, el factor de traducción correspondiente que tiene en cuenta la "ampliación" del usuario. Cuando se accede a un elemento 'svg' que no es un elemento svg más externo, no está definido qué comportamiento tiene este atributo.

```csharp
public SVGPoint CurrentTranslate { get; }
```

### El valor de la propiedad

La traducción actual.
