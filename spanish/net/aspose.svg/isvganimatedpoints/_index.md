## ISVGAnimatedPoints interface

La interfaz SVGAnimatedPoints admite elementos que tienen un atributo de "puntos" que contiene una lista de valores de coordenadas y que admite la capacidad de animar ese atributo. Además, el atributo de "puntos" en el elemento original al que se accede a través del XML DOM (p. utilizando la llamada al método getAttribute()) reflejará cualquier cambio realizado en points.

```csharp
public interface ISVGAnimatedPoints
```

## Propiedades

| Nombre | Descripción |
| --- | --- |
| [AnimatedPoints](../../aspose.svg/isvganimatedpoints/animatedpoints/) { get; } | Proporciona acceso a los contenidos animados actuales del atributo 'puntos'. Si el atributo o la propiedad dados se están animando, contiene el valor animado actual del atributo o la propiedad. Si el atributo o la propiedad dados no se están animando actualmente, contiene el mismo valor que los puntos. |
| [Points](../../aspose.svg/isvganimatedpoints/points/) { get; } | Proporciona acceso al contenido base (es decir, estático) del atributo 'puntos'. |
