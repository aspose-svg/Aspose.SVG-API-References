## ISVGAnimatedPoints.AnimatedPoints property

Proporciona acceso a los contenidos animados actuales del atributo 'puntos'. Si el atributo o la propiedad dados se están animando, contiene el valor animado actual del atributo o la propiedad. Si el atributo o la propiedad dados no se están animando actualmente, contiene el mismo valor que los puntos.

public SVGPointList AnimatedPoints { get; }
