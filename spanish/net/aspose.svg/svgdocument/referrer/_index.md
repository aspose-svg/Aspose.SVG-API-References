## SVGDocument.Referrer property

Devuelve el URI de la página que enlaza con esta página. El valor es una cadena vacía si el usuario navegó a la página directamente (no a través de un enlace, sino, por ejemplo, a través de un marcador).

```csharp
public string Referrer { get; }
```

### El valor de la propiedad

El remitente.
