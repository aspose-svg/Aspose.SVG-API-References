## SVGMarkerElement.OrientAngle property

Corresponde al atributo 'orientar' en el elemento 'marcador' dado. Si markerUnits es SVG_MARKER_ORIENT_ANGLE, el valor del ángulo para el atributo 'orientar'; de lo contrario, se establecerá en cero.

```csharp
public SVGAnimatedAngle OrientAngle { get; }
```

### El valor de la propiedad

El atributo 'orientar' del elemento 'marcador' dado.
