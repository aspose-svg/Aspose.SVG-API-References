## SVGViewElement.ViewTarget property

Corresponde al atributo 'viewTarget' en el elemento 'view' dado. Una lista de valores DOMString que contienen los nombres enumerados en el atributo 'viewTarget'. Cada uno de los valores de DOMString se puede asociar con el elemento correspondiente mediante la llamada al método getElementById().

public SVGStringList ViewTarget { get; }
