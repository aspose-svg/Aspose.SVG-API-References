## SVGElement.Style property

Corresponde al atributo 'estilo' en el elemento dado. Si la aplicación de usuario no admite estilos con CSS, este atributo siempre debe tener el valor null.

```csharp
public ICSSStyleDeclaration Style { get; }
```

### El valor de la propiedad

El estilo.
