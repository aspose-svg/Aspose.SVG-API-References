## IUrlSearchParams.Set method

Establece el valor del primer par de nombre-valor encontrado en el valor especificado y elimina los demás. Si no se encuentran pares de nombre-valor con el nombre especificado, se agregará uno nuevo a la lista.

```csharp
public void Set(string name, string value)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| name | String | Clave que se utilizará para encontrar pares nombre-valor. |
| value | String | Valor que se establecerá. |
