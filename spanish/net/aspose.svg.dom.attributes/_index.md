El **Aspose.Svg.Dom.Atributos** El espacio de nombres contiene clases de atributos unificados que especifican o califican cualquier objeto (nodo, elemento, ...) característica o comportamiento.

## Clases

| Clase | Descripción |
| --- | --- |
| [DOMConstructorAttribute](./domconstructorattribute/) | Especifica un constructor definido por el W3C. |
| [DOMNameAttribute](./domnameattribute/) | Especifica el nombre del objeto DOM oficial tal como lo define el W3C. |
| [DOMNamedPropertyGetterAttribute](./domnamedpropertygetterattribute/) | Especifica que el método se usará como captador de propiedad con nombre. |
| [DOMNoInterfaceObjectAttribute](./domnointerfaceobjectattribute/) | Si el atributo extendido [NoInterfaceObject] aparece en una interfaz, indica que no existirá un objeto de interfaz para la interfaz en el enlace de ECMAScript. |
| [DOMNullableAttribute](./domnullableattribute/) | Especifica que a un objeto DOM se le puede asignar un valor nulo. |
| [DOMObjectAttribute](./domobjectattribute/) | Especifica que el objeto marcado con este atributo está definido por el W3C. |
| [DOMTreatNullAsAttribute](./domtreatnullasattribute/) | Indica que el valor nulo del miembro se tratará como valor especificado. |
## Enumeración

| Enumeración | Descripción |
| --- | --- |
