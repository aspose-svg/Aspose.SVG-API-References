## Node.OwnerDocument property

El objeto Documento asociado con este nodo. Este es también el objeto Documento utilizado para crear nuevos nodos. Cuando este nodo es un Documento o un Tipo de documento que aún no se usa con ningún Documento, es nulo.

```csharp
public virtual Document OwnerDocument { get; }
```
