## Node.InsertBefore method

Inserta el nodo antes del nodo secundario existente. Si hijo es nulo, inserte el nodo al final de la lista de hijos. Si hijo es un objeto DocumentFragment, todos sus hijos se insertan, en el mismo orden, antes de hijo. Si el niño ya está en el árbol, primero se elimina.

```csharp
public Node InsertBefore(Node node, Node child)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| node | Node | El niño nuevo. |
| child | Node | El niño árbitro. |

### Valor_devuelto

Devuelve el nodo insertado
