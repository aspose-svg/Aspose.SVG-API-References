## TypeInfo.DERIVATION_RESTRICTION field

Si el esquema del documento es un Esquema XML [Esquema XML Parte 1], esta constante representa la derivación por restricción si se trata de tipos complejos, o una restricción si se trata de tipos simples.

```csharp
public const ulong DERIVATION_RESTRICTION;
```
