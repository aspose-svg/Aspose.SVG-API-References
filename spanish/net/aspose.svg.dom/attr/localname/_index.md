## Attr.LocalName property

Devuelve la parte local del nombre calificado de este nodo. Para los nodos de cualquier tipo que no sean ELEMENT_NODE y ATTRIBUTE_NODE y los nodos creados con un método DOM Nivel 1, como Document.createElement(), esto siempre es nulo.

```csharp
public override string LocalName { get; }
```

### El valor de la propiedad

El nombre del local.
