## Element.SetIdAttribute method

Si el parámetro isId es verdadero, este método declara que el atributo especificado es un atributo de ID determinado por el usuario.

```csharp
public void SetIdAttribute(string name, bool isId)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| name | String | El nombre del atributo. |
| isId | Boolean | si se establece en`verdadero` [es identificación]. |
