## Element.SetAttributeNS method

Agrega un nuevo atributo. Si un atributo con el mismo nombre local y URI de espacio de nombres ya está presente en el elemento, su prefijo se cambia para que sea la parte del prefijo del nombre calificado y su valor se cambia para que sea el parámetro de valor.

```csharp
public void SetAttributeNS(string namespaceURI, string qualifiedName, string value)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| namespaceURI | String | El URI del espacio de nombres. |
| qualifiedName | String | Nombre del calificado. |
| value | String | El valor. |
