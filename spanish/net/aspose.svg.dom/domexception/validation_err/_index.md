## DOMException.VALIDATION_ERR field

Si una llamada a un método como insertBefore o removeChild hiciera que el nodo no fuera válido con respecto a la "validez parcial", esta excepción se generaría y la operación no se realizaría. Este código se utiliza en [Validación de nivel 3 de DOM]. Consulte esta especificación para obtener más información.
