## Document.XmlStandalone property

Un atributo que especifica, como parte de la declaración XML, si este documento es independiente. Esto es falso cuando no se especifica.

```csharp
public bool XmlStandalone { get; set; }
```

### El valor de la propiedad

`verdadero` si XML es independiente; de lo contrario,`FALSO` .
