## Document.StrictErrorChecking property

Un atributo que especifica si se aplica o no la comprobación de errores. Cuando se establece en falso, la implementación es libre de no probar todos los casos de error posibles normalmente definidos en las operaciones DOM, y no generar ninguna DOMException en las operaciones DOM o informar errores al usar Document.normalizeDocument(). En caso de error, el comportamiento es indefinido. Este atributo es verdadero por defecto.

```csharp
public bool StrictErrorChecking { get; set; }
```

### El valor de la propiedad

`verdadero` si la comprobación de errores es estricta; de lo contrario,`falso` .
