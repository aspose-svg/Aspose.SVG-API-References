## Document.ReadyState property

Devuelve la preparación del documento. La "carga" mientras se carga el documento, "interactiva" una vez que finaliza el análisis pero sigue cargando subrecursos, y "completa" una vez que se ha cargado.

public string ReadyState { get; }
