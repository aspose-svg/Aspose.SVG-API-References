## Document.DefaultView property

El atributo defaultView IDL de la interfaz del Documento, al obtenerlo, debe devolver el objeto WindowProxy del contexto de navegación de este Documento, si este Documento tiene un contexto de navegación asociado, o nulo en caso contrario.

```csharp
public IWindow DefaultView { get; }
```
