El **Aspose.Svg.Dom (Modelo de objeto de documento)** El espacio de nombres proporciona una API que representa e interactúa con cualquier documento HTML, XML o SVG. El DOM es un modelo de documento cargado en el navegador y que representa el documento como un árbol de nodos, donde cada nodo representa parte del documento (por ejemplo, un elemento, texto cadena o comentario).
