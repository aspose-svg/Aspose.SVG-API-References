## IUserAgentService.Language property

El`Language` especifica el idioma principal para el contenido del elemento y para cualquiera de los atributos del elemento que contienen texto. Su valor debe ser un BCP 47 válido ( etiqueta de idioma o la cadena vacía. Establecer el atributo en la cadena vacía indica que se desconoce el idioma principal.

```csharp
public string Language { get; set; }
```

### El valor de la propiedad

El idioma del agente de usuario.
