El **Aspose.Svg.Servicios** el espacio de nombres contiene interfaces como protocolos para implementaciones de servicio.

## Interfaces

| Interfaz | Descripción |
| --- | --- |
| [IDeviceInformationService](./ideviceinformationservice/) | Una interfaz que describe un entorno en el que[`Document`](../aspose.svg.dom/document/) se presenta al usuario. |
| [INetworkService](./inetworkservice/) | Proporciona una interfaz para las operaciones de red. |
| [IRuntimeService](./iruntimeservice/) | Este servicio se utiliza para configurar propiedades relacionadas con el tiempo de ejecución. |
| [IService](./iservice/) | Define una interfaz base para la implementación del servicio. |
| [IServiceContainer](./iservicecontainer/) | Proporciona un contenedor para servicios. |
| [IServiceProvider](./iserviceprovider/) | Proporciona un contenedor inmutable para servicios. |
| [IUserAgentService](./iuseragentservice/) | Una interfaz que se describe como un entorno de agente de usuario. |
