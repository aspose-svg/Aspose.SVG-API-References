El **Aspose.Svg.Rendering.Imagen** El espacio de nombres proporciona clase de dispositivo específica, así como algunas opciones de representación clases responsables de la representación en formatos de trama: jpeg, png, bmp, gif, tiff.

## Clases

| Clase | Descripción |
| --- | --- |
| [ImageDevice](./imagedevice/) | Representa la representación en formatos de trama: jpeg, png, bmp, gif, tiff. |
| [ImageRenderingOptions](./imagerenderingoptions/) | Representa las opciones de representación para[`ImageDevice`](../aspose.svg.rendering.image/imagedevice/) . Esta opción se utiliza para especificar el formato de la imagen de salida, la compresión, la resolución, etc. |
| [TextOptions](./textoptions/) | Representa las opciones de representación de texto para[`ImageDevice`](../aspose.svg.rendering.image/imagedevice/) . |
## Enumeración

| Enumeración | Descripción |
| --- | --- |
| [Compression](./compression/) | Especifica los posibles esquemas de compresión para imágenes de mapa de bits en formato de archivo de imagen etiquetada (TIFF). |
| [ImageFormat](./imageformat/) | Especifica el formato de archivo de la imagen. |
