El **Aspose.Svg.Toolkit.Optimizadores** El espacio de nombres contiene clases para optimizar documentos SVG. El proceso de optimización implica eliminar elementos no utilizados o invisibles y sus atributos, fusionar grupos y reducir el tamaño de los segmentos de ruta.
