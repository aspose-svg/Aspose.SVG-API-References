## ITreeWalker.PreviousSibling method

Mueve el TreeWalker al hermano anterior del nodo actual y devuelve el nuevo nodo. Si el nodo actual no tiene un hermano anterior visible , devuelve nulo y retiene el nodo actual .

```csharp
public Node PreviousSibling()
```

### Valor_devuelto

El nuevo nodo, o nulo si el nodo actual no tiene ningún hermano anterior. en la vista lógica del TreeWalker.
