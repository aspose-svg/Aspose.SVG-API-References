## ITreeWalker.NextSibling method

Mueve el TreeWalker al siguiente hermano del nodo actual y devuelve el nuevo nodo. Si el nodo actual no tiene el siguiente hermano visible , devuelve nulo y retiene el nodo actual.

```csharp
public Node NextSibling()
```

### Valor_devuelto

El nuevo nodo, o nulo si el nodo actual no tiene el siguiente hermano. en la vista lógica del TreeWalker.
