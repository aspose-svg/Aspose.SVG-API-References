## ITreeWalker.PreviousNode method

Mueve el TreeWalker al nodo visible anterior en el orden del documento relativo al nodo actual y devuelve el nodo new . Si el nodo actual no tiene un nodo anterior, o si la búsqueda de anteriorNodo intenta ascender desde el nodo raíz del TreeWalker , devuelve nulo y retiene el nodo actual.

```csharp
public Node PreviousNode()
```

### Valor_devuelto

El nuevo nodo, o nulo si el nodo actual no tiene un nodo anterior en la vista lógica del TreeWalker.
