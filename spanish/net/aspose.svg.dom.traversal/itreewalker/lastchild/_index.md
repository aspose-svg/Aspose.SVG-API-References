## ITreeWalker.LastChild method

Mueve el TreeWalker al último elemento secundario visible del nodo actual y devuelve el nuevo nodo. Si el nodo actual no tiene hijos visibles, devuelve un valor nulo y retiene el nodo actual.

```csharp
public Node LastChild()
```

### Valor_devuelto

El nuevo nodo, o nulo si el nodo actual no tiene hijos en la vista lógica del TreeWalker.
