## ITreeWalker.ParentNode method

Se desplaza y devuelve el nodo antecesor visible más cercano del nodo actual . Si la búsqueda de parentNode intenta subir hacia arriba desde el nodo raíz del TreeWalker, o si no logra encontrar un nodo antepasado visible, este método conserva la posición actual y devuelve nulo.

```csharp
public Node ParentNode()
```

### Valor_devuelto

El nuevo nodo padre, o nulo si el nodo actual no tiene padre en la vista lógica del TreeWalker.
