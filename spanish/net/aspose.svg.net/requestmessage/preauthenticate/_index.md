## RequestMessage.PreAuthenticate property

Obtiene o establece un valor que indica si se debe enviar un encabezado de Autorización con la solicitud.

```csharp
public bool PreAuthenticate { get; set; }
```

### El valor de la propiedad

`verdadero` para enviar un encabezado de autorización HTTP con solicitudes después de que se haya realizado la autenticación; de lo contrario,`FALSO` . El valor predeterminado es`FALSO` .
