El **Aspose.Svg.Net**el espacio de nombres es presentado por clases e interfaces que son responsables de facilitar el procesamiento de la red.

## Clases

| Clase | Descripción |
| --- | --- |
| [ByteArrayContent](./bytearraycontent/) | Representa contenido basado en una matriz de bytes. |
| [Content](./content/) | Representa una clase base para un cuerpo de entidad HTTP y encabezados de contenido. |
| [ContentHeaders](./contentheaders/) | Contiene las cabeceras asociadas a un contenido. |
| [FormUrlEncodedContent](./formurlencodedcontent/) | Un contenedor para tuplas de nombre/valor codificadas usando el tipo MIME application/x-www-form-urlencoded. |
| [HttpMethod](./httpmethod/) | Representa la clase de utilidad para recuperar y comparar métodos HTTP estándar. |
| [MessageFilter](./messagefilter/) | Representa una clase base abstracta para diferentes clases de filtros utilizados para consultar mensajes |
| [MessageHandler](./messagehandler/) | Representa un tipo base para los controladores de mensajes. |
| [MessageHandlerCollection](./messagehandlercollection/) | Representa la colección de los[`MessageHandler`](../aspose.svg.net/messagehandler/) . |
| [MultipartContent](./multipartcontent/) | Representa un contenido de varias partes/*. |
| [MultipartFormDataContent](./multipartformdatacontent/) | Representar contenido para el algoritmo de codificación de datos de formulario/multiparte |
| [RequestHeaders](./requestheaders/) | Contiene encabezados de protocolo asociados con una solicitud. |
| [RequestMessage](./requestmessage/) | Representa un mensaje de solicitud. |
| [ResponseHeaders](./responseheaders/) | Contiene encabezados de protocolo asociados con una respuesta. |
| [ResponseMessage](./responsemessage/) | Representa un mensaje de respuesta. |
| [StreamContent](./streamcontent/) | Representa contenido basado en una secuencia. |
| [StringContent](./stringcontent/) | Representa contenido basado en una cadena. |
| [UrlResolver](./urlresolver/) | Representa la clase de utilidad para resolver la URL absoluta mediante un Identificador uniforme de recursos (URI). |
