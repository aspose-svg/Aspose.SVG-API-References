## OutputStream.Write method

Escribe una secuencia de bytes en el flujo de salida envuelto y avanza la posición actual dentro de este flujo por el número de bytes escritos.

```csharp
public override void Write(byte[] buffer, int offset, int count)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| buffer | Byte[] | Una matriz de bytes. Este método copia los bytes de conteo del búfer al flujo envuelto output . |
| offset | Int32 | El desplazamiento de bytes de base cero en el búfer en el que se empiezan a copiar bytes en la secuencia encapsulada output . |
| count | Int32 | El número de bytes que se escribirán en el flujo de salida empaquetado. |
