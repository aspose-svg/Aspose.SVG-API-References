## IFile interface

Un objeto File es un objeto Blob con un atributo de nombre, que es una cadena; se puede crear dentro de la aplicación web a través de un constructor, o es una referencia a una secuencia de bytes de un archivo del sistema de archivos subyacente (SO).

public interface IFile : IBlob
