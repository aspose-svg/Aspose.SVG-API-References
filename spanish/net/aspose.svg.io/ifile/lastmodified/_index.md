## IFile.LastModified property

La última fecha de modificación del archivo. Al obtener, si los agentes de usuario pueden hacer que esta información esté disponible, esto debe devolver un conjunto largo de la hora en que el archivo se modificó por última vez como el número de milisegundos desde la época de Unix.

public long LastModified { get; }
