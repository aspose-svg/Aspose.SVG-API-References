## ErrorEvent.FileName property

El atributo de nombre de archivo debe devolver el valor con el que se inicializó. Cuando se crea el objeto, este atributo debe inicializarse en la cadena vacía. Representa la URL absoluta del script en el que se produjo originalmente el error.

```csharp
public string FileName { get; }
```

### El valor de la propiedad

El nombre del archivo.
