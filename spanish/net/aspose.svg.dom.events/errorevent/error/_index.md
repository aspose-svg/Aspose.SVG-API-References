## ErrorEvent.Error property

El atributo de error debe devolver el valor con el que se inicializó. Cuando se crea el objeto, este atributo debe inicializarse como nulo. En su caso, se establece en el objeto que representa el error (por ejemplo, el objeto de excepción en el caso de una excepción DOM no detectada).

```csharp
public object Error { get; }
```

### El valor de la propiedad

El error.
