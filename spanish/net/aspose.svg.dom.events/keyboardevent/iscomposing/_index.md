## KeyboardEvent.IsComposing property

verdadero si el evento clave ocurre como parte de una sesión de composición, es decir, después de un evento de inicio de composición y antes del evento de finalización de composición correspondiente. El valor no inicializado de este atributo DEBE ser falso.
