## ITrueTypeFont.FullFontName property

Debe ser una combinación de "FamilyName" y "SubFamilyName". Excepción: si la fuente es "Regular" como se indica en "SubFamilyName", use solo el nombre de familia contenido en "FamilyName". Una excepción a la definición anterior de nombre de fuente completo es para las cadenas de la plataforma Microsoft para fuentes CFF OpenType: en este caso, la cadena de nombre de fuente completo debe ser idéntica a PostScript FontName en CFF Name INDEX.

```csharp
public string FullFontName { get; }
```

### Valor_devuelto

Nombre completo de la fuente
