El **Aspose.Svg.Dibujo** El espacio de nombres contiene objetos e interfaces para especificar medidas y unidades, así como objetos de atributos de dibujo como pinceles, colores y fuentes.

## Clases

| Clase | Descripción |
| --- | --- |
| [Angle](./angle/) | El tipo de datos de ángulo |
| [Color](./color/) | La clase Color le permite especificar colores como valores Rojo-Verde-Azul (RGB), valores Tono-Saturación-Luminosidad (HSL), valores Tono-Saturación-Valor (HSV), Tono-Blancura-Negrura (HWB) ), valores de luminosidad-AB (LAB), valores de Luminance-Chroma-Hue (LCH), valores de Cyan-Magenta-Yellow-Key (CMYK), valores de colores naturales (NCOL), o con un nombre de color . También está disponible un canal alfa para indicar transparencia. |
| [Dimension](./dimension/) | Proporciona la clase base para las dimensiones. El término general 'dimensión' se refiere a un número con una unidad adjunta, y se denota por[`UnitType`](../aspose.svg.drawing/unittype/) . |
| [Frequency](./frequency/) | La unidad de 'frecuencia'. |
| [Length](./length/) | Representa una unidad de medida de longitud. |
| [LengthOrAuto](./lengthorauto/) | Representa un contenedor para la longitud de almacenamiento o unidades 'automáticas'.. |
| [Margin](./margin/) | Representa el margen de la página. |
| [Numeric](./numeric/) | Proporciona la clase base para tipos numéricos. |
| [Page](./page/) | Representa que un objeto de página se usa para la página de salida de configuración. El valor de laguna para el tamaño de página es A4 (210x297 mm) |
| [Resolution](./resolution/) | Representa una unidad de resolución. |
| [Size](./size/) | Almacena valores que especifican Altura y Anchura. |
| [Time](./time/) | Representa una unidad de tiempo. |
| [Unit](./unit/) | Proporciona la clase base para las unidades de medida. |
| [UnitType](./unittype/) | Especifica la unidad de medida. |
