## ICSSKeyframesRule interface

La interfaz CSSKeyframesRule representa un conjunto completo de fotogramas clave para una única animación

```csharp
public interface ICSSKeyframesRule : ICSSRule
```

## Propiedades

| Nombre | Descripción |
| --- | --- |
| [CSSRules](../../aspose.svg.dom.css/icsskeyframesrule/cssrules/) { get; } | Este atributo da acceso a los fotogramas clave en la lista |
| [Name](../../aspose.svg.dom.css/icsskeyframesrule/name/) { get; } | Este atributo es el nombre de los fotogramas clave, utilizados por la propiedad 'animation-name'. |

## Métodos

| Nombre | Descripción |
| --- | --- |
| [AppendRule](../../aspose.svg.dom.css/icsskeyframesrule/appendrule/)(string) | El método appendRule agrega la CSSKeyframeRule pasada a la lista en la clave pasada |
| [DeleteRule](../../aspose.svg.dom.css/icsskeyframesrule/deleterule/)(string) | El método deleteRule elimina la CSSKeyframeRule con la clave pasada. Si no existe una regla con esta clave, el método no hace nada |
| [FindRule](../../aspose.svg.dom.css/icsskeyframesrule/findrule/)(string) | El método findRule devuelve la regla con una clave que coincide con la clave pasada. Si no existe tal regla, se devuelve un valor nulo |
