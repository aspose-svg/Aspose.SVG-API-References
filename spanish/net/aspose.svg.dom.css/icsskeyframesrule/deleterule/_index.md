## ICSSKeyframesRule.DeleteRule method

El método deleteRule elimina la CSSKeyframeRule con la clave pasada. Si no existe una regla con esta clave, el método no hace nada

```csharp
public void DeleteRule(string key)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| key | String | La clave que describe la regla que se va a eliminar. La clave debe resolverse en un número entre 0 y 1, o la regla se ignora |
