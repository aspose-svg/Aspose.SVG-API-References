## ICSSKeyframesRule.FindRule method

El método findRule devuelve la regla con una clave que coincide con la clave pasada. Si no existe tal regla, se devuelve un valor nulo

```csharp
public ICSSKeyframeRule FindRule(string key)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| key | String | La clave que describió la regla a encontrar. La clave debe resolverse en un número entre 0 y 1, o la regla se ignora. |

### Valor_devuelto

La regla encontrada
