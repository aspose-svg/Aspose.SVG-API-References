## ICSSRule.ParentRule property

Si esta regla está contenida dentro de otra regla (por ejemplo, una regla de estilo dentro de un bloque @media), esta es la regla contenedora. Si esta regla no está anidada dentro de ninguna otra regla, devuelve null.
