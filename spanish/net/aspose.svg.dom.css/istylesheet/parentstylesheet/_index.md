## IStyleSheet.ParentStyleSheet property

Para los lenguajes de hoja de estilo que admiten el concepto de inclusión de hoja de estilo, este atributo representa la hoja de estilo incluida, si existe. Si la hoja de estilo es una hoja de estilo de nivel superior o el lenguaje de la hoja de estilo no admite la inclusión, el valor de este atributo es nulo.

```csharp
public IStyleSheet ParentStyleSheet { get; }
```
