## ICSSRuleList indexer

Se utiliza para recuperar una regla CSS mediante el método item() (http://www.w3.org/TR/DOM-Level-2-Style/css.html#CSS-CSSRuleList). El orden de esta colección representa el orden de las reglas en la hoja de estilo CSS. Si el índice es mayor o igual que el número de reglas en la lista, esto devuelve nulo.

```csharp
public ICSSRule this[int index] { get; }
```

| Parámetro | Descripción |
| --- | --- |
| index | El índice de reglas |

### Valor_devuelto

La instancia de la regla CSS.

### El valor de la propiedad

El[`ICSSRule`](../../icssrule/) .
