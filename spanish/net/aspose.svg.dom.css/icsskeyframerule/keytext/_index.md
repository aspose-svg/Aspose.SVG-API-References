## ICSSKeyframeRule.KeyText property

Este atributo representa el selector de fotogramas clave como una lista de valores porcentuales separados por comas. Las palabras clave 'from' y 'to' se asignan a '0%' y '100%', respectivamente.

public string KeyText { get; }
