## CSSPrimitiveValue.SetIntValue method

Un método para establecer el valor int con una unidad específica. Si la propiedad adjunta con este valor no puede aceptar la unidad especificada o el valor int, el valor no cambiará y se generará una DOMException.

```csharp
public abstract void SetIntValue(ushort unitType, int intValue)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| unitType | UInt16 | Tipo de unidad. |
| intValue | Int32 | El valor int. |
