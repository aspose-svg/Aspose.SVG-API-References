## CSSPrimitiveValue.GetRectValue method

Este método se utiliza para obtener el valor Rect. Si este valor CSS no contiene un valor rect, se genera una DOMException. La modificación de la propiedad de estilo correspondiente se puede lograr usando la interfaz Rect.

```csharp
public abstract Rect GetRectValue()
```
