## IDocumentStyle.StyleSheets property

Una lista que contiene todas las hojas de estilo explícitamente vinculadas o incrustadas en un documento. Para documentos HTML, esto incluye hojas de estilo externas, incluidas a través del elemento HTML LINK y elementos STYLE en línea.
