## IStyleSheetList indexer

Se utiliza para recuperar una hoja de estilo por elemento de método (índice int) según http://www.w3.org/TR/2000/REC-DOM-Level-2-Style-20001113/stylesheets.html.

```csharp
public ICSSStyleSheet this[int index] { get; }
```

| Parámetro | Descripción |
| --- | --- |
| index | El índice de la hoja de estilo |

### Valor_devuelto

La instancia de la hoja de estilo CSS.

### El valor de la propiedad

El[`ICSSStyleSheet`](../../icssstylesheet/) .
