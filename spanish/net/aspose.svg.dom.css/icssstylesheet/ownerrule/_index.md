## ICSSStyleSheet.OwnerRule property

Si esta hoja de estilo proviene de una regla @import, el atributo ownRule contendrá CSSImportRule. En ese caso, el atributo ownNode en la interfaz StyleSheet será nulo. Si la hoja de estilo proviene de un elemento o una instrucción de procesamiento, el atributo ownRule será nulo y el atributo ownNode contendrá el Nodo.
