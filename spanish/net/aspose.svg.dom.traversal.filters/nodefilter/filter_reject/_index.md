## NodeFilter.FILTER_REJECT field

Rechazar el nodo. Los métodos de navegación definidos para NodeIterator o TreeWalker no devolverán este nodo. Para TreeWalker, los hijos de este nodo también serán rechazados. NodeIterators trata esto como un sinónimo para FILTER_SKIP.

```csharp
public const short FILTER_REJECT;
```
