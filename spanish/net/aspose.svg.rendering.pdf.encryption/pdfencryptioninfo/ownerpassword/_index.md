## PdfEncryptionInfo.OwnerPassword property

Obtiene o establece la contraseña del propietario.

```csharp
public string OwnerPassword { get; set; }
```

### El valor de la propiedad

La contraseña del propietario.

### Observaciones

Abrir el documento con la contraseña de propietario correcta (suponiendo que no sea la misma que la contraseña de usuario) permite el acceso total (del propietario) al documento. Este acceso ilimitado incluye la posibilidad de cambiar las contraseñas del documento y permisos de acceso.
