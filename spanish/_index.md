## Aspose.SVG para .NET

{{% alert color="primary" %}} 

![Aspose.SVG para el logotipo del producto .NET](../home_1.png)


Aspose.SVG para .NET API es una biblioteca multiplataforma que proporciona una amplia gama de funciones para el procesamiento y la representación de documentos SVG. La API mantiene el modelo de objetos de documento (DOM) y es totalmente compatible con las especificaciones oficiales de SVG, lo que le permite tener un control total sobre los nodos SVG y sus propiedades. Puede modificar fácilmente el árbol del documento, agregar y eliminar nodos, cambiar las propiedades de los nodos como se describe en las especificaciones oficiales. Con la API de Aspose.SVG, puede cargar, guardar y convertir rápidamente archivos SVG, así como leer y recorrer los elementos del archivo. La API es independiente de cualquier otro software. Le permite trabajar con archivos SVG en todas las plataformas compatibles con .NET (Windows, Linux y macOS).
