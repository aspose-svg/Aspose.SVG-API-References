## Aspose.SVG для .NET

{{% alert color="primary" %}} 

![Логотип продукта Aspose.SVG для .NET](../home_1.png)


Aspose.SVG для .NET API — это кроссплатформенная библиотека, предоставляющая широкий спектр функций для обработки и рендеринга документов SVG. API поддерживает объектную модель документа (DOM) и полностью совместим с официальными спецификациями SVG, что позволяет вам полностью контролировать узлы SVG и их свойства. Вы можете легко модифицировать дерево документов, добавлять и удалять узлы, изменять свойства узлов, как это описано в официальных спецификациях. Используя Aspose.SVG API, вы можете быстро загружать, сохранять и конвертировать SVG, а также читать и перемещаться по элементам файла. API не зависит от любого другого программного обеспечения. Он позволяет работать с файлами SVG на всех платформах, поддерживающих .NET (Windows, Linux и macOS).
