Пространство имен содержит методы для навигации по элементам и атрибутам в XML-документе.

| Интерфейс | Описание |
| --- | --- |

| [IXPathExpression](./ixpathexpression) | Интерфейс` XPathExpression` представляет проанализированное и разрешенное выражение XPath. |
| [IXPathNamespace](./ixpathnamespace) | Интерфейс XPathNamespace возвращается интерфейсами XPathResult для представления типа узла пространства имен XPath, которого нет в DOM. |
| [IXPathNSResolver](./ixpathnsresolver) | ` XPathNSResolver` интерфейс разрешает` префикс` строки в выражения для должным образом привязаны к строкам` namespaceURI` . [`IXPathEvaluator`](../aspose.svg.dom.xpath/ixpathevaluator)может создать реализацию [`IXPathNSResolver`](../aspose.svg.dom.xpath/ixpathnsresolver)из узел, или интерфейс может быть реализован любым приложением. |
| [IXPathResult](./ixpathresult) | Интерфейс` XPathResult` представляет результат оценки выражения XPath 1.0 в контексте конкретного узла. Поскольку вычисление выражения XPath может привести к различным типам результатов, этот объект позволяет обнаруживать и управлять типом и значением результата. |
