**Aspose.Svg.Rendering**пространство имен состоит из многочисленных объектов рендеринга, а также соответствующих классов опций низкого уровня, которые отвечают за рендеринг документов/файлов в реализацию IDevice.

## Классы

| Учебный класс | Описание |
| --- | --- |
| [CssOptions](./cssoptions/) | Представляет параметры рендеринга CSS. |
| [Device](./device/) | Представляет базовый класс для реализации устройств рендеринга, которые используются для рисования графики в различных форматах и средах. |
| [Device&lt;TGraphicContext,TRenderingOptions&gt;](./device-2/) | Представляет базовый класс для реализации конкретных устройств рендеринга. |
| [GraphicContext](./graphiccontext/) | Содержит текущие параметры управления графикой. Эти параметры определяют глобальную структуру, в которой выполняются графические операторы. |
| [PageSetup](./pagesetup/) | Представляет объект настройки страницы, используемый для набора страниц вывода конфигурации. |
| [Renderer](./renderer/) | Представляет базовый класс для всех средств визуализации и реализации интерфейса IDisposable. |
| [Renderer&lt;TDocument&gt;](./renderer-1/) | Представляет абстрактный класс для всех средств визуализации. |
| [RenderingOptions](./renderingoptions/) | Представляет параметры рендеринга. |
| [SvgRenderer](./svgrenderer/) | Представляет средство визуализации документа SVG. |
| [TextInfo](./textinfo/) | Содержит информацию о отображаемом тексте. |
