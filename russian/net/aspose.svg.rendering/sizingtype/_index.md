## SizingType enumeration

Представляет перечисление типов размеров страницы.

```csharp
public enum SizingType
```

### Ценности

| Имя | Ценность | Описание |
| --- | --- | --- |
| FitContent | `0` | Изменение заданных размеров страницы в соответствии с размером содержимого, которое она содержит. |
| ScaleContent | `1` | Масштабирование размера контента в соответствии с заданным размером страницы. |
| Contain | `2` | Подгонка размера содержимого к размеру страницы с сохранением предпочтительного соотношения сторон, насколько это возможно. |
| Crop | `3` | Размещение содержимого на странице и обрезка всего, что выходит за пределы заданного размера страницы. |
