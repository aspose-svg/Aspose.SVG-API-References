## InputEvent.IsComposing property

true, если входное событие происходит как часть сеанса композиции, т. е. после события CompositionStart и перед соответствующим событием CompositionEnd. Неинициализированное значение этого атрибута ДОЛЖНО быть ложным.

```csharp
public bool IsComposing { get; }
```

### Стоимость имущества

` true` если этот экземпляр компонуется; в противном случае` false` .
