## InputEvent.Data property

Данные содержат значение символов, сгенерированных методом ввода. Это МОЖЕТ быть одиночный символ Unicode или непустая последовательность символов Unicode [Unicode]. Символы СЛЕДУЕТ нормализовать, как определено формой нормализации Unicode NFC, определенной в [UAX15]. Этот атрибут МОЖЕТ содержать пустую строку.

```csharp
public string Data { get; }
```
