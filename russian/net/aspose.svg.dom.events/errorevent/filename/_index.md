## ErrorEvent.FileName property

Атрибут имени файла должен возвращать значение, которым он был инициализирован. При создании объекта этот атрибут должен быть инициализирован пустой строкой. Он представляет собой абсолютный URL-адрес сценария, в котором изначально возникла ошибка.
