## KeyboardEvent.IsComposing property

true, если ключевое событие происходит как часть сеанса композиции, т. е. после события CompositionStart и до соответствующего события CompositionEnd. Неинициализированное значение этого атрибута ДОЛЖНО быть ложным.

```csharp
public bool IsComposing { get; }
```
