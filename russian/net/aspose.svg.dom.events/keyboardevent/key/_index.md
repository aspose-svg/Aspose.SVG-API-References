## KeyboardEvent.Key property

Ключ содержит значение нажатой клавиши. Если значение имеет печатное представление, оно ДОЛЖНО быть непустой строкой символов Unicode, соответствующей алгоритму определения значения ключа, определенному в этой спецификации. Если значение является управляющим ключом, который не имеет печатного представления, оно ДОЛЖНО быть одним из значений ключа, определенных в наборе значений ключа, как определено алгоритмом определения значения ключа. Реализации, которые не могут идентифицировать ключ, ДОЛЖНЫ использовать значение ключа Unidentified.

```csharp
public string Key { get; }
```
