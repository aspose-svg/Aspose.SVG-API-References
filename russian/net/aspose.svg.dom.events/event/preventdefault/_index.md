## Event.PreventDefault method

Если событие можно отменить, метод`PreventDefault`используется для обозначения того, что событие должно быть отменено, означает, что никакие действия по умолчанию, обычно выполняемые реализацией в результате события, не будут выполняться.

```csharp
public void PreventDefault()
```

### Примечания

Если на любом этапе потока событий`PreventDefault`Метод вызывается, событие отменяется. Никаких действий по умолчанию, связанных с этим событием, не будет. Вызов этого метода для неотменяемого события не имеет никакого эффекта. После того, как`PreventDefault`был вызван, он будет действовать до конца распространения события. Этот метод можно использовать на любой стадии потока событий.
