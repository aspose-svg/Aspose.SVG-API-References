## Event.PreventDefault method

Если событие можно отменить,`PreventDefault` метод используется для обозначения того, что событие должно быть отменено, означает, что любое действие по умолчанию, обычно выполняемое реализацией в результате события, не произойдет.

```csharp
public void PreventDefault()
```

### Примечания

Если на любом этапе потока событий`PreventDefault`вызывается метод, событие отменяется. Любое действие по умолчанию, связанное с событием, не выполняется. Вызов этого метода для неотменяемого события не имеет никакого эффекта. Один раз`PreventDefault` был вызван, он будет действовать до конца распространения события. Этот метод можно использовать на любом этапе потока событий.
