**Aspose.Svg.Drawing** пространство имен содержит объекты и интерфейсы для указания измерения и единиц измерения, а также объекты атрибутов рисования, такие как кисти, цвета и шрифты.

## Классы

| Учебный класс | Описание |
| --- | --- |
| [Angle](./angle/) | Тип данных угла |
| [Color](./color/) | Класс Color позволяет задавать цвета как значения Red-Green-Blue (RGB), значения Hue-Saturation-Luminosity (HSL), значения Hue-Saturation-Value (HSV), Hue-Whiteness-Blackness (HWB) ) значения, значения яркости-AB (LAB), значения Luminance-Chroma-Hue (LCH), значения Cyan-Magenta-Yellow-Key (CMYK), значения естественных цветов (NCOL), или с названием цвета . Также доступен альфа-канал для обозначения прозрачности. |
| [Dimension](./dimension/) | Предоставляет базовый класс для измерений. Общий термин «размер» относится к числу с присоединенной к нему единицей и обозначается[`UnitType`](../aspose.svg.drawing/unittype/) . |
| [Frequency](./frequency/) | Единица "частоты". |
| [Length](./length/) | Представляет единицу измерения длины. |
| [LengthOrAuto](./lengthorauto/) | Представляет контейнер для длины хранения или «авто» единиц.. |
| [Margin](./margin/) | Представляет поля страницы. |
| [Numeric](./numeric/) | Предоставляет базовый класс для числовых типов. |
| [Page](./page/) | Представляет объект страницы, используемый для страницы вывода конфигурации. Значение пробела для размера страницы: A4 (210x297 мм) |
| [Resolution](./resolution/) | Представляет единицу разрешения. |
| [Size](./size/) | Сохраняет значения, которые определяют высоту и ширину. |
| [Time](./time/) | Представляет единицу времени. |
| [Unit](./unit/) | Предоставляет базовый класс для единиц измерения. |
| [UnitType](./unittype/) | Определяет единицу измерения. |
