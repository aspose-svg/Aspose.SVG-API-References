## IColorComponents.ToString method

Преобразует компоненты цвета в строковое представление.

```csharp
public string ToString(bool includeAlpha, bool asDecimal, int digits)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| includeAlpha | Boolean | Указывает, добавлять ли альфа-компонент. |
| asDecimal | Boolean | Указывает, сохраняются ли компоненты цвета в виде десятичного числа или в процентах. |
| digits | Int32 | Устанавливает точность округления для компонентов цвета. |

### Возвращаемое значение

Строковое представление компонентов цвета.
