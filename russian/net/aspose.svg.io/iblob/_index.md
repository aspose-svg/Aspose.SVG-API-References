## IBlob interface

Объект Blob ссылается на последовательность байтов и имеет атрибут размера, который представляет собой общее количество байтов в последовательности байтов, и атрибут типа, который представляет собой строку в кодировке ASCII в нижний регистр, представляющий тип носителя последовательности байтов.

| --- | --- |
| [Size](../../aspose.svg.io/iblob/size) { get; } | Возвращает размер последовательности байтов в байтах. При получении соответствующие пользовательские агенты должны возвращать общее количество байтов, которое может быть прочитано объектом FileReader или FileReaderSync, или 0, если в большом двоичном объекте нет байтов для чтения. |
| [Type](../../aspose.svg.io/iblob/type) { get; } | Строка в кодировке ASCII в нижнем регистре, представляющая тип мультимедиа BLOB-объекта. При получении пользовательские агенты должны возвращать тип Blob в виде строки в кодировке ASCII в нижнем регистре, таким образом, чтобы при преобразовании в последовательность байтов это был анализируемый тип MIME, или пустая строка – 0 байт – если тип определить невозможно. |
