## IBlob.Slice method

Возвращает новый объект Blob с байтами в диапазоне от необязательного начального параметра до необязательного конечного параметра, , и с атрибутом типа, который является значением необязательный параметр contentType.

```csharp
public IBlob Slice(ulong start, ulong end, string contentType)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| start | UInt64 | Параметр представляет собой значение начальной точки среза |
| end | UInt64 | Параметр представляет собой значение для начальной точки фрагмента |
| contentType | String | Параметр представляет собой тип носителя Blob |

### Возвращаемое значение

Возвращает новый объект Blob с диапазоном байтов от необязательного начального параметра до необязательного конечного параметра, но не включая его, и с атрибутом типа, который является значением необязательного параметра contentType.
