## OutputStream.Write method

Записывает последовательность байтов в обернутый поток output и продвигает текущую позицию в этом потоке на количество записанных байтов .

```csharp
public override void Write(byte[] buffer, int offset, int count)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| buffer | Byte[] | Массив байтов. Этот метод копирует количество байтов из буфера в обернутый поток output . |
| offset | Int32 | Отсчитываемое от нуля смещение байтов в буфере, с которого начинается копирование байтов в обернутый поток output . |
| count | Int32 | Количество байтов, которые будут записаны в обернутый поток вывода. |
