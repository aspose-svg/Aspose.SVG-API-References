В дополнение к стандартным инструментам IO (ввод-вывод)  **Aspose.Svg.IO** namespace содержит вспомогательные классы и интерфейсы.

## Классы

| Учебный класс | Описание |
| --- | --- |
| [FileCreateStreamProvider](./filecreatestreamprovider/) | ПредставляетFileStream реализация для предоставления потоков на устройства рендеринга. |
| [LocalFileSystemStorage](./localfilesystemstorage/) | Хранилище файловой системы. |
| [OutputStream](./outputstream/) | Суррогатный поток обертывает реальный выходной поток и контролирует доступ к нему. [`OutputStream`](../aspose.svg.io/outputstream/) содержит данные URI, описывающие расположение выходного потока. |
| [OutputStreamContext](./outputstreamcontext/) | Контекст для инициализации выходного потока. |
## Интерфейсы

| Интерфейс | Описание |
| --- | --- |
| [IBlob](./iblob/) | Объект Blob ссылается на последовательность байтов и имеет атрибут размера, который представляет собой общее количество байтов в последовательности байтов, и атрибут типа, который представляет собой строку в кодировке ASCII в нижнем регистре, представляющую тип носителя последовательности байтов. . |
| [ICreateStreamProvider](./icreatestreamprovider/) | Представляет интерфейс, который может быть реализован классами, передающими потоки на устройства рендеринга. |
| [IFile](./ifile/) | Объект File — это объект Blob с атрибутом имени, который представляет собой строку; он может быть создан в веб-приложении с помощью конструктора или является ссылкой на последовательность байтов из файла базовой файловой системы (ОС). |
| [IFileList](./ifilelist/) | Представляет интерфейс для списка файлов. |
| [IOutputStorage](./ioutputstorage/) | Обеспечивает создание и управление выходными потоками[`OutputStream`](../aspose.svg.io/outputstream/) . |
