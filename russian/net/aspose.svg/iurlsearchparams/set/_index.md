## IUrlSearchParams.Set method

Устанавливает значение первой найденной пары "имя-значение" в указанное значение и удаляет остальные. Если не найдено ни одной пары "имя-значение" с указанным именем, новая будет добавлена в список.

```csharp
public void Set(string name, string value)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| name | String | Ключ, который будет использоваться для поиска пар имя-значение. |
| value | String | Значение, которое будет установлено. |
