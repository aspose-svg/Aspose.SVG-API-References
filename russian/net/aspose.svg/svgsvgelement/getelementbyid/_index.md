## SVGSVGElement.GetElementById method

Выполняет поиск в этом фрагменте документа SVG (т. е. поиск ограничен подмножеством дерева документа) элемента, идентификатор которого задан elementId. Если элемент найден, этот элемент возвращается. Если такого элемента не существует, возвращает null. Поведение не определено, если этот идентификатор имеет более одного элемента.

```csharp
public Element GetElementById(string elementId)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| elementId | String | Уникальное значение идентификатора элемента. |

Соответствующий элемент.
