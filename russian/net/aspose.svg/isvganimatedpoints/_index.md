## ISVGAnimatedPoints interface

Интерфейс SVGAnimatedPoints поддерживает элементы с атрибутом 'points', который содержит список значений координат и поддерживает возможность анимирования этого атрибута. Кроме того, атрибут 'points' в исходном элементе, доступ к которому осуществляется через XML DOM (например, с помощью вызова метода getAttribute()), будет отражать любые изменения, внесенные в точки.
