Когда элемент «svg» встроен как компонент документа из другого пространства имен, например, когда элемент встроен в документ XHTML [XHTML], тогда объект SVGDocument не будет существовать; вместо этого корневым объектом в иерархии объектов документа будет объект Document другого типа, например объект HTMLDocument.
