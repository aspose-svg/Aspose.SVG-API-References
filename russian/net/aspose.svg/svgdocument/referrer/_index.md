## SVGDocument.Referrer property

Возвращает URI страницы, которая ссылается на эту страницу. Значение представляет собой пустую строку, если пользователь перешел на страницу напрямую (не по ссылке, а, например, через закладку).

```csharp
public string Referrer { get; }
```
