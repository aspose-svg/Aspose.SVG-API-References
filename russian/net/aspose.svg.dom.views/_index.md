**Aspose.Svg.Dom.Views** пространство имен описывает интерфейсы AbstractView и DocumentView в соответствии с спецификацией представлений объектной модели документа.

## Интерфейсы

| Интерфейс | Описание |
| --- | --- |
| [IAbstractView](./iabstractview/) | Базовый интерфейс, из которого должны быть получены все виды. |
| [IDocumentView](./idocumentview/) | Интерфейс DocumentView реализуется объектами Document в реализациях DOM, поддерживающих представления DOM. Он предоставляет атрибут для получения представления документа по умолчанию. |
