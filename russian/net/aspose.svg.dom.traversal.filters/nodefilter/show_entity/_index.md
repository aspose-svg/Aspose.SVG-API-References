## NodeFilter.SHOW_ENTITY field

Показать узлы Entity. Это имеет смысл только при создании итератора или обходчика дерева с узлом Entity в качестве корня ; в данном случае это означает, что узел Entity появится в первой позиции обхода. Поскольку объекты не являются частью дерева документа, они не отображаются, когда перемещается по дереву документа.

```csharp
public const long SHOW_ENTITY;
```
