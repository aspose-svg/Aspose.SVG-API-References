Пространство имен состоит из классов для интерактивности на основе событий, таких как временные события и масштабирование.

## Классы

| Учебный класс | Описание |
| --- | --- |
| [SVGZoomEvent](./svgzoomevent) | Событие масштабирования происходит, когда пользователь инициирует действие, которое приводит к изменению масштаба текущего представления фрагмента документа SVG. Обработчики событий распознаются только для элементов svg. |
| [TimeEvent](./timeevent) | Интерфейс TimeEvent предоставляет конкретную контекстуальную информацию, связанную с событиями Time. Возможны следующие типы событий: beginEvent, endEvent и RepeatEvent. |
