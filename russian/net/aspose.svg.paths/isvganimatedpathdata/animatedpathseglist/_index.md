## ISVGAnimatedPathData.AnimatedPathSegList property

Предоставляет доступ к текущему анимированному содержимому атрибута 'd' в форме, которая полностью соответствует синтаксису SVG. Если данный атрибут или свойство анимируется, содержит текущее анимированное значение атрибута или свойства, а сам объект и его содержимое доступны только для чтения. Если данный атрибут или свойство в настоящее время не анимируются, содержит то же значение, что и pathSegList.

```csharp
public SVGPathSegList AnimatedPathSegList { get; }
```
