## ICSSRule interface

Интерфейс CSSRule — это абстрактный базовый интерфейс для любого типа оператора CSS. Сюда входят как наборы правил, так и at-правила. Ожидается, что реализация сохранит все правила, указанные в таблице стилей CSS, даже если правило не распознано синтаксическим анализатором. Нераспознанные правила представлены с помощью!:ICSSUnknownRule интерфейс.
