## ICSSRule interface

Интерфейс CSSRule является абстрактным базовым интерфейсом для любого типа операторов CSS. Сюда входят как наборы правил, так и at-правила. Ожидается, что реализация сохранит все правила, указанные в таблице стилей CSS, даже если правило не распознано синтаксическим анализатором. Нераспознанные правила представлены с помощью интерфейса!:ICSSUnknownRule.
