## ICSSRule.ParentRule property

Если это правило содержится внутри другого правила (например, правило стиля внутри блока @media), это правило является содержащим. Если это правило не вложено в какие-либо другие правила, возвращается null.

```csharp
public ICSSRule ParentRule { get; }
```
