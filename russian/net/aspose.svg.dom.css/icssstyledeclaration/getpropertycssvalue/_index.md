## ICSSStyleDeclaration.GetPropertyCSSValue method

Используется для получения объектного представления значения свойства CSS, если оно было явно задано в этом блоке объявления. Этот метод возвращает null, если свойство является сокращенным свойством. К сокращенным значениям свойств можно получить доступ и изменить их только как строки с помощью методов getPropertyValue и setProperty.

```csharp
public CSSValue GetPropertyCSSValue(string propertyName)
```
