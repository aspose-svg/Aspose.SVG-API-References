## ICSSImportRule.StyleSheet property

Таблица стилей, на которую ссылается это правило, если она была загружена. Значение этого атрибута равно null, если таблица стилей еще не загружена или не будет загружена (например, если таблица стилей предназначена для типа носителя, не поддерживаемого пользовательским агентом).

```csharp
public ICSSStyleSheet StyleSheet { get; }
```
