## ImageVectorizerConfiguration.ImageSizeLimit property

Получает или задает максимальный размер изображения, определяемый умножением ширины и высоты изображения. Размер изображения будет масштабироваться на основе этого свойства. Значение по умолчанию: 1800000.

```csharp
public int ImageSizeLimit { get; set; }
```
