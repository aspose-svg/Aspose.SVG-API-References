## NodeList indexer

Метод возвращает индексный элемент в коллекции. Если индекс больше или равен количеству узлов в списке, возвращается null.

```csharp
public abstract Node this[int index] { get; }
```

| Параметр | Описание |

| index | Индекс. |

### Возвращаемое значение
