## DOMTokenList.Replace method

Заменяет существующий токен новым токеном. Ничего не делает, если первый токен не существует.

```csharp
public bool Replace(string token, string newToken)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| token | String | Токен, который вы хотите заменить. |
| newToken | String | Токен, которым вы хотите заменить старый токен. |

### Возвращаемое значение

логический`истинный` если токен был найден и заменен,`ЛОЖЬ` в противном случае.
