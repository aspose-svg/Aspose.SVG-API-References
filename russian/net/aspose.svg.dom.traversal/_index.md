Пространство имен **Aspose.Svg.Dom.Traversal** содержит методы, которые создают итераторы и обходчики дерева для навигации между элементами и проходят узел и его дочерние элементы в порядке документа.

## Интерфейсы

| Интерфейс | Описание |
| --- | --- |
| [IDocumentTraversal](./idocumenttraversal) | DocumentTraversal содержит методы, которые создают итераторы и обходчики дерева для обхода узла и его дочерних элементов в порядке документа (сначала глубина , обход в предварительном порядке, что эквивалентно порядку, в котором начальные теги встречаются в текстовом представлении документа). В DOM , которые поддерживают функцию обхода, DocumentTraversal будет реализовываться теми же объектами, которые реализуют интерфейс документа. |
| [IElementTraversal](./ielementtraversal) | Интерфейс ElementTraversal представляет собой набор атрибутов только для чтения, которые позволяют автору легко перемещаться между элементами в документе. В соответствующих реализациях Element Traversal все объекты, реализующие Element, должны также реализовывать интерфейс ElementTraversal. |
| [INodeFilter](./inodefilter) | Фильтры — это объекты, умеющие «отфильтровывать» узлы. Если NodeIterator или TreeWalker задан NodeFilter, он применяет фильтр до того, как вернет следующий узел. Если фильтр говорит принять узел, логика обхода возвращает его; в противном случае обход ищет следующий узел и делает вид, что отвергнутого узла там не было. |
| [INodeIterator](./inodeiterator) | Итераторы используются для перехода через набор узлов, например, набор узлов в NodeList, поддерево документа, управляемое конкретным узлом, результаты запроса или любой другой набор узлов. Набор итерируемых узлов определяется реализацией NodeIterator. Уровень DOM 2 определяет единственную реализацию NodeIterator для обхода поддерева документа в порядке порядка документа. Экземпляры этих итераторов создаются вызовом DocumentTraversal .createNodeIterator(). |

Набор итерируемых узлов определяется реализацией NodeIterator. Уровень DOM 2 определяет единственную реализацию NodeIterator для обхода поддерева документа в порядке порядка документа. Экземпляры этих итераторов создаются вызовом DocumentTraversal .createNodeIterator(). |
| [ITreeWalker](./itreewalker) | Объекты TreeWalker используются для навигации по дереву документа или поддереву с использованием представления документа, определяемого их флагами whatToShow и фильтром (если есть). Любая функция, которая выполняет навигацию с использованием TreeWalker, будет автоматически поддерживать любое представление, определенное TreeWalker. |
