## Node.InsertBefore method

Вставляет узел перед существующим дочерним узлом. Если дочерний элемент равен нулю, вставьте узел в конец списка дочерних элементов. Если дочерний элемент является объектом DocumentFragment, все его дочерние элементы вставляются в том же порядке перед дочерним. Если дочерний элемент уже находится в дереве, он сначала удаляется.

```csharp
public Node InsertBefore(Node node, Node child)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| node | Node | Новый потомок. |
| child | Node | Дочерний элемент ссылки. |

### Возвращаемое значение

Возвращает вставленный узел
