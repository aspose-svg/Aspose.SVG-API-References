## Node.InsertBefore method

Вставляет узел перед существующим дочерним узлом. Если дочерний элемент имеет значение null, вставьте узел в конец списка дочерних элементов. Если дочерний элемент является объектом DocumentFragment, все его дочерние элементы вставляются в том же порядке перед дочерним элементом. Если дочерний элемент уже есть в дереве, он сначала удаляется.

```csharp
public Node InsertBefore(Node node, Node child)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| node | Node | Новый ребенок. |
| child | Node | Ребенок-рефери. |

### Возвращаемое значение

Возвращает вставленный узел
