## Document.XmlVersion property

Атрибут, указывающий, как часть объявления XML, номер версии этого документа. Если объявления нет и если этот документ поддерживает функцию «XML», значение равно «1,0». Если этот документ не поддерживает функцию «XML», значение всегда равно null.

```csharp
public string XmlVersion { get; set; }
```

### Стоимость имущества

Версия XML.
