## Document.StyleSheets property

Список, содержащий все таблицы стилей, явно связанные с документом или встроенные в него. Для HTML-документов сюда входят внешние таблицы стилей, включаемые через HTML-элемент LINK, и встроенные элементы STYLE.

```csharp
public IStyleSheetList StyleSheets { get; }
```
