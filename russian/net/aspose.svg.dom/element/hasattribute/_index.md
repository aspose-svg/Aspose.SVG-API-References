## Element.HasAttribute method

Возвращает значение true, если атрибут с заданным именем указан в этом элементе или имеет значение по умолчанию, в противном случае — значение false.

```csharp
public bool HasAttribute(string name)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| name | String | Имя атрибута. |

### Возвращаемое значение

`истинный` если указанное имя имеет атрибут; в противном случае,`ЛОЖЬ` .
