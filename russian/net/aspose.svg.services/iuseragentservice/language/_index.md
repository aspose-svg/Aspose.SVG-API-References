## IUserAgentService.Language property

`Language` указывает основной язык для содержимого элемента и для любого из атрибутов элемента, содержащих текст. Его значение должно быть действительным BCP 47 ( языковой тег или пустая строка. Установка для атрибута пустой строки означает, что основной язык неизвестен.
