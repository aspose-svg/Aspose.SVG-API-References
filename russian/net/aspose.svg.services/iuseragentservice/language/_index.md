## IUserAgentService.Language property

Параметр`Language`указывает основной язык содержимого элемента и любых атрибутов элемента, содержащих текст. Его значение должно быть допустимым языковым тегом BCP 47 () или пустой строкой. Установка для атрибута пустой строки означает, что основной язык неизвестен.

public string Language { get; set; }
