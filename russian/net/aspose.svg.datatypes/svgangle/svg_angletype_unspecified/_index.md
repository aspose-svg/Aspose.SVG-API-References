## SVGAngle.SVG_ANGLETYPE_UNSPECIFIED field

Тип единицы измерения не указан (т. е. указано безразмерное значение). Для углов безразмерное значение обрабатывается так же, как если бы были указаны градусы.

```csharp
public const ushort SVG_ANGLETYPE_UNSPECIFIED;
```
