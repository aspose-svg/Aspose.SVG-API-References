## SVGTransform.SetMatrix method

Задает тип преобразования SVG_TRANSFORM_MATRIX с матрицей параметров, определяющей новое преобразование. Копируются значения из матрицы параметров, параметр матрицы не заменяет SVGTransform::matrix.

```csharp
public void SetMatrix(SVGMatrix matrix)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| matrix | SVGMatrix | Новая матрица трансформации. |

| исключение | условие |
