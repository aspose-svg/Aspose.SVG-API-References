## ResourceHandlingOptions.MaxHandlingDepth property

Получает или задает максимальную глубину обрабатываемых страниц. Глубина 1 означает, что будут обрабатываться только страницы, на которые есть прямые ссылки из сохраненного документа. Установка этого свойства в -1 приведет к обработке всех страниц. Значение по умолчанию: 0. .

```csharp
public int MaxHandlingDepth { get; set; }
```
