## IWindow.Prompt method

Отображает приглашение модального текстового поля с заданным сообщением, ожидает, пока пользователь его закроет, и возвращает значение, введенное пользователем. Если пользователь отменяет приглашение, вместо этого возвращается null. Если присутствует второй аргумент, то данное значение используется по умолчанию.

```csharp
public string Prompt(string message, string @default)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| message | String | Сообщение. |
| default | String | По умолчанию. |

### Возвращаемое значение

Возвращает значение, которое ввел пользователь
