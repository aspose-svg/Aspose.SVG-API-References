**Aspose.Svg.Window** пространство имен связано с объектом окна , который представляет окно , содержащее документ DOM. Он включает в себя классы и интерфейсы , связанные с активным адресом документа и историей просмотров.

## Классы

| Учебный класс | Описание |
| --- | --- |
| [Location](./location/) | Объекты Location предоставляют представление адреса активного документа контекста просмотра своего документа и позволяют изменять текущую запись истории сеанса контекста просмотра, добавляя или заменяя записи в объекте истории. |
## Интерфейсы

| Интерфейс | Описание |
| --- | --- |
| [IWindow](./iwindow/) | Объект окна представляет собой окно, содержащее документ DOM. |
| [IWindowEventHandlers](./iwindoweventhandlers/) | Представляет интерфейс, который должен быть унаследован окном object |
| [IWindowTimers](./iwindowtimers/) | Позволяет авторам планировать обратные вызовы на основе таймера. |
