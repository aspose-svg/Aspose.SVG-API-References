## IWindowTimers.SetTimeout method

```csharp
public int SetTimeout(object handler, int timeout, params object[] args)
```

| Параметр | Тип | Описание |

| handler | Object | Обработчик. |
| timeout | Int32 | Тайм-аут. |
| args | Object[] | Аргументы. |
