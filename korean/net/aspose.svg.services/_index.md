**Aspose.Svg.Services** 네임스페이스는 서비스 구현을 위한 프로토콜로 인터페이스를 포함합니다.

## 인터페이스

| 상호 작용 | 설명 |
| --- | --- |
| [IDeviceInformationService](./ideviceinformationservice/) | 환경을 설명하는 인터페이스[`Document`](../aspose.svg.dom/document/) 사용자에게 표시됩니다. |
| [INetworkService](./inetworkservice/) | 네트워크 작업을 위한 인터페이스를 제공합니다. |
| [IRuntimeService](./iruntimeservice/) | 이 서비스는 런타임 관련 속성을 구성하는 데 사용됩니다. |
| [IService](./iservice/) | 서비스 구현을 위한 기본 인터페이스를 정의합니다. |
| [IServiceContainer](./iservicecontainer/) | 서비스를 위한 컨테이너를 제공합니다. |
| [IServiceProvider](./iserviceprovider/) | 서비스에 대한 변경 불가능한 컨테이너를 제공합니다. |
| [IUserAgentService](./iuseragentservice/) | 사용자 에이전트 환경을 설명하는 인터페이스입니다. |
