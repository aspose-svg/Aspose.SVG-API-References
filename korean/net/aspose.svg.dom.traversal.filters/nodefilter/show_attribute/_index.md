## NodeFilter.SHOW_ATTRIBUTE field

Attr 노드를 표시합니다. 이는 속성 노드를 루트로 사용하여 반복자 또는 트리 워커를 생성할 때만 의미가 있습니다. 이 경우 속성 노드 가 반복 또는 탐색의 첫 번째 위치에 나타납니다. 속성은 다른 노드의 자식이 아니므로 문서 트리를 탐색할 때 나타나지 않습니다.

```csharp
public const long SHOW_ATTRIBUTE;
```
