## NodeFilter.SHOW_NOTATION field

표기법 노드를 표시합니다. 이는 Notation 노드를 루트로 사용하여 반복자 또는 트리 워커를 생성할 때만 의미가 있습니다. 이 경우 Notation 노드가 탐색의 첫 번째 위치에 나타납니다. 표기법은 문서 트리의 일부가 아니므로 문서 트리를 탐색할 때 표시되지 않습니다.

```csharp
public const long SHOW_NOTATION;
```
