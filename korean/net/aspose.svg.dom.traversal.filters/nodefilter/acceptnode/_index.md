## NodeFilter.AcceptNode method

지정된 노드가 a TreeWalker 또는 NodeIterator의 논리적 보기에 표시되는지 여부를 테스트합니다. 이 function 는 TreeWalker 및 NodeIterator의 구현에 의해 호출됩니다. 일반적으로 from 사용자 코드에서 직접 호출되지 않습니다. (same 필터를 사용하여 자체 애플리케이션 로직을 안내하려는 경우 그렇게 할 수 있습니다.)

```csharp
public abstract short AcceptNode(Node n)
```

| 모수 | 유형 | 설명 |
| --- | --- | --- |
| n | Node | 필터를 통과하는지 여부를 확인하는 노드입니다. |

### 반환 값

위에 정의된 대로 노드가 수락되는지, 거부되는지, 건너뛰는지 여부를 결정하는 상수입니다.
