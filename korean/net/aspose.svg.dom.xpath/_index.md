네임스페이스에는 XML 문서의 요소와 특성을 탐색하는 메서드가 포함되어 있습니다.

## 인터페이스

| 상호 작용 | 설명 |
| --- | --- |
| [IXPathEvaluator](./ixpathevaluator/) | XPath 식의 평가는 다음에 의해 제공됩니다.[`IXPathEvaluator`](../aspose.svg.dom.xpath/ixpathevaluator/) . |
| [IXPathExpression](./ixpathexpression/) | `XPath 표현식`인터페이스는 구문 분석되고 해결된 XPath 식을 나타냅니다. |
| [IXPathNamespace](./ixpathnamespace/) | XPathNamespace 인터페이스는 DOM에 없는 XPath 네임스페이스 노드 유형을 나타내기 위해 XPathResult 인터페이스에 의해 반환됩니다. |
| [IXPathNSResolver](./ixpathnsresolver/) | `XPathNS리졸버` 인터페이스 허가`접두사` 의 문자열은 적절하게 바인딩될 표현식입니다.`네임스페이스URI` 문자열. [`IXPathEvaluator`](../aspose.svg.dom.xpath/ixpathevaluator/) 의 구현을 구성할 수 있습니다.[`IXPathNSResolver`](../aspose.svg.dom.xpath/ixpathnsresolver/) 노드에서 또는 인터페이스가 응용 프로그램에 의해 구현될 수 있습니다. |
| [IXPathResult](./ixpathresult/) | `XPath 결과` 인터페이스는 특정 노드의 컨텍스트 내에서 XPath 1.0 식의 평가 결과를 나타냅니다. XPath 식의 평가 는 다양한 결과 유형을 초래할 수 있으므로 이 개체를 사용하면 결과의 유형과 값을 검색하고 조작할 수 있습니다. |
