## IXPathNamespace.OwnerElement property

[`Element`](../../../aspose.svg.dom/element/)네임스페이스가 요청되었을 때 범위 내에 있던 네임스페이스입니다. 이것은 문서가 와 같이 변경되어 네임스페이스가 해당 요소의 범위를 벗어나고 이 노드가 더 이상 XPath에 의해 에서 발견되지 않는 경우에도 반환된 네임스페이스 노드에서 변경되지 않습니다.

```csharp
public Element OwnerElement { get; }
```
