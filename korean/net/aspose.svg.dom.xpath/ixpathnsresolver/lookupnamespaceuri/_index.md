## IXPathNSResolver.LookupNamespaceURI method

지정된 네임스페이스 접두사와 관련된 네임스페이스 URI를 찾습니다. XPath 평가자는 다음과 함께 이것을 호출해서는 안 됩니다.`없는` 또는 빈 인수, 이 작업을 수행한 결과가 정의되지 않았기 때문입니다.

```csharp
public string LookupNamespaceURI(string prefix)
```

| 모수 | 유형 | 설명 |
| --- | --- | --- |
| prefix | String | 찾을 접두사입니다. |

### 반환 값

연결된 네임스페이스 URI를 반환하거나`없는` 없는 경우 가 발견됩니다.
