**Aspose.Svg.Dom.Attributes** 네임스페이스는 를 지정하거나 모든 개체(노드, 요소, ...) 기능 또는 동작을 한정하는 통합 속성 클래스를 포함합니다.

## 클래스

| 수업 | 설명 |
| --- | --- |
| [DOMConstructorAttribute](./domconstructorattribute/) | W3C에서 정의한 생성자를 지정합니다. |
| [DOMNameAttribute](./domnameattribute/) | W3C에서 정의한 공식 DOM 개체 이름을 지정합니다. |
| [DOMNamedPropertyGetterAttribute](./domnamedpropertygetterattribute/) | 메서드가 명명된 속성 getter로 사용되도록 지정합니다. |
| [DOMNoInterfaceObjectAttribute](./domnointerfaceobjectattribute/) | [NoInterfaceObject] 확장 속성이 인터페이스에 나타나면 ECMAScript 바인딩의 인터페이스에 대한 인터페이스 개체가 존재하지 않음을 나타냅니다. |
| [DOMNullableAttribute](./domnullableattribute/) | DOM 개체에 null 값을 할당할 수 있음을 지정합니다. |
| [DOMObjectAttribute](./domobjectattribute/) | W3C. 에서 정의한 이 특성으로 개체가 표시되도록 지정합니다. |
| [DOMTreatNullAsAttribute](./domtreatnullasattribute/) | 멤버 값의 null이 지정된 값으로 처리됨을 나타냅니다. |
