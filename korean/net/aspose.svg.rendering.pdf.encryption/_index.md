**Aspose.Svg.Rendering.Pdf.Encryption** namespace contains 암호, 암호화 알고리즘 등과 같은 pdf 암호화 세부 정보를 설명하는 데이터 클래스.

## 클래스

| 수업 | 설명 |
| --- | --- |
| [PdfEncryptionInfo](./pdfencryptioninfo/) | PDF 암호화에 대한 세부 정보를 포함합니다. |
## 열거

| 열거 | 설명 |
| --- | --- |
| [PdfEncryptionAlgorithm](./pdfencryptionalgorithm/) | 암호화 모드 열거형. 알고리즘 및 키 길이를 사용하여 설명합니다. 이 열거형은 기능을 더 늘릴 수 있도록 확장되었습니다. 이 열거형은 "베이스-투-코어" 패턴을 구현합니다. |
| [PdfPermissions](./pdfpermissions/) | 이 열거형은 pdf. 에 대한 사용자의 권한을 나타냅니다. |
