## IWindow.Top property

브라우징 컨텍스트 b에 있는 Document의 Window 객체에 대한 최상위 IDL 속성은 최상위 브라우징 컨텍스트의 WindowProxy 객체(최상위 브라우징 컨텍스트 자체인 경우 자체 WindowProxy 객체)를 반환해야 합니다. 그렇지 않으면 하나 또는 자체 WindowProxy 객체를 갖습니다(예: 분리된 중첩 브라우징 컨텍스트인 경우).

```csharp
public IWindow Top { get; }
```
