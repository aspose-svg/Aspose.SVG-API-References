## IWindow.Confirm method

지정된 메시지와 함께 모달 확인/취소 프롬프트를 표시하고 사용자가 메시지를 닫을 때까지 기다렸다가 사용자가 확인을 클릭하면 true를 반환하고 사용자가 취소를 클릭하면 false를 반환합니다.

```csharp
public bool Confirm(string message)
```

| 모수 | 유형 | 설명 |
| --- | --- | --- |
| message | String | 메시지. |

### 반환 값

사용자가 확인을 클릭하면 true를 반환하고 사용자가 취소를 클릭하면 false를 반환합니다.
