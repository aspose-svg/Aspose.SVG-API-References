**Aspose.Svg.Window** 네임스페이스는 DOM 문서를 포함하는 window 를 나타내는 for window 객체와 관련됩니다. 여기에는 활성 문서 주소 및 검색 기록과 관련된 클래스 및 인터페이스 가 포함됩니다.

## 클래스

| 수업 | 설명 |
| --- | --- |
| [Location](./location/) | 위치 개체는 해당 문서의 검색 컨텍스트의 활성 문서 주소를 나타내며 기록 개체의 항목을 추가하거나 교체하여 검색 컨텍스트 세션 기록의 현재 항목을 변경할 수 있습니다. |
## 인터페이스

| 상호 작용 | 설명 |
| --- | --- |
| [IWindow](./iwindow/) | 창 개체는 DOM 문서를 포함하는 창을 나타냅니다. |
| [IWindowEventHandlers](./iwindoweventhandlers/) | 는 window object 가 상속해야 하는 인터페이스를 나타냅니다. |
| [IWindowTimers](./iwindowtimers/) | 작성자가 타이머 기반 콜백을 예약할 수 있습니다. |
