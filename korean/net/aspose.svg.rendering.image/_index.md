**Aspose.Svg.Rendering.Image** 네임스페이스 제공 특정 장치 클래스 및 몇 가지 렌더링 옵션 래스터 형식으로 렌더링하는 클래스: jpeg, png, bmp, gif, tiff.

## 클래스

| 수업 | 설명 |
| --- | --- |
| [ImageDevice](./imagedevice/) | 래스터 형식으로의 렌더링을 나타냅니다: jpeg, png, bmp, gif, tiff. |
| [ImageRenderingOptions](./imagerenderingoptions/) | 에 대한 렌더링 옵션을 나타냅니다.[`ImageDevice`](../aspose.svg.rendering.image/imagedevice/) . 이 옵션은 출력 이미지 형식, 압축, 해상도 등을 지정하는 데 사용됩니다. |
| [TextOptions](./textoptions/) | 에 대한 텍스트 렌더링 옵션을 나타냅니다.[`ImageDevice`](../aspose.svg.rendering.image/imagedevice/) . |
## 열거

| 열거 | 설명 |
| --- | --- |
| [Compression](./compression/) | TIFF(Tagged Image File Format) 비트맵 이미지에 가능한 압축 체계를 지정합니다. |
| [ImageFormat](./imageformat/) | 이미지의 파일 형식을 지정합니다. |
