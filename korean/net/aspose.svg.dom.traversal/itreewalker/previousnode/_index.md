## ITreeWalker.PreviousNode method

TreeWalker를 현재 노드를 기준으로 문서 순서로 이전에 보이는 노드로 이동하고 new 노드를 반환합니다. 현재 노드에 이전 노드가 없거나 for previousNode 검색이 the TreeWalker의 루트 노드에서 위로 올라가려고 하면 null을 반환하고 현재 노드를 유지합니다.

```csharp
public Node PreviousNode()
```

### 반환 값

새 노드 또는 현재 노드에 TreeWalker의 논리적 보기에 이전 노드가 없는 경우 null입니다.
