## ITreeWalker.NextNode method

현재 노드를 기준으로 document 순서로 TreeWalker를 다음 가시 노드로 이동하고 새 노드를 반환합니다. 만약 the 현재 노드가 다음 노드가 없거나, nextNode에 대한 검색이 TreeWalker의 root 노드에서 위로 단계적으로 를 시도하면 null을 반환하고 현재 노드를 유지합니다.

```csharp
public Node NextNode()
```

### 반환 값

새 노드 또는 현재 노드에 TreeWalker의 논리적 보기에서 다음 노드가 없는 경우 null입니다.
