## ITraversal.WhatToShow property

이 속성은 반복자를 통해 표시되는 노드 유형을 결정합니다. 사용 가능한 상수 세트는 NodeFilter 인터페이스에서 정의됩니다. 에 의해 승인되지 않은 노드는 whatToShow를 건너뛸 수 있지만 그 자식은 여전히 고려될 수 있습니다. 이 건너뛰기는 필터(있는 경우 )보다 우선합니다.

```csharp
public long WhatToShow { get; }
```
