## ResourceHandlingOptions.MaxHandlingDepth property

처리할 페이지의 최대 깊이를 가져오거나 설정합니다. 깊이 1은 저장된 문서에서 직접 참조되는 페이지만 처리됨을 의미합니다. 이 속성을 -1로 설정하면 모든 페이지가 처리됩니다. 기본값은 0. 입니다.

```csharp
public int MaxHandlingDepth { get; set; }
```
