## Toggle(string, bool?) {#toggle_1}

토큰이 있으면 목록에서 제거하고, 없으면 목록에 토큰을 추가합니다.

```csharp
public bool Toggle(string token, bool? force)
```

| 모수 | 유형 | 설명 |
| --- | --- | --- |
| token | String | 전환할 토큰입니다. |
| force | Nullable`1 | 포함된 경우 토글을 단방향 작업으로 전환하는 부울입니다. false로 설정하면 토큰이 제거만 되고 추가되지는 않습니다. true로 설정하면 토큰이 추가되기만 하고 제거되지는 않습니다. |

### 반환 값

호출 후 토큰이 목록에 있는지 여부를 나타내는 부울입니다.
