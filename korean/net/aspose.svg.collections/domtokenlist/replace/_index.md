## DOMTokenList.Replace method

기존 토큰을 새 토큰으로 바꿉니다. 첫 번째 토큰이 존재하지 않으면 아무 작업도 수행하지 않습니다.

```csharp
public bool Replace(string token, string newToken)
```

| 모수 | 유형 | 설명 |
| --- | --- | --- |
| token | String | 교체할 토큰입니다. |
| newToken | String | 이전 토큰을 대체할 토큰입니다. |

### 반환 값

부울`진실` 토큰이 발견되어 교체된 경우`거짓` 그렇지 않으면.
