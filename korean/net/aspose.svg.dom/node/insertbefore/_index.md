## Node.InsertBefore method

기존 자식 노드 자식 앞에 노드를 삽입합니다. child가 null이면 children 목록 끝에 노드를 삽입합니다. child가 DocumentFragment 객체이면 모든 자식이 동일한 순서로 child 앞에 삽입됩니다. 자식이 이미 트리에 있으면 먼저 제거됩니다.

```csharp
public Node InsertBefore(Node node, Node child)
```

| 모수 | 유형 | 설명 |
| --- | --- | --- |
| node | Node | 새 아이. |
| child | Node | 심판 자식. |
