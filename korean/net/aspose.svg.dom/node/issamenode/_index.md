## Node.IsSameNode method

이 노드가 주어진 노드와 동일한 노드인지 여부를 반환합니다. 이 메서드는 구현에서 반환된 두 개의 노드 참조가 동일한 개체를 참조하는지 여부를 확인하는 방법을 제공합니다. 두 노드 참조가 동일한 개체에 대한 참조인 경우 프록시를 통하더라도 참조는 완전히 상호 교환 가능하게 사용될 수 있으므로 모든 속성이 동일한 값을 가지며 두 참조에서 동일한 DOM 메서드를 호출하면 항상 정확히 동일한 효과가 발생합니다.

```csharp
public bool IsSameNode(Node otherNode)
```

| 모수 | 유형 | 설명 |
| --- | --- | --- |
| otherNode | Node | 다른 노드입니다. |

### 반환 값

`진실`if [같은 노드] [지정된 기타]; 그렇지 않으면,`거짓` .
