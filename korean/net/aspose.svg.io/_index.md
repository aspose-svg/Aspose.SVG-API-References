표준 IO(입출력) tools 외에도 **Aspose.Svg.IO** 네임스페이스에는 헬퍼 클래스와 인터페이스가 포함되어 있습니다.

## 클래스

| 수업 | 설명 |
| --- | --- |
| [FileCreateStreamProvider](./filecreatestreamprovider/) | 나타냅니다FileStream 렌더링 장치에 스트림을 제공하기 위한 구현. |
| [LocalFileSystemStorage](./localfilesystemstorage/) | 파일 시스템 스토리지. |
| [OutputStream](./outputstream/) | 서로게이트 스트림은 실제 출력 스트림을 래핑하고 이에 대한 액세스를 제어합니다. [`OutputStream`](../aspose.svg.io/outputstream/) 출력 스트림의 위치를 설명하는 URI 데이터를 포함합니다. |
| [OutputStreamContext](./outputstreamcontext/) | 출력 스트림 초기화를 위한 컨텍스트입니다. |
## 인터페이스

| 상호 작용 | 설명 |
| --- | --- |
| [IBlob](./iblob/) | Blob 객체는 바이트 시퀀스를 참조하며 바이트 시퀀스의 총 바이트 수인 크기 속성과 바이트 시퀀스의 미디어 유형을 나타내는 소문자 ASCII 인코딩 문자열인 유형 속성을 가집니다. . |
| [ICreateStreamProvider](./icreatestreamprovider/) | 렌더링 장치에 스트림을 제공하는 클래스로 구현할 수 있는 인터페이스를 나타냅니다. |
| [IFile](./ifile/) | 파일 개체는 문자열인 이름 특성을 가진 Blob 개체입니다. 생성자를 통해 웹 응용 프로그램 내에서 만들 수 있거나 기본(OS) 파일 시스템의 파일에서 바이트 시퀀스에 대한 참조입니다. |
| [IFileList](./ifilelist/) | 파일 목록에 대한 인터페이스를 나타냅니다. |
| [IOutputStorage](./ioutputstorage/) | 출력 스트림 생성 및 관리 제공[`OutputStream`](../aspose.svg.io/outputstream/) . |
