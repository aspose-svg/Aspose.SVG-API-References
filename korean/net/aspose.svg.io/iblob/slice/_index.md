## IBlob.Slice method

선택적인 시작 매개변수에서 선택적인 종료 매개변수인 까지의 바이트 범위와 선택적인 contentType 매개변수의 값인 type 속성을 가진 새 Blob 객체를 반환합니다.

```csharp
public IBlob Slice(ulong start, ulong end, string contentType)
```

| 모수 | 유형 | 설명 |
| --- | --- | --- |
| start | UInt64 | 매개변수는 슬라이스의 시작점에 대한 값입니다. |
| end | UInt64 | 매개변수는 슬라이스의 시작점에 대한 값입니다. |
| contentType | String | 매개변수는 Blob의 미디어 유형입니다. |

### 반환 값

선택적인 시작 매개변수에서 선택적인 끝 매개변수까지는 포함하지 않는 범위의 바이트와 선택적인 contentType 매개변수의 값인 유형 속성이 있는 새 Blob 객체를 반환합니다.
