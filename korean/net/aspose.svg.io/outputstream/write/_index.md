## OutputStream.Write method

래핑된 output 스트림에 바이트 시퀀스를 쓰고 이 스트림 내의 현재 위치를 쓰여진 바이트 수만큼 앞으로 이동합니다.

```csharp
public override void Write(byte[] buffer, int offset, int count)
```

| 모수 | 유형 | 설명 |
| --- | --- | --- |
| buffer | Byte[] | 바이트 배열입니다. 이 메서드는 버퍼에서 래핑된 output 스트림으로 count 바이트를 복사합니다. |
| offset | Int32 | 래핑된 output 스트림에 바이트 복사를 시작할 버퍼의 0부터 시작하는 바이트 오프셋입니다. |
| count | Int32 | 래핑된 출력 스트림에 쓸 바이트 수입니다. |
