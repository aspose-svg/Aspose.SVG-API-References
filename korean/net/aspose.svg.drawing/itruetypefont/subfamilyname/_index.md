## ITrueTypeFont.SubFamilyName property

글꼴 하위 패밀리 이름은 동일한 글꼴 패밀리 이름을 가진 그룹의 글꼴을 구별합니다. 스타일(기울임꼴, 기울임꼴) 및 두께(밝게, 굵게, 검은색 등)를 처리하는 것으로 가정합니다. 두께나 스타일에 특별한 차이가 없는 글꼴(예: 기울임꼴이 아닌 중간 두께, fsSelection 비트 6 설정)은 이 위치에 "Regular" 문자열을 저장해야 합니다.

```csharp
public string SubFamilyName { get; }
```

### 반환 값

글꼴의 하위 패밀리 이름
