## FromRgb(byte, byte, byte) {#fromrgb}

요청된 ged, 녹색, 파란색 값을 가진 새 색상을 반환합니다. 모든 색상 구성 요소는 0-255 범위에 있어야 합니다.

```csharp
public static Color FromRgb(byte red, byte green, byte blue)
```

| 모수 | 유형 | 설명 |
| --- | --- | --- |
| red | Byte | 색상의 빨간색 구성 요소를 나타내는 바이트입니다. |
| green | Byte | 색상의 녹색 구성 요소를 나타내는 바이트입니다. |
| blue | Byte | 색상의 파란색 구성 요소를 나타내는 바이트입니다. |
