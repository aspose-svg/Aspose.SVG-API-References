## Event.PreventDefault method

이벤트가 취소 가능한 경우`PreventDefault` 메서드는 이벤트가 취소됨을 나타내는 데 사용됩니다. 이벤트의 결과로 구현에서 일반적으로 수행하는 기본 작업이 발생하지 않음을 의미합니다.

```csharp
public void PreventDefault()
```

### 비고

이벤트 흐름의 어느 단계에서든`PreventDefault`메서드가 호출되면 이벤트가 취소됩니다. 이벤트와 관련된 모든 기본 작업은 발생하지 않습니다. 취소할 수 없는 이벤트에 대해 이 메서드를 호출해도 효과가 없습니다. 한 번`PreventDefault` 이벤트가 전파되는 동안 계속 유효합니다. 이 메서드는 이벤트 흐름의 모든 단계에서 사용할 수 있습니다.
