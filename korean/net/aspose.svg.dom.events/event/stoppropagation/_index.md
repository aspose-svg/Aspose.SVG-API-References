## Event.StopPropagation method

`StopPropagation` 방법은 이벤트 흐름 중에 이벤트의 추가 전파를 방지하는 데 사용됩니다.

```csharp
public void StopPropagation()
```

### 비고

이 메서드가 호출되면[`IEventListener`](../../ieventlistener/) 이벤트는 트리를 통해 전파되는 것을 중지합니다. 이벤트는 현재 시스템의 모든 리스너에 대한 발송을 완료합니다.[`IEventTarget`](../../ieventtarget/) 이벤트 흐름이 중지되기 전에. 이 방법은 이벤트 흐름의 모든 단계에서 사용할 수 있습니다.
