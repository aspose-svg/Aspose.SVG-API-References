## IDocumentCSS.GetOverrideStyle method

이 메서드는 지정된 요소 및 지정된 의사 요소에 대한 재정의 스타일 선언을 검색하는 데 사용됩니다.

```csharp
public ICSSStyleDeclaration GetOverrideStyle(Element elt, string pseudoElt)
```

| 모수 | 유형 | 설명 |
| --- | --- | --- |
| elt | Element | 스타일을 수정할 요소입니다. 이 매개변수는 null일 수 없습니다. |
| pseudoElt | String | 의사 요소 또는 없는 경우 null입니다. |

### 반환 값

재정의 스타일 선언
