## ICSSKeyframesRule.FindRule method

findRule 메서드는 전달된 키와 일치하는 키가 있는 규칙을 반환합니다. 해당 규칙이 없으면 null 값이 반환됩니다

```csharp
public ICSSKeyframeRule FindRule(string key)
```

| 모수 | 유형 | 설명 |
| --- | --- | --- |
| key | String | 찾을 규칙을 설명하는 키입니다. 키는 0과 1 사이의 숫자로 확인되어야 합니다. 그렇지 않으면 규칙이 무시됩니다. |

### 반환 값

발견된 규칙
