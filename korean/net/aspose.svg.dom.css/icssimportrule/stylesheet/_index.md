## ICSSImportRule.StyleSheet property

이 규칙이 참조하는 스타일 시트(로드된 경우). 스타일 시트가 아직 로드되지 않았거나 로드되지 않을 경우(예: 스타일 시트가 사용자 에이전트에서 지원하지 않는 미디어 유형용인 경우) 이 속성의 값은 null입니다.

```csharp
public ICSSStyleSheet StyleSheet { get; }
```
