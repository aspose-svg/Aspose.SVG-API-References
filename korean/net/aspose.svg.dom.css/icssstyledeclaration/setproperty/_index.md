## SetProperty(string, string) {#setproperty}

이 선언 블록 내에서 기본 우선 순위로 속성 값을 설정하는 데 사용됩니다. 기본 우선 순위는 "중요"하지 않습니다(예: String.Empty ).

```csharp
public void SetProperty(string propertyName, string value)
```

| 모수 | 유형 | 설명 |
| --- | --- | --- |
| propertyName | String | 속성의 이름입니다. |
| value | String | 가치. |
