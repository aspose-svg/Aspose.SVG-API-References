## ICSSRuleList indexer

item() 메서드로 CSS 규칙을 검색하는 데 사용됩니다(http://www.w3.org/TR/DOM-Level-2-Style/css.html#CSS-CSSRuleList). 이 컬렉션의 순서는 CSS 스타일 시트의 규칙 순서를 나타냅니다. 인덱스가 목록의 규칙 수보다 크거나 같으면 null을 반환합니다.

```csharp
public ICSSRule this[int index] { get; }
```

| 모수 | 설명 |
| --- | --- |
| index | 규칙 인덱스 |
