## SVGAngle.ValueAsString property

unitType으로 표현되는 단위의 문자열 값인 각도 값입니다. 이 속성을 설정하면 이 설정을 반영하도록 value, valueInSpecifiedUnits 및 unitType이 자동으로 업데이트됩니다.

```csharp
public string ValueAsString { get; set; }
```

### 자산 가치

문자열 값.
