## MimeType Inequality operator

두 Mime 유형을 비교하고 두 개가 일치하는지 나타내는 부울을 반환합니다.

```csharp
public static bool operator !=(MimeType a, MimeType b)
```

| 모수 | 유형 | 설명 |
| --- | --- | --- |
| a | MimeType | 비교할 첫 번째 Mime입니다. |
| b | MimeType | 비교할 두 번째 Mime입니다. |

### 반환 값

진실 두 우선 순위가 같지 않은 경우, 그렇지 않은 경우거짓 .
