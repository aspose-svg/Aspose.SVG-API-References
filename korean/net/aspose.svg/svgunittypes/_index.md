## SVGUnitTypes enumeration

SVGUnitTypes 열거형은 일반적으로 사용되는 상수 집합을 정의하며 SVGGradientElement, SVGPatternElement, SVGClipPathElement, SVGMaskElement 및 SVGFilterElement에서 사용하는 기본 인터페이스입니다.

```csharp
public enum SVGUnitTypes
```

### 가치

| 이름 | 값 | 설명 |
| --- | --- | --- |
| SVG_UNIT_TYPE_UNKNOWN | `0` | 유형이 미리 정의된 유형 중 하나가 아닙니다. 이 유형의 새 값을 정의하거나 기존 값을 이 유형으로 전환하려는 시도는 유효하지 않습니다. |
| SVG_UNIT_TYPE_USERSPACEONUSE | `1` | 값 'userSpaceOnUse'에 해당합니다. |
| SVG_UNIT_TYPE_OBJECTBOUNDINGBOX | `2` | 값 'objectBoundingBox'에 해당합니다. |
