## SVGTextContentElement.GetNumberOfChars method

렌더링 여부에 관계없이 'tref' 참조에서 참조된 문자를 포함하여 현재 요소 내에서 렌더링에 사용할 수 있는 총 문자 수를 반환합니다. 사실상 이것은 DOM Level 3 Core([DOM3], 섹션 1.4)의 Node::textContent 속성 길이와 동일합니다. 해당 속성이 'tref' 요소도 확장된 경우.

```csharp
public long GetNumberOfChars()
```

### 반환 값

총 문자 수입니다.
