## Metered.SetMeteredKey method

미터링된 공개 및 개인 키를 설정합니다. 미터링된 라이센스를 구입하면 응용 프로그램을 시작할 때 이 API를 호출해야 하며 일반적으로 이 정도면 충분합니다. 단, 항상 소비 데이터 업로드에 실패하고 24시간을 초과하면 라이센스가 평가 상태로 설정됩니다. 이러한 경우를 방지하려면 라이센스 상태를 정기적으로 확인해야 하며 평가 상태이면 이 API를 다시 호출하십시오.

```csharp
public void SetMeteredKey(string publicKey, string privateKey)
```

| 모수 | 유형 | 설명 |
| --- | --- | --- |
| publicKey | String | 공개 키 |
| privateKey | String | 개인 키 |
