## SVGMarkerElement.OrientAngle property

지정된 'marker' 요소의 'orient' 속성에 해당합니다. markerUnits가 SVG_MARKER_ORIENT_ANGLE인 경우 속성 'orient'의 각도 값입니다. 그렇지 않으면 0으로 설정됩니다.

```csharp
public SVGAnimatedAngle OrientAngle { get; }
```

### 자산 가치

지정된 '마커' 요소의 '방향' 속성입니다.
