## SVGSVGElement.GetElementById method

id가 elementId에 의해 제공되는 요소에 대해 이 SVG 문서 조각을 검색합니다(즉, 검색은 문서 트리의 하위 집합으로 제한됨). 요소가 발견되면 해당 요소가 반환됩니다. 해당 요소가 없으면 null을 반환합니다. 둘 이상의 요소에 이 id. 가 있으면 동작이 정의되지 않습니다.

```csharp
public Element GetElementById(string elementId)
```

| 모수 | 유형 | 설명 |
| --- | --- | --- |
| elementId | String | 요소의 고유 ID 값입니다. |

### 반환 값

일치하는 요소입니다.
