## ISVGAnimatedPathData.AnimatedPathSegList property

SVG 구문과 일대일로 일치하는 형식으로 'd' 속성의 현재 애니메이션 콘텐츠에 대한 액세스를 제공합니다. 주어진 속성 또는 속성이 애니메이션되는 경우 속성 또는 속성의 현재 애니메이션 값을 포함하고 개체 자체와 해당 내용이 모두 읽기 전용입니다. 지정된 속성 또는 속성이 현재 애니메이션되지 않는 경우 pathSegList. 와 동일한 값을 포함합니다.

```csharp
public SVGPathSegList AnimatedPathSegList { get; }
```
