## IStyleSheet interface

واجهة StyleSheet هي الواجهة الأساسية المجردة لأي نوع من أوراق الأنماط. إنها تمثل ورقة نمط واحدة مرتبطة بمستند منظم.

```csharp
public interface IStyleSheet
```
