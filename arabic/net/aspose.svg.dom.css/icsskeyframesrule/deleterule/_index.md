```csharp
public void DeleteRule(string key)
```

| معامل | يكتب | وصف |
| --- | --- | --- |
| key | String | المفتاح الذي يصف القاعدة المطلوب حذفها. يجب أن يتم حل المفتاح إلى رقم بين 0 و 1 ، أو يتم تجاهل القاعدة |
