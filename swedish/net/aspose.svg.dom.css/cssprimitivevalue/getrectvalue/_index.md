## CSSPrimitiveValue.GetRectValue method

Denna metod används för att få Rect-värdet. Om detta CSS-värde inte innehåller ett rect-värde, höjs ett DOMException. Modifiering av motsvarande stilegenskap kan uppnås med Rect-gränssnittet.

```csharp
public abstract Rect GetRectValue()
```

### Returvärde

Returnerar Rect-värdet
