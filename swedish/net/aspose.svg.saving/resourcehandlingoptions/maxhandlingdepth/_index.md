## ResourceHandlingOptions.MaxHandlingDepth property

Hämtar eller ställer in maximalt djup för sidor som kommer att hanteras. Djup 1 innebär att endast sidor som direkt refereras från det sparade dokumentet kommer att hanteras. Att ställa in den här egenskapen till -1 kommer att leda till hantering av alla sidor. Standardvärdet är 0.

```csharp
public int MaxHandlingDepth { get; set; }
```
