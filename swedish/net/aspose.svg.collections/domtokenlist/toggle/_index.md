## Toggle(string, bool?) {#toggle_1}

Tar bort token från listan om den finns, eller lägger till token i listan om den inte gör det.

```csharp
public bool Toggle(string token, bool? force)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| token | String | Token du vill växla. |
| force | Nullable`1 | En Boolean som, om den ingår, förvandlar växeln till en enkelriktad operation. Om det är inställt på false kommer token bara att tas bort, men inte läggas till. Om satt till sant kommer token bara att läggas till, men inte tas bort. |

### Returvärde

En boolesk som indikerar om token finns i listan efter samtalet.
