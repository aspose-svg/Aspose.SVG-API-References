## OutputStream.Write method

Skriver en sekvens av byte till den omslutna output -strömmen och flyttar fram den aktuella positionen inom denna ström med antalet bytes som skrivs.

```csharp
public override void Write(byte[] buffer, int offset, int count)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| buffer | Byte[] | En uppsättning byte. Den här metoden kopierar antalet byte från bufferten till den omslutna output -strömmen. |
| offset | Int32 | Den nollbaserade byteförskjutningen i bufferten för att börja kopiera byte till den omslutna output -strömmen. |
| count | Int32 | Antalet byte som ska skrivas till den lindade utströmmen. |
