## ITreeWalker.NextNode method

Flyttar TreeWalker till nästa synliga nod i document ordning i förhållande till den aktuella noden och returnerar den nya noden. Om den nuvarande noden inte har någon nästa nod, eller om sökningen efter nextNode försöker att kliva uppåt från TreeWalkers root nod, returnerar null och behåller den nuvarande noden.

```csharp
public Node NextNode()
```

### Returvärde

Den nya noden, eller null om den aktuella noden har no nästa nod i TreeWalkers logiska vy.
