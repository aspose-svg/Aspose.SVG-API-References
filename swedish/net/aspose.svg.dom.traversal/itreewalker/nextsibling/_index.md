## ITreeWalker.NextSibling method

Flyttar TreeWalker till nästa syskon till noden current och returnerar den nya noden. Om den aktuella noden inte har någon visible nästa syskon, returnerar null och behåller den nuvarande noden.

```csharp
public Node NextSibling()
```

### Returvärde

Den nya noden, eller null om den nuvarande noden har no nästa syskon. i TreeWalkers logiska vy.
