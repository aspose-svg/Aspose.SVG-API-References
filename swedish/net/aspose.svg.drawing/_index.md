Den **Aspose.Svg.Drawing** namnutrymmet innehåller objekt och -gränssnitt för att specificera mått och enheter samt ritattributobjekt som penslar, färger och typsnitt.

## Klasser

| Klass | Beskrivning |
| --- | --- |
| [Angle](./angle/) | Vinkeldatatypen |
| [Color](./color/) | Med klassen Färg kan du ange färger som Röd-Grön-Blå (RGB) värden, Nyans-Mättnad-Luminositet (HSL) värden, Nyans-Mättnad-Value (HSV) värden,_B Hue-Hvithet (Black-Hvithet) ) värden, ljushet-AB (LAB) värden, Luminance-Chroma-Hue (LCH) värden, Cyan-Magenta-Yellow-Key (CMYK) värden, Naturliga färger (NCOL) färgnamn,_x00 . En alfakanal är också tillgänglig för att indikera transparens. |
| [Dimension](./dimension/) | Tillhandahåller basklassen för dimensioner. Den allmänna termen 'dimension' hänvisar till ett tal med en enhet kopplad till det, och betecknas med[`UnitType`](../aspose.svg.drawing/unittype/) . |
| [Frequency](./frequency/) | "frekvens"-enheten. |
| [Length](./length/) | Representerar en längdmåttenhet. |
| [LengthOrAuto](./lengthorauto/) | Representerar en behållare för lagringslängd eller 'auto'-enheter.. |
| [Margin](./margin/) | Representerar sidmarginal. |
| [Numeric](./numeric/) | Tillhandahåller basklassen för numeriska typer. |
| [Page](./page/) | Representerar ett sidobjekt som används för konfigurationsutdatasida. Lakunavärdet för sidstorlek är A4(210x297mm) |
| [Resolution](./resolution/) | Representerar en upplösningsenhet. |
| [Size](./size/) | Lagrar ett värde som anger en höjd och bredd. |
| [Time](./time/) | Representerar en tidsenhet. |
| [Unit](./unit/) | Tillhandahåller basklassen för måttenheter. |
| [UnitType](./unittype/) | Anger måttenheten. |
