## IUserAgentService.Language property

Den`Language` anger det primära språket för elementets innehåll och för något av elementets attribut som innehåller text. Dess värde måste vara en giltig BCP 47 ( språktaggen eller den tomma strängen. Att ställa in attributet till den tomma strängen indikerar att det primära språket är okänt.

```csharp
public string Language { get; set; }
```
