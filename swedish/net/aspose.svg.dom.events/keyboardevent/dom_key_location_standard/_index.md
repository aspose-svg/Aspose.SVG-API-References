## KeyboardEvent.DOM_KEY_LOCATION_STANDARD field

Nyckelaktiveringen FÅR INTE särskiljas som den vänstra eller högra versionen av nyckeln och (förutom NumLock-tangenten) härrörde inte från det numeriska tangentbordet (eller härrörde inte från en virtuell tangent som motsvarar det numeriska tangentbordet).

public const ulong DOM_KEY_LOCATION_STANDARD;
