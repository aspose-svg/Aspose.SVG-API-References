## Aspose.SVG för .NET

{{% alert color="primary" %}} 

![Aspose.SVG för .NET-produktlogotypen](../home_1.png)


Aspose.SVG för .NET API är ett plattformsoberoende bibliotek som tillhandahåller ett brett utbud av funktioner för bearbetning och rendering av SVG-dokument. API:n upprätthåller Document Object Model (DOM) och är helt kompatibel med de officiella SVG-specifikationerna, vilket gör att du kan ha total kontroll över SVG-noder och deras egenskaper. Du kan enkelt ändra dokumentträdet, lägga till och ta bort noder, ändra nodegenskaperna som det beskrivs i de officiella specifikationerna. Med hjälp av Aspose.SVG API kan du snabbt ladda, spara och konvertera SVG:er samt läsa och gå igenom elementen i filen. API:t är oberoende av någon annan programvara. Det gör att du kan arbeta med SVG-filer på alla plattformar som stöder .NET (Windows, Linux och macOS).
