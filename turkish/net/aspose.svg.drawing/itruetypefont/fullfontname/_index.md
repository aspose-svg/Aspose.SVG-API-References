## ITrueTypeFont.FullFontName property

Bu, "FamilyName" ve "SubFamilyName" kombinasyonu olmalıdır. İstisna: Yazı tipi "AltAileAdı" içinde belirtildiği gibi "Normal" ise, o zaman yalnızca "AileAdı" içinde bulunan aile adını kullanın. Yukarıdaki Tam yazı tipi adı tanımının bir istisnası, CFF OpenType yazı tipleri için Microsoft platform dizeleri içindir: bu durumda, Tam yazı tipi adı dizesi, CFF Adı INDEX. 'deki PostScript Yazı TipiAdı ile aynı olmalıdır

```csharp
public string FullFontName { get; }
```
