## Metered.SetMeteredKey method

Ölçülü açık ve özel anahtarı ayarlar. Ölçülü lisans satın alırsanız, uygulamayı başlattığınızda bu API çağrılmalıdır, normalde bu yeterlidir. Ancak, her zaman tüketim verilerini yükleyemez ve 24 saati aşarsa, lisans değerlendirme durumuna ayarlanır, böyle bir durumla karşılaşmamak için, lisans durumunu düzenli olarak kontrol etmelisiniz, değerlendirme durumuysa, bu API'yi tekrar arayın.

```csharp
public void SetMeteredKey(string publicKey, string privateKey)
```

| Parametre | Tip | Tanım |
| --- | --- | --- |
| publicKey | String | Genel anahtar |
| privateKey | String | Özel anahtar |
