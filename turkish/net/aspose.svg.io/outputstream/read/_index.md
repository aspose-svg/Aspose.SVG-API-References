## OutputStream.Read method

Sarılmış çıktı akışından bir bayt dizisini okur ve akış içindeki konumu okunan bayt sayısı kadar ilerletir.

```csharp
public override int Read(byte[] buffer, int offset, int count)
```

| Parametre | Tip | Tanım |
| --- | --- | --- |
| buffer | Byte[] | Bir bayt dizisi. Bu yöntem döndüğünde, arabellek, belirtilen bayt dizisini içerir ve offset ve (offset + count - 1) arasındaki değerler, sarılmış çıktı akışından okunan baytlarla değiştirilen ile değiştirilir. |
| offset | Int32 | Sarılmış çıktı akışından read verilerini depolamaya başlamak için arabellekteki sıfır tabanlı bayt ofseti. |
| count | Int32 | Sarılmış çıktı akışından okunacak maksimum bayt sayısı. |

### Geri dönüş değeri

Arabelleğe okunan toplam bayt sayısı. Bu, şu anda çok sayıda bayt mevcut değilse istenen bayt sayısından veya akışın sonuna ulaşılmışsa sıfır (0) değerinden daha az olabilir.
