## OutputStream.Write method

```csharp
public override void Write(byte[] buffer, int offset, int count)
```

| Parametre | Tip | Tanım |
| --- | --- | --- |
| buffer | Byte[] | Bir bayt dizisi. Bu yöntem, sayılan baytları arabellekten sarılmış output akışına kopyalar. |
| offset | Int32 | Sarılmış output akışına baytların kopyalanmaya başlanacağı arabellekteki sıfır tabanlı bayt ofseti. |
| count | Int32 | Sarılmış çıktı akışına yazılacak bayt sayısı. |
