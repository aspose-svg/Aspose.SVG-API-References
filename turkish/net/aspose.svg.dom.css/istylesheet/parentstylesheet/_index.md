## IStyleSheet.ParentStyleSheet property

Stil sayfası dahil etme kavramını destekleyen stil sayfası dilleri için, bu öznitelik varsa dahil olan stil sayfasını temsil eder. Stil sayfası üst düzey bir stil sayfasıysa veya stil sayfası dili eklemeyi desteklemiyorsa, bu özelliğin değeri null.

```csharp
public IStyleSheet ParentStyleSheet { get; }
```
