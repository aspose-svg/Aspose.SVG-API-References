## ICSSRule interface

CSSRule arabirimi, herhangi bir CSS deyimi türü için soyut temel arabirimdir. Buna hem kural kümeleri hem de kurallar dahildir. Kural ayrıştırıcı tarafından tanınmasa bile, bir uygulamanın CSS stil sayfasında belirtilen tüm kuralları koruması beklenir. Tanınmayan kurallar kullanılarak temsil edilir!:ICSSUnknownRule arayüz.
