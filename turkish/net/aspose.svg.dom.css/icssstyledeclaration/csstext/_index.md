## ICSSStyleDeclaration.CSSText property

Bildirim bloğunun ayrıştırılabilir metinsel temsili (çevreleyen kaşlı ayraçlar hariç). Bu özniteliğin ayarlanması, yeni değerin ayrıştırılmasına ve bildirim bloğundaki özelliklerin kaldırılması veya eklenmesi de dahil olmak üzere tüm özelliklerin sıfırlanmasına neden olacaktır.

```csharp
public string CSSText { get; set; }
```
