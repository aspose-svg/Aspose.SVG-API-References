## ResourceHandlingOptions.MaxHandlingDepth property

İşlenecek maksimum sayfa derinliğini alır veya ayarlar. Derinlik 1, yalnızca kaydedilen belgeden doğrudan referans verilen sayfaların işleneceği anlamına gelir. Bu özelliğin -1 olarak ayarlanması, tüm sayfaların işlenmesine yol açacaktır. Varsayılan değer 0.

```csharp
public int MaxHandlingDepth { get; set; }
```
