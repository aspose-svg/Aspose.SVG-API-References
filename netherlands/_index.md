## Aspose.SVG voor .NET

{{% alert color="primary" %}} 

![Aspose.SVG voor .NET-productlogo](../home_1.png)


Aspose.SVG voor .NET API is een platformonafhankelijke bibliotheek die een breed scala aan functies biedt voor het verwerken en weergeven van SVG-documenten. De API onderhoudt het Document Object Model (DOM) en is volledig compatibel met de officiële SVG-specificaties, waardoor u totale controle heeft over SVG-knooppunten en hun eigenschappen. U kunt eenvoudig de documentboom wijzigen, knooppunten toevoegen en verwijderen, de knooppunteigenschappen wijzigen zoals beschreven in de officiële specificaties. Met de Aspose.SVG API kunt u snel SVG's laden, opslaan en converteren, en de elementen van het bestand lezen en doorlopen. De API is onafhankelijk van andere software. Hiermee kunt u met SVG-bestanden werken op alle platforms die .NET ondersteunen (Windows, Linux en macOS).
