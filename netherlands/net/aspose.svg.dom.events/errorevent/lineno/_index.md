## ErrorEvent.LineNo property

Het lineno-attribuut moet de waarde teruggeven waarnaar het was geïnitialiseerd. Wanneer het object wordt gemaakt, moet dit attribuut op nul worden geïnitialiseerd. Het vertegenwoordigt het regelnummer waar de fout is opgetreden in het script.

public uint LineNo { get; }
