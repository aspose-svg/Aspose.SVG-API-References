## InputEvent.Data property

De gegevens bevatten de waarde van de tekens die zijn gegenereerd door een invoermethode. Dit KAN een enkel Unicode-teken zijn of een niet-lege reeks Unicode-tekens [Unicode]. Tekens MOETEN worden genormaliseerd zoals gedefinieerd door de Unicode-normalisatievorm NFC, gedefinieerd in [UAX15]. Dit attribuut KAN de lege string bevatten.

public string Data { get; }
