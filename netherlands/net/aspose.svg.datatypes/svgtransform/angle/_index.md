## SVGTransform.Angle property

Een gemakskenmerk voor SVG_TRANSFORM_ROTATE, SVG_TRANSFORM_SKEWX en SVG_TRANSFORM_SKEWY. Het bevat de opgegeven hoek. Voor SVG_TRANSFORM_MATRIX, SVG_TRANSFORM_TRANSLATE en SVG_TRANSFORM_SCALE is de hoek nul.

```csharp
public float Angle { get; }
```
