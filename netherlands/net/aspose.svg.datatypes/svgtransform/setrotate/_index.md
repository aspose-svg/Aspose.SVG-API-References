## SVGTransform.SetRotate method

Stelt het transformatietype in op SVG_TRANSFORM_ROTATE, waarbij parameter angle de rotatiehoek definieert en parameters cx en cy het optionele rotatiecentrum definiëren.

```csharp
public void SetRotate(float angle, float cx, float cy)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| angle | Single | De draaihoek. |
| cx | Single | De x-coördinaat van het rotatiecentrum. |
| cy | Single | De y-coördinaat van het rotatiecentrum. |
