| Klas | Beschrijving |
| --- | --- |
| [ContentDispositionHeaderValue](./contentdispositionheadervalue/) | Vertegenwoordigt de waarde van de koptekst voor content-disposition. |
| [ContentTypeHeaderValue](./contenttypeheadervalue/) | Vertegenwoordigt een waarde van een Content-Type-header. |
| [NameValueHeaderValue](./namevalueheadervalue/) | Vertegenwoordigt een naam/waarde-paar dat een headerwaarde beschrijft. |
