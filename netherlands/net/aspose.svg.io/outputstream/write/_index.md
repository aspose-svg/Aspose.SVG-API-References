## OutputStream.Write method

```csharp
public override void Write(byte[] buffer, int offset, int count)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| buffer | Byte[] | Een reeks bytes. Deze methode kopieert telbytes van de buffer naar de ingepakte output stream. |
| offset | Int32 | De op nul gebaseerde byte-offset in de buffer waarop het kopiëren van bytes naar de ingepakte output -stream moet beginnen. |
| count | Int32 | Het aantal bytes dat naar de ingepakte uitvoerstroom moet worden geschreven. |
