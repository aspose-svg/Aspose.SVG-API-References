## IFile.LastModified property

De laatste wijzigingsdatum van het bestand. Bij het ophalen, als user agents deze informatie beschikbaar kunnen maken, moet dit een lange lange set teruggeven aan de tijd dat het bestand voor het laatst is gewijzigd als het aantal milliseconden sinds het Unix-tijdperk.
