## IBlob.Type property

De ASCII-gecodeerde tekenreeks in kleine letters die het mediatype van de blob vertegenwoordigt. Bij het ophalen moeten user-agents het type van een blob retourneren als een ASCII-gecodeerde tekenreeks in kleine letters, zodat wanneer deze wordt geconverteerd naar een byte reeks, is het een ontleedbaar MIME-type, of de lege tekenreeks – 0 bytes – als het type niet kan worden bepaald.
