## IUrlSearchParams.Set method

Stelt de waarde van het eerst gevonden naam-waarde-paar in op de opgegeven waarde en verwijdert de andere. Als er geen naam-waarde-paren met de opgegeven naam worden gevonden, wordt een nieuwe aan de lijst toegevoegd.

```csharp
public void Set(string name, string value)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| name | String | Sleutel die zal worden gebruikt om naam-waardeparen te vinden. |
| value | String | Waarde die wordt ingesteld. |
