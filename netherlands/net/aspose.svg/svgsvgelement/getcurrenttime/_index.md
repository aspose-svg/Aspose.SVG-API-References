## SVGSVGElement.GetCurrentTime method

Retourneert de huidige tijd in seconden ten opzichte van de starttijd voor het huidige SVG-documentfragment. Als getCurrentTime wordt aangeroepen voordat de documenttijdlijn is begonnen (bijvoorbeeld doordat een script wordt uitgevoerd in een 'script'-element voordat de SVGLoad-gebeurtenis van het document wordt verzonden), wordt 0 geretourneerd.

```csharp
public float GetCurrentTime()
```

### Winstwaarde

De huidige tijd in seconden, of 0 als de documenttijdlijn nog niet is begonnen.
