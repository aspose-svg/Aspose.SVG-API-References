## ICSSRuleList indexer

Wordt gebruikt om een CSS-regel op te halen met methode item() (http://www.w3.org/TR/DOM-Level-2-Style/css.html#CSS-CSSRuleList). De volgorde in deze verzameling vertegenwoordigt de volgorde van de regels in de CSS-stijlpagina. Als index groter is dan of gelijk is aan het aantal regels in de lijst, retourneert dit null.

```csharp
public ICSSRule this[int index] { get; }
```

| index | De regelindex |
