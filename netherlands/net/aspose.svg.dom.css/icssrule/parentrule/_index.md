## ICSSRule.ParentRule property

Als deze regel binnen een andere regel staat (bijvoorbeeld een stijlregel binnen een @media-blok), is dit de bevattende regel. Als deze regel niet is genest binnen andere regels, retourneert dit null.

public ICSSRule ParentRule { get; }
