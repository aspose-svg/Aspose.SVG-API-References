## ICSSStyleDeclaration.CSSText property

De ontleedbare tekstuele representatie van het declaratieblok (exclusief de omringende accolades). Als u dit attribuut instelt, wordt de nieuwe waarde geparseerd en worden alle eigenschappen in het declaratieblok opnieuw ingesteld, inclusief het verwijderen of toevoegen van eigenschappen.

```csharp
public string CSSText { get; set; }
```
