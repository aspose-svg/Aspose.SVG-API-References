## ICSSKeyframesRule.DeleteRule method

De methode deleteRule verwijdert de CSSKeyframeRule met de doorgegeven sleutel. Als er geen regel met deze sleutel bestaat, doet de methode niets

```csharp
public void DeleteRule(string key)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| key | String | De sleutel die de te verwijderen regel beschrijft. De sleutel moet resulteren in een getal tussen 0 en 1, anders wordt de regel genegeerd |
