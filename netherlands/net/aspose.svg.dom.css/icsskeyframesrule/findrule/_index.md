## ICSSKeyframesRule.FindRule method

De methode findRule retourneert de regel met een sleutel die overeenkomt met de doorgegeven sleutel. Als een dergelijke regel niet bestaat, wordt een null-waarde geretourneerd

```csharp
public ICSSKeyframeRule FindRule(string key)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| key | String | De sleutel die de te vinden regel beschrijft. De sleutel moet resulteren in een getal tussen 0 en 1, anders wordt de regel genegeerd. |

De gevonden regel
