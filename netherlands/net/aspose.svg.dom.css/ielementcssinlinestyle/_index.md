## IElementCSSInlineStyle interface

Inline stijlinformatie gekoppeld aan elementen wordt weergegeven via het stijlattribuut. Dit vertegenwoordigt de inhoud van het STYLE-attribuut voor HTML-elementen (of elementen in andere schema's of DTD's die het STYLE-attribuut op dezelfde manier gebruiken).
