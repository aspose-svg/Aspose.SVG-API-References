## ICSSImportRule.StyleSheet property

De stijlpagina waarnaar deze regel verwijst, als deze is geladen. De waarde van dit attribuut is null als de stijlpagina nog niet is geladen of als deze niet zal worden geladen (bijv. als de stijlpagina voor een mediatype is dat niet wordt ondersteund door de user agent).

public ICSSStyleSheet StyleSheet { get; }
