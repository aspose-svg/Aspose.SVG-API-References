## IDocumentStyle interface

De DocumentStyle-interface biedt een mechanisme waarmee de stijlbladen die in een document zijn ingesloten, kunnen worden opgehaald. De verwachting is dat een instantie van de DocumentStyle-interface kan worden verkregen door bindingspecifieke castingmethoden te gebruiken op een instantie van de Document-interface.
