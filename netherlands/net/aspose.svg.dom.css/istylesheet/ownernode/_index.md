## IStyleSheet.OwnerNode property

Het knooppunt dat deze stijlpagina aan het document koppelt. Voor HTML kan dit het corresponderende LINK- of STYLE-element zijn. Voor XML kan dit de instructie voor koppelingsverwerking zijn. Voor stijlbladen die zijn opgenomen in andere stijlbladen, is de waarde van dit attribuut null.
