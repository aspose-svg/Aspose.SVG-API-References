## CSSPrimitiveValue.SetFloatValue method

Een methode om de float-waarde in te stellen met een gespecificeerde eenheid. Als de eigenschap die aan deze waarde is gekoppeld, de opgegeven eenheid of de float-waarde niet kan accepteren, blijft de waarde ongewijzigd en wordt er een DOMException verhoogd.

```csharp
public abstract void SetFloatValue(ushort unitType, float floatValue)
```

| unitType | UInt16 | Type van de eenheid. |
| floatValue | Single | De floatwaarde. |
