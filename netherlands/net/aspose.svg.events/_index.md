De naamruimte bestaat uit klassen voor op gebeurtenissen gebaseerde interactiviteit zoals tijdgebeurtenissen en zoomen.

## Klassen

| Klas | Beschrijving |
| --- | --- |
| [SVGZoomEvent](./svgzoomevent/) | De zoomgebeurtenis vindt plaats wanneer de gebruiker een actie start die ervoor zorgt dat de huidige weergave van het SVG-documentfragment opnieuw wordt geschaald. Event handlers worden alleen herkend op 'svg' elementen. |
| [TimeEvent](./timeevent/) | De TimeEvent-interface biedt specifieke contextuele informatie over tijdgebeurtenissen. De verschillende typen gebeurtenissen die kunnen optreden zijn: beginEvent, endEvent en repeatEvent. |
