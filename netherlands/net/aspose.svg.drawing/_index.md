De **Aspose.Svg.Tekening** naamruimte bevat objecten en interfaces om afmetingen en eenheden te specificeren, evenals tekenattribuutobjecten zoals penselen, kleuren en lettertypen.

## Klassen

| Klas | Beschrijving |
| --- | --- |
| [Angle](./angle/) | Het hoekgegevenstype |
| [Color](./color/) | Met de klasse Color kunt u kleuren opgeven als rood-groen-blauw (RGB)-waarden, Hue-Saturation-Luminosity (HSL)-waarden, Hue-Saturation-Value (HSV)-waarden, Hue-Whiteness-Blackness (HWB ) waarden, lichtheid-AB (LAB) waarden, Luminantie-Chroma-Hue (LCH) waarden, Cyan-Magenta-Yellow-Key (CMYK) waarden, Natuurlijke kleuren (NCOL) waarden, of met een kleurnaam . Er is ook een alfakanaal beschikbaar om transparantie aan te geven. |
| [Dimension](./dimension/) | Biedt de basisklasse voor dimensies. De algemene term 'dimensie' verwijst naar een getal waaraan een eenheid is gekoppeld en wordt aangeduid met[`UnitType`](../aspose.svg.drawing/unittype/) . |
| [Frequency](./frequency/) | De eenheid 'frequentie'. |
| [Length](./length/) | Vertegenwoordigt een lengtemeeteenheid. |
| [LengthOrAuto](./lengthorauto/) | Vertegenwoordigt een container voor opslaglengte of 'auto'-eenheden.. |
| [Margin](./margin/) | Vertegenwoordigt paginamarge. |
| [Numeric](./numeric/) | Biedt de basisklasse voor numerieke typen. |
| [Page](./page/) | Geeft aan dat een pagina-object wordt gebruikt voor configuratie-uitvoerpagina. De lacune-waarde voor paginaformaat is A4(210x297mm) |
| [Resolution](./resolution/) | Vertegenwoordigt een resolutie-eenheid. |
| [Size](./size/) | Slaat waarden op die een hoogte en breedte specificeren. |
| [Time](./time/) | Vertegenwoordigt een tijdseenheid. |
| [Unit](./unit/) | Biedt de basisklasse voor meeteenheden. |
| [UnitType](./unittype/) | Specificeert de meeteenheid. |
