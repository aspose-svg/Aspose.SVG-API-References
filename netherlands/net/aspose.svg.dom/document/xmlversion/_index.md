## Document.XmlVersion property

Een attribuut dat, als onderdeel van de XML-declaratie, het versienummer van dit document specificeert. Als er geen declaratie is en als dit document de functie "XML" ondersteunt, is de waarde "1.0". Als dit document de "XML"-functie niet ondersteunt, is de waarde altijd null.

```csharp
public string XmlVersion { get; set; }
```

### Eigendoms-waarde

De XML-versie.
