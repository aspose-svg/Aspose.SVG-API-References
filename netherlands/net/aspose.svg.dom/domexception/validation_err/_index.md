## DOMException.VALIDATION_ERR field

Als een aanroep van een methode zoals insertBefore of removeChild de Node ongeldig zou maken met betrekking tot "gedeeltelijke geldigheid", zou deze uitzondering worden gegenereerd en zou de bewerking niet worden uitgevoerd. Deze code wordt gebruikt in [DOM Level 3 Validation]. Raadpleeg deze specificatie voor meer informatie.

```csharp
public const ushort VALIDATION_ERR;
```
