## Node.InsertBefore method

Voegt het knooppunt in vóór het bestaande onderliggende knooppunt. Als kind null is, voegt u een knooppunt in aan het einde van de lijst met kinderen. Als kind een DocumentFragment-object is, worden alle kinderen in dezelfde volgorde ingevoegd vóór kind. Als het kind al in de boom zit, wordt het eerst verwijderd.

```csharp
public Node InsertBefore(Node node, Node child)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| node | Node | Het nieuwe kind. |
| child | Node | Het scheidskind. |

### Winstwaarde

Retourneert ingevoegd knooppunt
