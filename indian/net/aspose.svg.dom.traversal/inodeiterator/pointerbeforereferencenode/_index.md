## INodeIterator.PointerBeforeReferenceNode property

इस ध्वज का मान निर्धारित करता है कि क्या entity संदर्भ नोड के बच्चे पुनरावर्तक को दिखाई दे रहे हैं। अगर झूठा है, तो वे और उनके वंशज खारिज कर दिए जाएंगे। ध्यान दें कि यह अस्वीकृति whatToShow और फ़िल्टर पर पूर्वता लेती है। साथ ही ध्यान दें कि यह वर्तमान में एकमात्र ऐसी स्थिति है जहां नोडइटरेटर अलग-अलग नोड्स को छोड़कर के बजाय एक पूर्ण उपट्री को अस्वीकार कर सकते हैं। the iterator बनाते समय इकाई संदर्भ node को छुपाएं और विस्तृत EntityReferences को सही पर सेट करें। उस दस्तावेज़ का एक दृश्य बनाने के लिए जिसमें इकाई संदर्भ नोड है, लेकिन कोई इकाई विस्तार नहीं है, इकाई संदर्भ नोड दिखाने के लिए whatToShow फ़्लैग का उपयोग करें और गलत पर सेट विस्तृत EntityReferences का उपयोग करें।

```csharp
public bool PointerBeforeReferenceNode { get; }
```
