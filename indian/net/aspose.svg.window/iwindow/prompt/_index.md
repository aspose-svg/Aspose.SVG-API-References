## IWindow.Prompt method

दिए गए संदेश के साथ एक मॉडल टेक्स्ट फ़ील्ड संकेत प्रदर्शित करता है, उपयोगकर्ता द्वारा इसे खारिज करने की प्रतीक्षा करता है, और उपयोगकर्ता द्वारा दर्ज मान लौटाता है। यदि उपयोगकर्ता प्रॉम्प्ट को रद्द कर देता है, तो इसके बजाय शून्य वापस आ जाता है। यदि दूसरा तर्क मौजूद है, तो दिए गए मान को डिफ़ॉल्ट के रूप में उपयोग किया जाता है।

```csharp
public string Prompt(string message, string @default)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| message | String | संदेश। |
| default | String | डिफ़ॉल्ट। |

### प्रतिलाभ की मात्रा

उपयोगकर्ता द्वारा दर्ज किया गया मान लौटाता है
