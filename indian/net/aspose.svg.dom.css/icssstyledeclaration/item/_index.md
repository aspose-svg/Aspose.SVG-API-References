## ICSSStyleDeclaration indexer

उन गुणों को पुनः प्राप्त करने के लिए उपयोग किया जाता है जिन्हें इस घोषणा ब्लॉक में स्पष्ट रूप से सेट किया गया है। इस पद्धति का उपयोग करके प्राप्त गुणों के क्रम को वह क्रम नहीं होना चाहिए जिसमें वे सेट किए गए थे। इस विधि का उपयोग इस घोषणा ब्लॉक में सभी संपत्तियों पर पुनरावृति करने के लिए किया जा सकता है।

```csharp
public string this[int index] { get; }
```
