## ICSSKeyframesRule.FindRule method

FindRule विधि पास की गई कुंजी से मेल खाने वाली कुंजी के साथ नियम लौटाती है। यदि ऐसा कोई नियम मौजूद नहीं है, तो एक शून्य मान दिया जाता है

```csharp
public ICSSKeyframeRule FindRule(string key)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| key | String | कुंजी जो खोजने के नियम का वर्णन करती है। कुंजी को 0 और 1 के बीच की संख्या को हल करना चाहिए, या नियम को अनदेखा कर दिया जाता है। |
