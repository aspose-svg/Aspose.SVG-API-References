## IStyleSheet.Disabled property

गलत है अगर स्टाइल शीट दस्तावेज़ पर लागू होती है। सच है अगर यह नहीं है। इस विशेषता को संशोधित करने से दस्तावेज़ के लिए शैली का एक नया रिज़ॉल्यूशन हो सकता है। एक स्टाइलशीट केवल तभी लागू होती है जब उपयुक्त माध्यम परिभाषा दोनों मौजूद हों और अक्षम विशेषता झूठी हो। इसलिए, यदि मीडिया वर्तमान उपयोगकर्ता एजेंट पर लागू नहीं होता है, तो अक्षम विशेषता को अनदेखा कर दिया जाता है।

```csharp
public bool Disabled { get; set; }
```
