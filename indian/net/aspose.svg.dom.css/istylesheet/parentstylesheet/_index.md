## IStyleSheet.ParentStyleSheet property

स्टाइल शीट भाषाओं के लिए जो स्टाइल शीट समावेशन की अवधारणा का समर्थन करती हैं, यह विशेषता स्टाइल शीट सहित, यदि कोई मौजूद है, का प्रतिनिधित्व करती है। यदि स्टाइल शीट एक शीर्ष-स्तरीय स्टाइल शीट है, या स्टाइल शीट भाषा समावेशन का समर्थन नहीं करती है, तो इस विशेषता का मान शून्य है।

```csharp
public IStyleSheet ParentStyleSheet { get; }
```
