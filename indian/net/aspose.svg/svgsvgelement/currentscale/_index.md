## SVGSVGElement.CurrentScale property

सबसे बाहरी svg तत्व पर, यह विशेषता उपयोगकर्ता के आवर्धन और पैनिंग संचालन को ध्यान में रखते हुए प्रारंभिक दृश्य के सापेक्ष वर्तमान स्केल कारक को इंगित करती है, जैसा कि आवर्धन और पैनिंग के तहत वर्णित है। DOM विशेषताएँ करंटस्केल और करंटट्रांसलेट 2x3 मैट्रिक्स [abcdef] = [currentScale 0 0 currentScale currentTranslate.x currentTranslate.y] के बराबर हैं। यदि "आवर्धन" सक्षम है (यानी, ज़ूमएंडपैन = "बढ़ाना"), तो प्रभाव ऐसा होता है जैसे एसवीजी दस्तावेज़ खंड पर बाहरीतम स्तर पर एक अतिरिक्त परिवर्तन रखा गया हो (यानी, सबसे बाहरी एसवीजी तत्व के बाहर)। जब एक्सेस किया गया एक 'svg' तत्व जो सबसे बाहरी svg तत्व नहीं है, यह अपरिभाषित है कि इस विशेषता का क्या व्यवहार है।

```csharp
public float CurrentScale { get; set; }
```
