## ITrueTypeFont.FullFontName property

यह "FamilyName" और "SubFamilyName" का संयोजन होना चाहिए। अपवाद: यदि फ़ॉन्ट "नियमित" है जैसा कि "SubFamilyName" में इंगित किया गया है, तो केवल "FamilyName" में निहित पारिवारिक नाम का उपयोग करें। पूर्ण फ़ॉन्ट नाम की उपरोक्त परिभाषा का एक अपवाद Microsoft प्लेटफ़ॉर्म स्ट्रिंग्स के लिए है CFF ओपन टाइप फ़ॉन्ट्स के लिए: इस मामले में, पूर्ण फ़ॉन्ट नाम स्ट्रिंग CFF नाम INDEX. में पोस्टस्क्रिप्ट फ़ॉन्टनाम के समान होना चाहिए।

```csharp
public string FullFontName { get; }
```
