## KeyboardEvent.Key property

कुंजी दबाए गए कुंजी का कुंजी मान रखती है। यदि मान में एक मुद्रित प्रतिनिधित्व है, तो यह एक गैर-रिक्त यूनिकोड वर्ण स्ट्रिंग होना चाहिए, जो इस विनिर्देश में परिभाषित कुंजी मान निर्धारित करने के लिए एल्गोरिथम के अनुरूप हो। यदि मान एक नियंत्रण कुंजी है जिसका कोई मुद्रित प्रतिनिधित्व नहीं है, तो यह कुंजी मान सेट में परिभाषित प्रमुख मानों में से एक होना चाहिए, जैसा कि कुंजी मान निर्धारित करने के लिए एल्गोरिथम द्वारा निर्धारित किया गया है। कार्यान्वयन जो एक कुंजी की पहचान करने में असमर्थ हैं, उन्हें अज्ञात कुंजी मान का उपयोग करना चाहिए।

```csharp
public string Key { get; }
```
