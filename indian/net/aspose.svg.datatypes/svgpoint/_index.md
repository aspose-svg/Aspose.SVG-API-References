## SVGPoint class

कई SVG DOM इंटरफेस वर्ग SVGPoint के ऑब्जेक्ट को संदर्भित करते हैं। एक SVGPoint एक (x, y) निर्देशांक जोड़ी है। जब मैट्रिक्स ऑपरेशंस में उपयोग किया जाता है, तो एक SVGPoint को फॉर्म के वेक्टर के रूप में माना जाता है: [x] [y] [1] यदि एक SVGRect ऑब्जेक्ट को केवल पढ़ने के लिए नामित किया गया है, तो इसके गुणों में से किसी एक को असाइन करने का प्रयास किया जाएगा परिणामस्वरूप एक अपवाद फेंका जा रहा है।

```csharp
public class SVGPoint : SVGValueType
```
