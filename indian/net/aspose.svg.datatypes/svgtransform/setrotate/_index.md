## SVGTransform.SetRotate method

```csharp
public void SetRotate(float angle, float cx, float cy)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| angle | Single | घूर्णन कोण। |
| cx | Single | रोटेशन के केंद्र का एक्स समन्वय। |
| cy | Single | रोटेशन के केंद्र का वाई समन्वय। |
