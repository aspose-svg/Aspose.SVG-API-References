## Node.InsertBefore method

मौजूदा चाइल्ड नोड चाइल्ड से पहले नोड डालें। यदि बच्चा शून्य है, तो बच्चों की सूची के अंत में नोड डालें। यदि बच्चा पहले से ही पेड़ में है, तो उसे पहले हटा दिया जाता है।

```csharp
public Node InsertBefore(Node node, Node child)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| node | Node | नया बच्चा। |
| child | Node | रेफरी बच्चा। |

सम्मिलित नोड लौटाता है
