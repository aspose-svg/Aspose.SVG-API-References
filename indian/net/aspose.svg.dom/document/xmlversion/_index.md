## Document.XmlVersion property

एक्सएमएल घोषणा के भाग के रूप में निर्दिष्ट एक विशेषता, इस दस्तावेज़ की संस्करण संख्या। यदि कोई घोषणा नहीं है और यदि यह दस्तावेज़ "XML" सुविधा का समर्थन करता है, तो मान "1.0" है। यदि यह दस्तावेज़ "XML" सुविधा का समर्थन नहीं करता है, तो मान हमेशा शून्य होता है.

```csharp
public string XmlVersion { get; set; }
```
