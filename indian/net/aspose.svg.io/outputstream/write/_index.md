## OutputStream.Write method

लपेटे गए आउटपुट स्ट्रीम में बाइट्स का अनुक्रम लिखता है और इस स्ट्रीम के भीतर वर्तमान स्थिति को बाइट्स की संख्या से आगे बढ़ाता है।

```csharp
public override void Write(byte[] buffer, int offset, int count)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| buffer | Byte[] | बाइट्स की एक सरणी। यह विधि बफ़र से लिपटे आउटपुट स्ट्रीम में बाइट्स की गणना करती है। |
| offset | Int32 | बफ़र में शून्य-आधारित बाइट ऑफ़सेट जिस पर बाइट्स को लिपटे आउटपुट स्ट्रीम में कॉपी करना शुरू करना है। |
| count | Int32 | लिपटे आउटपुट स्ट्रीम में लिखे जाने वाले बाइट्स की संख्या। |
