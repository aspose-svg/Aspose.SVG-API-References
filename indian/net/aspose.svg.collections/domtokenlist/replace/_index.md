## DOMTokenList.Replace method

मौजूदा टोकन को नए टोकन से बदलता है। यदि पहला टोकन मौजूद नहीं है तो कुछ नहीं करता है।

```csharp
public bool Replace(string token, string newToken)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| token | String | वह टोकन जिसे आप बदलना चाहते हैं। |
| newToken | String | वह टोकन जिसे आप पुराने टोकन से बदलना चाहते हैं। |

### प्रतिलाभ की मात्रा

बूलियन`सत्य` अगर टोकन मिल गया और बदल दिया गया,`असत्य` अन्यथा।
