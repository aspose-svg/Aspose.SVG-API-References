द **Aspose.Svg.Services** सेवा कार्यान्वयन के लिए प्रोटोकॉल के रूप में नाम स्थान में इंटरफ़ेस शामिल है.

## इंटरफेस

| इंटरफेस | विवरण |
| --- | --- |
| [IDeviceInformationService](./ideviceinformationservice/) | एक इंटरफ़ेस जो एक ऐसे वातावरण का वर्णन करता है जिसमें[`Document`](../aspose.svg.dom/document/) उपयोगकर्ता को प्रस्तुत किया जाता है। |
| [INetworkService](./inetworkservice/) | नेटवर्क संचालन के लिए एक इंटरफ़ेस प्रदान करता है। |
| [IRuntimeService](./iruntimeservice/) | इस सेवा का उपयोग रनटाइम संबंधित गुणों को कॉन्फ़िगर करने के लिए किया जाता है। |
| [IService](./iservice/) | सेवा कार्यान्वयन के लिए एक आधार इंटरफ़ेस परिभाषित करता है। |
| [IServiceContainer](./iservicecontainer/) | सेवाओं के लिए एक कंटेनर प्रदान करता है। |
| [IServiceProvider](./iserviceprovider/) | सेवाओं के लिए एक अपरिवर्तनीय कंटेनर प्रदान करता है। |
| [IUserAgentService](./iuseragentservice/) | एक इंटरफ़ेस जिसे एक उपयोगकर्ता एजेंट वातावरण के रूप में वर्णित किया गया है। |
