## .NET के लिए Aspose.SVG

{{% alert color="primary" %}} 

![.NET उत्पाद लोगो के लिए Aspose.SVG](../home_1.png)


.NET API के लिए Aspose.SVG एक क्रॉस-प्लेटफ़ॉर्म लाइब्रेरी है जो SVG दस्तावेज़ों के प्रसंस्करण और प्रतिपादन के लिए कई प्रकार की सुविधाएँ प्रदान करता है। एपीआई दस्तावेज़ ऑब्जेक्ट मॉडल (डीओएम) को बनाए रखता है और आधिकारिक एसवीजी विनिर्देशों के साथ पूरी तरह से संगत है, जिससे आप एसवीजी नोड्स और उनके गुणों पर पूर्ण नियंत्रण रख सकते हैं। आप दस्तावेज़ ट्री को आसानी से संशोधित कर सकते हैं, नोड्स जोड़ और हटा सकते हैं, नोड गुणों को बदल सकते हैं जैसा कि आधिकारिक विनिर्देशों में वर्णित है। Aspose.SVG API का उपयोग करके आप एसवीजी को जल्दी से लोड, सेव और कन्वर्ट कर सकते हैं और साथ ही फ़ाइल के तत्वों को पढ़ और ट्रैवर्स कर सकते हैं। एपीआई किसी भी अन्य सॉफ्टवेयर से स्वतंत्र है। यह आपको .NET (Windows, Linux, और macOS) को सपोर्ट करने वाले सभी प्लेटफॉर्म पर SVG फाइलों के साथ काम करने में सक्षम बनाता है।
