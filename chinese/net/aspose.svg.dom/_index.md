的 **Aspose.Svg.Dom（文档对象模型）** namespace 提供 API， 表示任何 HTML、XML 或 SVG 文档并与之交互。 DOM 是加载在浏览器中的文档模型， 将文档表示为节点树，其中每个 node 表示文档的一部分（例如元素、文本字符串或注释）.
