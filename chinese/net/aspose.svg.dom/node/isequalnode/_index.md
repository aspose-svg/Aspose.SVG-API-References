## Node.IsEqualNode method

测试两个节点是否相等。 此方法测试节点的相等性，而不是相同性（即两个节点是否是对同一对象的引用），可以使用 Node.isSameNode() 进行测试。所有相同的节点也将相等，但反过来可能不正确。

```csharp
public bool IsEqualNode(Node otherNode)
```

| 范围 | 类型 | 描述 |
| --- | --- | --- |
| otherNode | Node | 另一个节点。 |

### 返回值

`真的` if [等于节点] [指定的arg]；否则，`错误的` .
