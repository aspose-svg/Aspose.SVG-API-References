## Node.InsertBefore method

在现有子节点 child 之前插入节点。如果 child 为 null，则在子节点列表的末尾插入节点。 如果 child 是 DocumentFragment 对象，则其所有子节点均按相同顺序插入到 child 之前。如果孩子已经在树中，则首先将其删除。

```csharp
public Node InsertBefore(Node node, Node child)
```

| 范围 | 类型 | 描述 |
| --- | --- | --- |
| node | Node | 新的孩子。 |
| child | Node | 裁判的孩子。 |

### 返回值

返回插入的节点
