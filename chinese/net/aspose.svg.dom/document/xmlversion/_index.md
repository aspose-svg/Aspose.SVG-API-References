## Document.XmlVersion property

作为 XML 声明的一部分，指定此文档版本号的属性。如果没有声明并且此文档支持“XML”功能，则值为“1.0”。如果此文档不支持“XML”功能，则该值始终为空。

```csharp
public string XmlVersion { get; set; }
```

### 适当的价值

XML 版本。
