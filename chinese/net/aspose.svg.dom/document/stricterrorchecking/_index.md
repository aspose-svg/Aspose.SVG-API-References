## Document.StrictErrorChecking property

指定是否强制执行错误检查的属性。当设置为 false 时，实现可以自由地不测试通常在 DOM 操作上定义的每个可能的错误情况，并且在使用 Document.normalizeDocument() 时不会在 DOM 操作上引发任何 DOMException 或报告错误。如果出现错误，则行为未定义。此属性默认为 true。

```csharp
public bool StrictErrorChecking { get; set; }
```

### 适当的价值

` true` 如果严格错误检查；否则，` false` 。
