## ITraversal.WhatToShow property

此属性确定哪些节点类型通过 迭代器呈现。可用的常量集在 NodeFilter 接口中定义。未被 whatToShow 接受的节点将被跳过，但它们的子节点可能仍 被考虑。请注意，此跳过优先于过滤器 （如果有）。

```csharp
public long WhatToShow { get; }
```
