## SVGLength.ValueAsString property

作为字符串值的值，单位由 unitType 表示。设置此属性将导致值、valueInSpecifiedUnits 和 unitType 自动更新以反映此设置。

```csharp
public string ValueAsString { get; set; }
```

### 适当的价值

字符串形式的值。
