## SVGTransform.SetRotate method

将变换类型设置为 SVG_TRANSFORM_ROTATE，参数 angle 定义旋转角度，参数 cx 和 cy 定义可选的旋转中心。

```csharp
public void SetRotate(float angle, float cx, float cy)
```

| 范围 | 类型 | 描述 |
| --- | --- | --- |
| angle | Single | 旋转角度。 |
| cx | Single | 旋转中心的 x 坐标。 |
| cy | Single | 旋转中心的 y 坐标。 |
