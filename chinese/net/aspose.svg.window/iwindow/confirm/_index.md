## IWindow.Confirm method

显示带有给定消息的模态 OK/Cancel 提示，等待用户关闭它，如果用户单击 OK 则返回 true，如果用户单击 Cancel 则返回 false。

```csharp
public bool Confirm(string message)
```

| 范围 | 类型 | 描述 |
| --- | --- | --- |
| message | String | 消息。 |

### 返回值

如果用户单击“确定”，则返回 true；如果用户单击“取消”，则返回 false
