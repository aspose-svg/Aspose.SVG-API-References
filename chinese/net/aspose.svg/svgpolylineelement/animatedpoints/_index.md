## SVGPolylineElement.AnimatedPoints property

提供对“点”属性的当前动画内容的访问。如果给定的属性或属性正在动画，则包含属性或属性的当前动画值。如果给定的属性或特性当前未被动画化，则包含与 points. 相同的值

```csharp
public SVGPointList AnimatedPoints { get; }
```

### 适当的价值

动画点.
