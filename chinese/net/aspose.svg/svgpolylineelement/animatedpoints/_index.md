## SVGPolylineElement.AnimatedPoints property

提供对“points”属性的当前动画内容的访问。如果给定的属性或属性正在被动画化，则包含属性或属性的当前动画值。如果给定的属性或属性当前没有被动画化，则包含与点相同的值。

public SVGPointList AnimatedPoints { get; }
