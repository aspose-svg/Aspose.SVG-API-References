## SVGMarkerElement.OrientAngle property

对应于给定“标记”元素上的属性“orient”。如果 markerUnits 是 SVG_MARKER_ORIENT_ANGLE，则属性 'orient' 的角度值；否则，它将被设置为零。

```csharp
public SVGAnimatedAngle OrientAngle { get; }
```

### 适当的价值

给定“标记”元素的属性“orient”。
