## NodeFilter.FILTER_SKIP field

跳过此单个节点。为 NodeIterator 或 TreeWalker 定义的导航方法将不会返回 此节点。对于 NodeIterator 和 TreeWalker，该节点的子节点仍将被 考虑。

```csharp
public const short FILTER_SKIP;
```
