## NodeFilter.SHOW_ENTITY field

显示实体节点。这仅在创建 以实体节点作为其 根的迭代器或树遍历器时才有意义；在这种情况下，意味着 Entity 节点将出现在遍历的第一个位置。由于 实体不是文档树的一部分，因此当 遍历文档树时它们不会出现。

```csharp
public const long SHOW_ENTITY;
```
