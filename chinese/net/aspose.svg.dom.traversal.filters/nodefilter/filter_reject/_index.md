## NodeFilter.FILTER_REJECT field

拒绝该节点。为 NodeIterator 或 TreeWalker 定义的导航方法将不会返回 此节点。对于 TreeWalker，此节点 的子节点也将被拒绝。 NodeIterators 将其视为 FILTER_SKIP. 的 同义词

```csharp
public const short FILTER_REJECT;
```
