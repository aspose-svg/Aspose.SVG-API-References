## OutputStream.Read method

从包装的输出流 中读取字节序列，并将流中的位置前进读取的字节数。

```csharp
public override int Read(byte[] buffer, int offset, int count)
```

| 范围 | 类型 | 描述 |
| --- | --- | --- |
| buffer | Byte[] | 一个字节数组。当此方法返回时，缓冲区包含指定的 字节数组，其中偏移量和 (offset + count - 1) 之间的值替换为 从包装的输出流中读取的字节。 |
| offset | Int32 | 缓冲区中从零开始的字节偏移量，从该偏移量开始存储来自包装输出流的数据 read 。 |
| count | Int32 | 要从包装的输出流中读取的最大字节数。 |

### 返回值

读入缓冲区的总字节数。如果当前没有那么多字节可用，这可能小于请求的字节数 number ，如果已到达流的末尾，则可能小于零 (0) 。
