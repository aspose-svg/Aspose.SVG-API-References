## IDocumentStyle.StyleSheets property

包含所有样式表的列表，这些样式表明确链接到或嵌入到文档中。对于 HTML 文档，这包括通过 HTML LINK 元素包含的外部样式表和内联 STYLE 元素。

public IStyleSheetList StyleSheets { get; }
