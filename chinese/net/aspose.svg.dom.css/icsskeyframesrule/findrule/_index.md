## ICSSKeyframesRule.FindRule method

findRule 方法返回具有与传递的键匹配的键的规则。如果不存在这样的规则，则返回空值

```csharp
public ICSSKeyframeRule FindRule(string key)
```

| 范围 | 类型 | 描述 |
| --- | --- | --- |
| key | String | 描述要查找的规则的键。密钥必须解析为 0 到 1 之间的数字，否则规则将被忽略。 |

### 返回值

找到的规则
