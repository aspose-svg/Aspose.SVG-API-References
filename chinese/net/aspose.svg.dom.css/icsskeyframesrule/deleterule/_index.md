## ICSSKeyframesRule.DeleteRule method

deleteRule 方法使用传递的键删除 CSSKeyframeRule。如果不存在具有此键的规则，则该方法不执行任何操作

```csharp
public void DeleteRule(string key)
```

| 范围 | 类型 | 描述 |
| --- | --- | --- |
| key | String | 描述要删除的规则的键。密钥必须解析为 0 到 1 之间的数字，否则规则将被忽略 |
