## CSSPrimitiveValue.GetRectValue method

此方法用于获取 Rect 值。如果此 CSS 值不包含 rect 值，则会引发 DOMException。修改相应的样式属性可以使用Rect接口实现。

```csharp
public abstract Rect GetRectValue()
```
