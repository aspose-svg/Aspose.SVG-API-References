## CSSPrimitiveValue.GetRGBColorValue method

此方法用于获取 RGB 颜色。如果此 CSS 值不包含 RGB 颜色值，则会引发 DOMException。修改相应的样式属性可以使用RGBColor接口实现。

```csharp
public abstract RGBColor GetRGBColorValue()
```

### 返回值

返回 RGB 颜色值
