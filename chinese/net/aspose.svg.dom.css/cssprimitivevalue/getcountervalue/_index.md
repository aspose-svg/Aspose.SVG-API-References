## CSSPrimitiveValue.GetCounterValue method

此方法用于获取计数器值。如果此 CSS 值不包含计数器值，则会引发 DOMException。修改相应的样式属性可以使用Counter接口实现。

```csharp
public abstract Counter GetCounterValue()
```

### 返回值

返回计数器值
