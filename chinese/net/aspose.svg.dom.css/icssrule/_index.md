## ICSSRule interface

CSSRule 接口是任何类型的 CSS 语句的抽象基接口。这包括规则集和规则。实现应保留 CSS 样式表中指定的所有规则，即使解析器无法识别该规则。无法识别的规则使用!:ICSSUnknownRule接口.
