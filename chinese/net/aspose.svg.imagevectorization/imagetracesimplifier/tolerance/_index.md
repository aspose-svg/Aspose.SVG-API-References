## ImageTraceSimplifier.Tolerance property

公差值决定了允许从迹线中消除的点的最大误差公差。 它必须在 0 到 4 的范围内。任何更高或更低的值将相应地与该范围的最小值和最大值对齐。 默认值为 0.3.

public float Tolerance { get; set; }
