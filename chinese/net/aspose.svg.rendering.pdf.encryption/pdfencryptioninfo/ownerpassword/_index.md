## PdfEncryptionInfo.OwnerPassword property

获取或设置所有者密码。

```csharp
public string OwnerPassword { get; set; }
```

### 适当的价值

所有者密码。

### 评论

使用正确的所有者密码（假设它不是 与用户密码相同）打开文档允许完全（所有者）访问文档。这个 无限制访问包括更改文档密码和 访问权限的能力。
