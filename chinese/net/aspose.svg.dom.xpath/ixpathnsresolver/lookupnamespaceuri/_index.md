## IXPathNSResolver.LookupNamespaceURI method

查找与给定命名空间前缀关联的命名空间 URI。 XPath 求值器绝不能用`无效的`或空的 参数，因为这样做的结果是未定义的。

```csharp
public string LookupNamespaceURI(string prefix)
```

| 范围 | 类型 | 描述 |
| --- | --- | --- |
| prefix | String | 要查找的前缀。 |

### 返回值

返回关联的命名空间 URI 或`无效的`如果没有找到 。
