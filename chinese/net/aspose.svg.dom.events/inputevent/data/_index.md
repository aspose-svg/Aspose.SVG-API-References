## InputEvent.Data property

该数据保存输入法生成的字符值。这可以是单个 Unicode 字符或 Unicode 字符的非空序列 [Unicode]。字符应该按照 [UAX15] 中定义的 Unicode 规范化形式 NFC 的定义进行规范化。该属性可以包含空字符串。

```csharp
public string Data { get; }
```
