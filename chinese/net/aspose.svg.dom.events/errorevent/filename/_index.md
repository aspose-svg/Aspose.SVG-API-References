## ErrorEvent.FileName property

文件名属性必须返回它被初始化的值。创建对象时，必须将此属性初始化为空字符串。它表示最初发生错误的脚本的绝对 URL。

```csharp
public string FileName { get; }
```

### 适当的价值

文件名。
