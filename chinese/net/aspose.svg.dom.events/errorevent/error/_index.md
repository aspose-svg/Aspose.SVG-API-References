## ErrorEvent.Error property

错误属性必须返回它被初始化的值。创建对象时，必须将此属性初始化为空。在适当的情况下，它被设置为表示错误的对象（例如，在未捕获的 DOM 异常情况下的异常对象）。

```csharp
public object Error { get; }
```

### 适当的价值

错误。
