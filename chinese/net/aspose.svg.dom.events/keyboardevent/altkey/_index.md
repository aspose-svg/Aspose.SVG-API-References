## KeyboardEvent.AltKey property

如果 Alt（替代）（或“Option”）键修饰符处于活动状态则为真。此属性的未初始化值必须为 false.

```csharp
public bool AltKey { get; }
```

### 适当的价值

`真的`如果是 Alt；否则，`错误的` .
