## KeyboardEvent.Repeat property

如果持续按下键则为真。按住一个键必须导致重复事件 keydown，beforeinput，input 按此顺序，以系统配置确定的速率。对于具有长按键行为的移动设备，具有 repeat 属性值为 true 的第一个按键事件必须作为长按键的指示。为了开始重复必须按下键的时间长度取决于配置。

```csharp
public bool Repeat { get; }
```

### 适当的价值

`真的`如果重复；否则，`错误的` .
