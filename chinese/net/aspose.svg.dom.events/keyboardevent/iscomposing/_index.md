## KeyboardEvent.IsComposing property

如果键事件作为组合会话的一部分发生，即在组合开始事件之后和相应的组合结束事件之前发生，则为真。此属性的未初始化值必须为 false.

```csharp
public bool IsComposing { get; }
```

### 适当的价值

`真的`如果这个实例正在作曲；否则，`错误的` .
