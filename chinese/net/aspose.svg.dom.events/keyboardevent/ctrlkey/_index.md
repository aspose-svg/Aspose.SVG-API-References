## KeyboardEvent.CtrlKey property

如果 Control（控制）键修饰符处于活动状态则为真。 此属性的未初始化值必须为假。

```csharp
public bool CtrlKey { get; }
```

### 适当的价值

`真的`如果控制；否则，`错误的` .
