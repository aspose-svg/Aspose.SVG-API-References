## KeyboardEvent class

KeyboardEvent 接口提供与键盘设备相关的特定上下文信息。每个键盘事件都使用一个值引用一个键。键盘事件通常针对具有焦点的元素。

```csharp
public class KeyboardEvent : UIEvent
```
