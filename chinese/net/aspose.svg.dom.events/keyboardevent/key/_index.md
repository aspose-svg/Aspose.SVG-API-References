## KeyboardEvent.Key property

键保存按下的键的键值。如果该值具有打印表示，则它必须是非空 Unicode 字符串，符合本规范定义的确定键值的算法。如果该值是没有打印表示的控制键，则它必须是键值集中定义的键值之一，由用于确定键值的算法确定。无法识别密钥的实现必须使用密钥值 Unidentified.

```csharp
public string Key { get; }
```
