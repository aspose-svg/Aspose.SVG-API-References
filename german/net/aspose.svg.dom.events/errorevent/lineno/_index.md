## ErrorEvent.LineNo property

Das lineno-Attribut muss den Wert zurückgeben, mit dem es initialisiert wurde. Wenn das Objekt erstellt wird, muss dieses Attribut auf Null initialisiert werden. Es stellt die Zeilennummer dar, in der der Fehler im Skript aufgetreten ist.
