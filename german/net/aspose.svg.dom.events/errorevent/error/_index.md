## ErrorEvent.Error property

Das Fehlerattribut muss den Wert zurückgeben, auf den es initialisiert wurde. Wenn das Objekt erstellt wird, muss dieses Attribut auf null initialisiert werden. Gegebenenfalls wird es auf das Objekt gesetzt, das den Fehler darstellt (z. B. das Ausnahmeobjekt im Fall einer nicht abgefangenen DOM-Ausnahme).

public object Error { get; }
