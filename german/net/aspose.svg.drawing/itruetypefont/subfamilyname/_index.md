## ITrueTypeFont.SubFamilyName property

Der Name der Schriftunterfamilie unterscheidet die Schrift in einer Gruppe mit demselben Schriftfamiliennamen. Es wird davon ausgegangen, dass dies Stil (kursiv, schräg) und Gewicht (leicht, fett, schwarz usw.) anspricht. Eine Schriftart ohne besondere Unterschiede in Gewicht oder Stil (z. B. mittleres Gewicht, nicht kursiv und fsSelection-Bit 6 gesetzt) sollte den String "Regular" an dieser Position gespeichert haben.

```csharp
public string SubFamilyName { get; }
```

### Rückgabewert
