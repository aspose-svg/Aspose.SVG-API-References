## PdfDevice.MoveTo method

Beginnt einen neuen Teilpfad, indem der aktuelle Punkt auf die Koordinaten des Parameters pt verschoben wird, wobei alle Verbindungsliniensegmente weggelassen werden. Wenn die vorherige Pfadkonstruktionsmethode im aktuellen Pfad auch "MoveTo" war, überschreibt das neue "MoveTo" sie; keine Spur der vorherigen "MoveTo"-Operation bleibt im Pfad.

```csharp
public override void MoveTo(PointF pt)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| pt | PointF | Punkt, wohin der Pfad verschoben werden soll. |
