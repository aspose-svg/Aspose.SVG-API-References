## SVGDocument.GetOverrideStyle method

Diese Methode wird verwendet, um die Überschreibungsstildeklaration für ein bestimmtes Element und ein bestimmtes Pseudoelement abzurufen.

```csharp
public ICSSStyleDeclaration GetOverrideStyle(Element elt, string pseudoElt)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| elt | Element | Das Element, dessen Stil geändert werden soll. Dieser Parameter darf nicht null sein. |
| pseudoElt | String | Das Pseudoelement oder null, wenn keines vorhanden ist. |

### Rückgabewert
