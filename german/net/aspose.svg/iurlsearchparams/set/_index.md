## IUrlSearchParams.Set method

Setzt den Wert des ersten gefundenen Name-Wert-Paares auf den angegebenen Wert und entfernt die anderen. Wenn keine Name-Wert-Paare mit dem angegebenen Namen gefunden werden, wird ein neues an die Liste angehängt.

```csharp
public void Set(string name, string value)
```

| value | String | Wert, der gesetzt wird. |
