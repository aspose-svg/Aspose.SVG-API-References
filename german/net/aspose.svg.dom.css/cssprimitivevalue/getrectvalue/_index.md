## CSSPrimitiveValue.GetRectValue method

Diese Methode wird verwendet, um den Rect-Wert zu erhalten. Wenn dieser CSS-Wert keinen Rect-Wert enthält, wird eine DOMException ausgelöst. Die Änderung der entsprechenden Stileigenschaft kann über die Rect-Schnittstelle erreicht werden.

```csharp
public abstract Rect GetRectValue()
```

### Rückgabewert

Gibt Rect-Wert zurück
