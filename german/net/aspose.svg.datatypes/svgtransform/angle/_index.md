## SVGTransform.Angle property

Ein praktisches Attribut für SVG_TRANSFORM_ROTATE, SVG_TRANSFORM_SKEWX und SVG_TRANSFORM_SKEWY. Es enthält den angegebenen Winkel. Für SVG_TRANSFORM_MATRIX, SVG_TRANSFORM_TRANSLATE und SVG_TRANSFORM_SCALE ist der Winkel Null.

```csharp
public float Angle { get; }
```

### Eigentumswert

Der angegebene Winkel.
