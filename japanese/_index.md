## Aspose.SVG for .NET

{{% alert color="primary" %}} 

![Aspose.SVG for .NET 製品ロゴ](../home_1.png)


Aspose.SVG for .NET API は、SVG ドキュメントの処理とレンダリングのための幅広い機能を提供するクロスプラットフォーム ライブラリです。 API はドキュメント オブジェクト モデル (DOM) を維持し、公式の SVG 仕様と完全に互換性があるため、SVG ノードとそのプロパティを完全に制御できます。 ドキュメント ツリーの変更、ノードの追加と削除、ノード プロパティの変更は、公式の仕様に記載されているとおりに簡単に行うことができます。 Aspose.SVG API を使用すると、SVG をすばやくロード、保存、変換したり、ファイルの要素を読み取ったりトラバースしたりできます。 API は他のソフトウェアから独立しています。 これにより、.NET をサポートするすべてのプラットフォーム (Windows、Linux、および macOS) で SVG ファイルを操作できます。
