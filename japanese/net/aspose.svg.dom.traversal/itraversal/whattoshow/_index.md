## ITraversal.WhatToShow property

この属性は、 イテレーターを介して提示されるノード タイプを決定します。使用可能な定数のセットは、 NodeFilter インターフェイスで定義されています。 whatToShow によって 受け入れられないノードはスキップされますが、それらの子はまだ 考慮される場合があります。このスキップは、フィルタ (存在する場合) よりも優先されることに注意してください.

```csharp
public long WhatToShow { get; }
```

### プロパティ値

表示するもの.
