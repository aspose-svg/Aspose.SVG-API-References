## Element.HasAttribute method

指定された名前の属性がこの要素で指定されている場合、またはデフォルト値がある場合は true、それ以外の場合は false を返します。

```csharp
public bool HasAttribute(string name)
```

| パラメータ | タイプ | 説明 |
| --- | --- | --- |
| name | String | 属性名。 |

### 戻り値

`真実`指定された名前に属性がある場合。さもないと、`間違い` .
