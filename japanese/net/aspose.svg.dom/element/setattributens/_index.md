## Element.SetAttributeNS method

新しい属性を追加します。同じローカル名と名前空間 URI を持つ属性が要素に既に存在する場合、そのプレフィックスは修飾された名前のプレフィックス部分に変更され、その値は値パラメーターに変更されます.

```csharp
public void SetAttributeNS(string namespaceURI, string qualifiedName, string value)
```

| パラメータ | タイプ | 説明 |
| --- | --- | --- |
| namespaceURI | String | 名前空間 URI。 |
| qualifiedName | String | 資格者の名前。 |
| value | String | 値。 |
