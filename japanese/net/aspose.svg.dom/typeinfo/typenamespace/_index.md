## TypeInfo.TypeNamespace property

型の名前空間を取得します。関連付けられた要素または属性に対して宣言された型の名前空間、または要素に宣言がない場合、または名前空間情報が利用できない場合は null です。

```csharp
public string TypeNamespace { get; }
```

### プロパティ値

タイプの名前空間.
