## Document.XmlVersion property

XML 宣言の一部として、このドキュメントのバージョン番号を指定する属性。宣言がなく、このドキュメントが「XML」機能をサポートしている場合、値は「1.0」です。このドキュメントが「XML」機能をサポートしていない場合、値は常に null. です。

```csharp
public string XmlVersion { get; set; }
```

### プロパティ値

XML バージョン。
