## Attr.LocalName property

このノードの修飾名のローカル部分を返します。 ELEMENT_NODE および ATTRIBUTE_NODE 以外のタイプのノード、および Document.createElement() などの DOM レベル 1 メソッドで作成されたノードの場合、これは常に null です。

```csharp
public override string LocalName { get; }
```

### プロパティ値

ローカルの名前。
