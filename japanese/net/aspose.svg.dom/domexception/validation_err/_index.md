## DOMException.VALIDATION_ERR field

insertBefore や removeChild などのメソッドを呼び出すと、Node が「部分的な有効性」に関して無効になる場合、この例外が発生し、操作は実行されません。このコードは [DOM Level 3 Validation] で使用されます。詳細については、この仕様を参照してください。

```csharp
public const ushort VALIDATION_ERR;
```
