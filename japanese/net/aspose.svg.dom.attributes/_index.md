**Aspose.Svg.Dom.Attributes**名前空間には、 を指定するか、任意のオブジェクト (ノード、要素など) の機能または動作を修飾する統合属性クラスが含まれています.

## クラス

| クラス | 説明 |
| --- | --- |
| [DOMConstructorAttribute](./domconstructorattribute/) | W3C で定義されているコンストラクターを指定します。 |
| [DOMNameAttribute](./domnameattribute/) | W3C で定義されている正式な DOM オブジェクト名を指定します。 |
| [DOMNamedPropertyGetterAttribute](./domnamedpropertygetterattribute/) | メソッドが名前付きプロパティ getter として使用されることを指定します。 |
| [DOMNoInterfaceObjectAttribute](./domnointerfaceobjectattribute/) | [NoInterfaceObject] 拡張属性がインターフェイスに表示される場合、ECMAScript バインディングのインターフェイスにインターフェイス オブジェクトが存在しないことを示します。 |
| [DOMNullableAttribute](./domnullableattribute/) | null 値を割り当てることができる DOM オブジェクトを指定します。 |
| [DOMObjectAttribute](./domobjectattribute/) | オブジェクトがこの属性でマークされていることを指定します。この属性は W3C によって定義されています。 |
| [DOMTreatNullAsAttribute](./domtreatnullasattribute/) | メンバー値の null を指定値として扱うことを示します。 |
## 列挙

| 列挙 | 説明 |
| --- | --- |
