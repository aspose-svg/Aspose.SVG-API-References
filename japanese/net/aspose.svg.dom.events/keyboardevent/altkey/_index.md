## KeyboardEvent.AltKey property

Alt (代替) (または "Option") キー修飾子がアクティブだった場合は true。この属性の初期化されていない値は false でなければなりません.

```csharp
public bool AltKey { get; }
```

### プロパティ値

`真実` Altの場合;さもないと、`間違い` .
