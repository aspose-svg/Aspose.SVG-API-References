## ErrorEvent.Error property

error 属性は、初期化された値を返す必要があります。オブジェクトの作成時に、この属性を null に初期化する必要があります。必要に応じて、エラーを表すオブジェクトに設定されます (例: キャッチされていない DOM 例外の場合の例外オブジェクト).

```csharp
public object Error { get; }
```

### プロパティ値

エラー.
