## ImageVectorizerConfiguration.ImageSizeLimit property

画像の幅と高さの乗算によって決定される画像の最大寸法を取得または設定します。 画像のサイズは、このプロパティに基づいてスケーリングされます。 デフォルト値は 1800000 です。

```csharp
public int ImageSizeLimit { get; set; }
```
