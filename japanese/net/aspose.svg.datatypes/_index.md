名前空間は、すべての SVG 固有のデータ型へのアクセスを提供します.

## クラス

| クラス | 説明 |
| --- | --- |
| [SVGAngle](./svgangle/) | SVGAngle インターフェイスは、角度の基本データ型に対応します。 |
| [SVGAnimatedAngle](./svganimatedangle/) | アニメーション化できる角度の基本データ型の属性に使用されます. |
| [SVGAnimatedBoolean](./svganimatedboolean/) | アニメート可能なブール型の属性に使用されます。 |
| [SVGAnimatedEnumeration](./svganimatedenumeration/) | 値が特定の列挙型の定数である必要があり、アニメーション化できる属性に使用されます。 |
| [SVGAnimatedInteger](./svganimatedinteger/) | アニメートできる基本型整数の属性に使用されます。 |
| [SVGAnimatedLength](./svganimatedlength/) | アニメーション化できる基本タイプの長さの属性に使用されます。 |
| [SVGAnimatedLengthList](./svganimatedlengthlist/) | アニメーション化できるタイプ SVGLengthList の属性に使用されます。 |
| [SVGAnimatedNumber](./svganimatednumber/) | アニメ化できる基本型番号の属性に使用します. |
| [SVGAnimatedNumberList](./svganimatednumberlist/) | 数値のリストを取り、アニメーション化できる属性に使用されます。 |
| [SVGAnimatedPreserveAspectRatio](./svganimatedpreserveaspectratio/) | アニメート可能なタイプ SVGPreserveAspectRatio の属性に使用されます。 |
| [SVGAnimatedRect](./svganimatedrect/) | アニメート可能なタイプ SVGRect の属性に使用されます。 |
| [SVGAnimatedString](./svganimatedstring/) | アニメーション化できる DOMString 型の属性に使用されます。 |
| [SVGAnimatedTransformList](./svganimatedtransformlist/) | SVG の多くの要素で使用でき、アニメーション化できる「transform」属性など、一連の変換を指定するさまざまな属性に使用されます。 |
| [SVGAnimatedValue&lt;T&gt;](./svganimatedvalue-1/) | アニメートできるタイプの属性に使用されます。 |
| [SVGLength](./svglength/) | SVGLength インターフェイスは長さの基本データ型に対応します。 SVGLength オブジェクトは読み取り専用として指定できます。つまり、オブジェクトを変更しようとすると、以下で説明するように例外がスローされます。 |
| [SVGLengthList](./svglengthlist/) | このインターフェイスは、SVGLength オブジェクトのリストを定義します。 |
| [SVGMatrix](./svgmatrix/) | SVG のグラフィック操作の多くは、 [ace] [bdf] の形式の 2x3 行列を使用します。行列演算のために 3x3 行列に展開すると、 [ace] [bdf] [0 0 1] |
| [SVGNumber](./svgnumber/) | 基本型番号の属性に使用. |
| [SVGNumberList](./svgnumberlist/) | このインターフェイスは、SVGNumber オブジェクトのリストを定義します。 |
| [SVGPoint](./svgpoint/) | SVG DOM インターフェイスの多くは、クラス SVGPoint のオブジェクトを参照します。 SVGPoint は (x, y) 座標ペアです。行列演算で使用される場合、SVGPoint は次の形式のベクトルとして扱われます: [x] [y] [1] 例外がスローされます。 |
| [SVGPointList](./svgpointlist/) | このインターフェイスは、SVGPoint オブジェクトのリストを定義します。 |
| [SVGPreserveAspectRatio](./svgpreserveaspectratio/) | SVGPreserveAspectRatio インターフェースは、一部の SVG 要素で使用できる「preserveAspectRatio」属性に対応します。 |
| [SVGRect](./svgrect/) | 長方形のジオメトリを表します。長方形は、X の最小値、Y の最小値、および幅と高さを識別する (x,y) 座標ペアで構成されるものとして定義されます。これらは通常、負でない値に制限されます。 |
| [SVGStringList](./svgstringlist/) | SVGStringList には、他の SVGxxxList インターフェイスと同じ属性とメソッドがあります。実装者は、さまざまな SVGxxxList インターフェイスを実装するために単一の基本クラスを使用することを検討する場合があります. |
| [SVGTransform](./svgtransform/) | SVGTransform は、SVGTransformList 内のコンポーネント変換の 1 つのインターフェイスです。したがって、SVGTransform オブジェクトは、'transform' 属性仕様内の単一のコンポーネント (例えば、'scale(…)' または 'matrix(…)') に対応します。 |
| [SVGTransformList](./svgtransformlist/) | このインターフェイスは、SVGTransform オブジェクトのリストを定義します。 |
| [SVGValueType](./svgvaluetype/) | SVGValueType タイプは、基本 SVG 値タイプを表すために使用されます。 |
