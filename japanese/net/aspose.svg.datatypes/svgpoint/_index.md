## SVGPoint class

SVG DOM インターフェイスの多くは、クラス SVGPoint のオブジェクトを参照します。 SVGPoint は (x, y) 座標ペアです。行列演算で使用される場合、SVGPoint は次の形式のベクトルとして扱われます: [x] [y] [1] 例外がスローされます。

public class SVGPoint : SVGValueType
