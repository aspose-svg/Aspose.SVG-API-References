## SVGTransform.SetRotate method

変換タイプを SVG_TRANSFORM_ROTATE に設定します。パラメーター angle は回転角度を定義し、パラメーター cx および cy はオプションの回転中心を定義します。

```csharp
public void SetRotate(float angle, float cx, float cy)
```

| パラメータ | タイプ | 説明 |
| --- | --- | --- |
| angle | Single | 回転角度。 |
| cx | Single | 回転中心の x 座標。 |
| cy | Single | 回転中心の y 座標。 |
