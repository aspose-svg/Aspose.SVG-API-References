**Aspose.Svg.Dom.Views**名前空間は、AbstractView および DocumentView インターフェースを記述します Document Object Model Views 仕様に従います.

## インターフェース

| インターフェース | 説明 |
| --- | --- |
| [IAbstractView](./iabstractview/) | すべてのビューが派生する基本インターフェイス。 |
| [IDocumentView](./idocumentview/) | DocumentView インターフェイスは、DOM ビューをサポートする DOM 実装の Document オブジェクトによって実装されます。ドキュメントのデフォルト ビューを取得するための属性を提供します。 |
