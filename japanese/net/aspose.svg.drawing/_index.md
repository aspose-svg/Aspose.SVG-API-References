**Aspose.Svg.Drawing**名前空間には、測定と単位を指定するためのオブジェクトと インターフェイス、およびブラシ、色、フォントなどの 描画属性オブジェクトが含まれています.

## クラス

| クラス | 説明 |
| --- | --- |
| [Angle](./angle/) | 角度データ型 |
| [Color](./color/) | Color クラスでは、色を 赤緑青 (RGB) 値、 色相彩度明度 (HSL) 値、 色相彩度値 (HSV) 値、 色相白度黒度 (HWB) として指定できます。 ) 値、 明度-AB (LAB) 値、 輝度-彩度-色相 (LCH) 値、 シアン-マゼンタ-イエロー-キー (CMYK) 値、 ナチュラル カラー (NCOL) 値、 または色名付き. 透明度を示すアルファチャンネルも利用できます. |
| [Dimension](./dimension/) | ディメンションの基本クラスを提供します。 「次元」という一般的な用語は、数値に単位を付けたものを指し、次のように表されます。[`UnitType`](../aspose.svg.drawing/unittype/) . |
| [Frequency](./frequency/) | 「周波数」単位。 |
| [Length](./length/) | 長さ測定単位を表します。 |
| [LengthOrAuto](./lengthorauto/) | ストレージの長さまたは「自動」単位のコンテナーを表します.. |
| [Margin](./margin/) | ページ余白を表します。 |
| [Numeric](./numeric/) | 数値型の基本クラスを提供します。 |
| [Page](./page/) | 構成出力ページに使用されるページ オブジェクトを表します。 ページ サイズのラクナ値は A4(210x297mm) です。 |
| [Resolution](./resolution/) | 解像度単位を表します。 |
| [Size](./size/) | 高さと幅を指定する値を格納します。 |
| [Time](./time/) | 時間単位を表します。 |
| [Unit](./unit/) | 測定単位の基本クラスを提供します。 |
| [UnitType](./unittype/) | 測定単位を指定します。 |
## インターフェース

| --- | --- |
| [IBrush](./ibrush/) | ブラシタイプの取得方法を宣言します。 |
| [IColorComponents](./icolorcomponents/) | 色成分を処理するためのメソッドとプロパティを宣言します. |
| [IGradientBrush](./igradientbrush/) | グラデーション ブラシの共通プロパティを取得するためのメソッドを宣言します。 |
| [IInterpolationColor](./iinterpolationcolor/) | 補間色を取得するメソッドを宣言します。 |
| [ILinearGradientBrush](./ilineargradientbrush/) | 線形グラデーションでブラシ インターフェイスを定義します。 |
| [ISolidBrush](./isolidbrush/) | 単一色のブラシ インターフェイスを定義します |
| [ITextureBrush](./itexturebrush/) | 画像を使用して形状の内部を塗りつぶすブラシ インターフェイスを定義します。 |
| [ITransformableBrush](./itransformablebrush/) | 変換行列とラップ モードを取得するためのメソッドを宣言します。 |
| [ITrueTypeFont](./itruetypefont/) | TrueType フォントを操作するためのメソッドを宣言します。 |
