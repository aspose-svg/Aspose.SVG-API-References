**Aspose.Svg.Rendering.Image**名前空間は、 固有のデバイス クラスといくつかのレンダリング オプション ラスター形式へのレンダリングを担当するクラスを提供します: jpeg、png、bmp、gif、tiff.

## クラス

| クラス | 説明 |
| --- | --- |
| [ImageDevice](./imagedevice/) | ラスター形式へのレンダリングを表します: jpeg、png、bmp、gif、tiff. |
| [ImageRenderingOptions](./imagerenderingoptions/) | のレンダリング オプションを表します[`ImageDevice`](../aspose.svg.rendering.image/imagedevice/).このオプションは、出力画像フォーマット、圧縮、解像度などを指定するために使用されます. |
| [TextOptions](./textoptions/) | のテキスト レンダリング オプションを表します[`ImageDevice`](../aspose.svg.rendering.image/imagedevice/) . |
## 列挙

| 列挙 | 説明 |
| --- | --- |
| [Compression](./compression/) | Tagged Image File Format (TIFF) ビットマップ イメージの可能な圧縮方式を指定します。 |
| [ImageFormat](./imageformat/) | 画像のファイル形式を指定します。 |
