標準 IO (入出力) tools に加えて、 **Aspose.Svg.IO**名前空間には、ヘルパー クラスとインターフェイスが含まれています。

## クラス

| クラス | 説明 |
| --- | --- |
| [FileCreateStreamProvider](./filecreatestreamprovider/) | を表しますFileStreamレンダリング デバイスにストリームを提供するための実装. |
| [LocalFileSystemStorage](./localfilesystemstorage/) | ファイル システム ストレージ。 |
| [OutputStream](./outputstream/) | サロゲート ストリームは実際の出力ストリームをラップし、それへのアクセスを制御します。 [`OutputStream`](../aspose.svg.io/outputstream/)出力ストリームの場所を記述する URI データが含まれています。 |
| [OutputStreamContext](./outputstreamcontext/) | 出力ストリームの初期化のコンテキスト。 |
## インターフェース

| インターフェース | 説明 |
| --- | --- |
| [IBlob](./iblob/) | Blob オブジェクトはバイト シーケンスを参照し、バイト シーケンスの合計バイト数である size 属性と、バイト シーケンスのメディア タイプを表す小文字の ASCII エンコード文字列である type 属性を持ちます。 . |
| [ICreateStreamProvider](./icreatestreamprovider/) | レンダリング デバイスにストリームを提供するクラスによって実装できるインターフェイスを表します。 |
| [IFile](./ifile/) | File オブジェクトは、文字列である name 属性を持つ Blob オブジェクトです。コンストラクターを介して Web アプリケーション内で作成するか、基になる (OS) ファイル システムのファイルからのバイト シーケンスへの参照です。 |
| [IFileList](./ifilelist/) | ファイルのリストのインターフェイスを表します。 |
| [IOutputStorage](./ioutputstorage/) | 出力ストリームの作成と管理を提供します[`OutputStream`](../aspose.svg.io/outputstream/) . |
