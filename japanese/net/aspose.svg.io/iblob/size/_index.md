## IBlob.Size property

バイト シーケンスのサイズをバイト数で返します。 取得時に、適合するユーザー エージェントは、FileReader または FileReaderSync オブジェクトによって読み取ることができる合計バイト数を返す必要があります。Blob に読み取るバイトがない場合は 0 を返します。 .

public ulong Size { get; }
