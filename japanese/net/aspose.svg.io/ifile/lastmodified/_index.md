## IFile.LastModified property

ファイルの最終更新日。取得時に、ユーザー エージェントがこの情報を利用可能にすることができる場合、 これは、Unix エポックからのミリ秒数として、ファイルが最後に変更された時刻の long long セットを返さなければなりません。

```csharp
public long LastModified { get; }
```
