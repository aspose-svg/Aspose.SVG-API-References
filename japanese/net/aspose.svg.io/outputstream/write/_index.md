## OutputStream.Write method

ラップされた output ストリームに一連のバイトを書き込み、書き込まれた バイト数だけこのストリーム内の現在位置を進めます。

```csharp
public override void Write(byte[] buffer, int offset, int count)
```

| パラメータ | タイプ | 説明 |
| --- | --- | --- |
| buffer | Byte[] | バイトの配列。このメソッドは、count バイトをバッファからラップされた output ストリームにコピーします。 |
| offset | Int32 | ラップされた output ストリームへのバイトのコピーを開始する、バッファー内のゼロから始まるバイト オフセット。 |
| count | Int32 | ラップされた出力ストリームに書き込まれるバイト数。 |
