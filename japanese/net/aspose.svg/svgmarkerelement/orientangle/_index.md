## SVGMarkerElement.OrientAngle property

指定された「マーカー」要素の属性「方向」に対応します。 markerUnits が SVG_MARKER_ORIENT_ANGLE の場合、属性「orient」の角度値。それ以外の場合は、ゼロに設定されます.

```csharp
public SVGAnimatedAngle OrientAngle { get; }
```

### プロパティ値

指定された「マーカー」要素の属性「方向」。
