## IUrlSearchParams.Set method

最初に見つかった名前と値のペアの値を指定された値に設定し、その他を削除します。 指定された名前の名前と値のペアが見つからない場合、新しいペアがリストに追加されます。

```csharp
public void Set(string name, string value)
```

| パラメータ | タイプ | 説明 |
| --- | --- | --- |
| name | String | 名前と値のペアを検索するために使用されるキー。 |
| value | String | 設定される値。 |
