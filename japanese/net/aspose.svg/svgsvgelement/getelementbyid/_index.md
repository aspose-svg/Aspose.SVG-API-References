## SVGSVGElement.GetElementById method

この SVG ドキュメント フラグメントを検索します (つまり、検索はドキュメント ツリーのサブセットに制限されます)。 Element が見つかった場合、その Element が返されます。そのような要素が存在しない場合は、null を返します。複数の要素がこの id. を持つ場合、動作は定義されません。

```csharp
public Element GetElementById(string elementId)
```

| パラメータ | タイプ | 説明 |
| --- | --- | --- |
| elementId | String | 要素の一意の ID 値。 |

### 戻り値

一致する要素。
