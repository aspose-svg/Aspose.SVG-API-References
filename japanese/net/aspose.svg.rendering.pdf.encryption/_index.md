**Aspose.Svg.Rendering.Pdf.Encryption**名前空間には、パスワード、 暗号化アルゴリズムなどの PDF 暗号化の詳細を記述する データ クラスが含まれます。

## クラス

| クラス | 説明 |
| --- | --- |
| [PdfEncryptionInfo](./pdfencryptioninfo/) | PDF 暗号化の詳細が含まれます。 |
## 列挙

| 列挙 | 説明 |
| --- | --- |
| [PdfEncryptionAlgorithm](./pdfencryptionalgorithm/) | 暗号化モード列挙。アルゴリズムとキーの長さを使用して記述します。 この列挙型は、機能をさらに増やすことができるように拡張されています。 この列挙型は、「ベースからコア」パターンを実装します。 |
| [PdfPermissions](./pdfpermissions/) | この列挙型は、PDF に対するユーザーの権限を表します。 |
