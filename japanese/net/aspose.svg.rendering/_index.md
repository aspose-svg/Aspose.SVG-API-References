**Aspose.Svg.Rendering**名前空間は、 多数のレンダラー オブジェクトと、 ドキュメント/ファイルを IDevice 実装にレンダリングする役割を担う適切な 低レベル オプション クラスで構成されます。

## クラス

| クラス | 説明 |
| --- | --- |
| [CssOptions](./cssoptions/) | css レンダリング オプションを表します。 |
| [Device](./device/) | さまざまな形式と環境でグラフィックスを描画するために使用されるレンダリング デバイスを実装するための基本クラスを表します。 |
| [Device&lt;TGraphicContext,TRenderingOptions&gt;](./device-2/) | 特定のレンダリング デバイスを実装するための基本クラスを表します。 |
| [GraphicContext](./graphiccontext/) | 現在のグラフィック コントロール パラメータを保持します。 これらのパラメータは、グラフィック オペレータが実行されるグローバル フレームワークを定義します。 |
| [PageSetup](./pagesetup/) | 構成出力ページセットに使用されるページ設定オブジェクトを表します。 |
| [Renderer](./renderer/) | すべてのレンダラーと実装の基本クラスを表します IDisposable インターフェイス. |
| [Renderer&lt;TDocument&gt;](./renderer-1/) | すべてのレンダラーの抽象クラスを表します. |
| [RenderingOptions](./renderingoptions/) | レンダリング オプションを表します。 |
| [SvgRenderer](./svgrenderer/) | SVG ドキュメント レンダラーを表します。 |
| [TextInfo](./textinfo/) | レンダリングされたテキストに関する情報が含まれています. |
## 構造物

| 構造 | 説明 |
| --- | --- |
| [GlyphInfo](./glyphinfo/) | グリフ関連の情報が含まれています. |
## インターフェース

| インターフェース | 説明 |
| --- | --- |
| [IDevice](./idevice/) | パス、テキスト、画像などのグラフィック要素のカスタム レンダリングをサポートするメソッドとプロパティを定義します。 |
## 列挙

| 列挙 | 説明 |
| --- | --- |
| [AtPagePriority](./atpagepriority/) | ページ サイズ宣言を適用する順序を指定します。 |
| [ImageType](./imagetype/) | 画像のファイル形式を指定します |
| [MediaType](./mediatype/) | レンダリング中に使用可能なメディア タイプを指定します。 |
| [SizingType](./sizingtype/) | ページ サイズの種類の列挙を表します。 |
