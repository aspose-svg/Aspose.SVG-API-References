## CSSPrimitiveValue.GetCounterValue method

このメソッドは、カウンター値を取得するために使用されます。この CSS 値にカウンター値が含まれていない場合、DOMException が発生します。 Counter インターフェイスを使用して、対応するスタイル プロパティを変更できます。

```csharp
public abstract Counter GetCounterValue()
```

### 戻り値

カウンタ値を返します
