## ICSSKeyframesRule.FindRule method

findRule メソッドは、渡されたキーと一致するキーを持つルールを返します。そのようなルールが存在しない場合は、null 値が返されます

```csharp
public ICSSKeyframeRule FindRule(string key)
```

| パラメータ | タイプ | 説明 |
| --- | --- | --- |
| key | String | 検索するルールを記述したキー。キーは 0 から 1 の間の数値に解決される必要があります。そうでない場合、ルールは無視されます。 |

見つかったルール
