## ICSSMediaRule interface

CSSMediaRule インターフェイスは、CSS スタイル シートの @media ルールを表します。 @media ルールを使用して、特定のメディア タイプのスタイル ルールを区切ることができます。

```csharp
public interface ICSSMediaRule : ICSSRule
```
