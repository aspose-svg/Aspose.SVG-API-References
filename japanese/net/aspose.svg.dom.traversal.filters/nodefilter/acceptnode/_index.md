## NodeFilter.AcceptNode method

指定されたノードが a TreeWalker または NodeIterator の論理ビューに表示されるかどうかをテストします。この function は、TreeWalker および NodeIterator の実装によって呼び出されます。通常、 ユーザー コードから直接呼び出されることはありません。 (ただし、same フィルターを使用して独自のアプリケーション ロジックをガイドする場合は、そうすることができます。)

```csharp
public abstract short AcceptNode(Node n)
```

| パラメータ | タイプ | 説明 |
| --- | --- | --- |
| n | Node | ノードを使用して、フィルターを通過するかどうかを確認します。 |

### 戻り値

上記で定義したように、ノードが受け入れられるか、 拒否されるか、スキップされるかを決定する定数.
