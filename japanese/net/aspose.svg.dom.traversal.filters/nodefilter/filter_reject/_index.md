## NodeFilter.FILTER_REJECT field

ノードを拒否します。 NodeIterator または TreeWalker に定義されたナビゲーション メソッドは、 このノードを返しません。 TreeWalker の場合、このノード の子も拒否されます。 NodeIterators は、これを FILTER_SKIP. の シノニムとして扱います。

```csharp
public const short FILTER_REJECT;
```
