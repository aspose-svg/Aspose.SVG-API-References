## NodeFilter.FILTER_SKIP field

この単一ノードをスキップします。 NodeIterator または TreeWalker に定義されたナビゲーション メソッドは、 このノードを返しません。 NodeIterator と TreeWalker の両方で、このノードの子は引き続き と見なされます。

```csharp
public const short FILTER_SKIP;
```
