## ISVGAnimatedPoints.AnimatedPoints property

Fornisce l'accesso ai contenuti animati correnti dell'attributo 'punti'. Se l'attributo o la proprietà specificata viene animata, contiene il valore animato corrente dell'attributo o della proprietà. Se l'attributo o la proprietà specificata non è attualmente animata, contiene lo stesso valore dei punti.

```csharp
public SVGPointList AnimatedPoints { get; }
```

### Valore della proprietà

I punti animati.
