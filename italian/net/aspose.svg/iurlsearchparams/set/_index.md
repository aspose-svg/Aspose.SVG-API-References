## IUrlSearchParams.Set method

Imposta il valore della prima coppia nome-valore trovata sul valore specificato e rimuove le altre. Se non vengono trovate coppie nome-valore con il nome specificato, ne verrà aggiunta una nuova all'elenco.

```csharp
public void Set(string name, string value)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| name | String | Chiave che verrà utilizzata per trovare le coppie nome-valore. |
| value | String | Valore che verrà impostato. |
