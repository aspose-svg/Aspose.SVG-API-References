## SVGViewElement.ViewTarget property

Corrisponde all'attributo 'viewTarget' sull'elemento 'view' dato. Un elenco di valori DOMString che contengono i nomi elencati nell'attributo 'viewTarget'. Ciascuno dei valori DOMString può essere associato all'elemento corrispondente utilizzando la chiamata al metodo getElementById().

```csharp
public SVGStringList ViewTarget { get; }
```

### Valore della proprietà

L'attributo 'viewTarget' del dato elemento 'view'.
