## SVGDocument.Referrer property

Restituisce l'URI della pagina collegata a questa pagina. Il valore è una stringa vuota se l'utente è passato direttamente alla pagina (non tramite un collegamento, ma, ad esempio, tramite un segnalibro).

```csharp
public string Referrer { get; }
```

### Valore della proprietà

Il referrer.
