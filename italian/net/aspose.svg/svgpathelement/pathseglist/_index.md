## SVGPathElement.PathSegList property

Fornisce l'accesso ai contenuti di base (cioè statici) dell'attributo 'd' in una forma che corrisponde uno a uno con la sintassi di SVG. Pertanto, se l'attributo 'd' ha un comando "absolute moveto (M)" e un comando "absolute arcto (A)", pathSegList avrà due voci: un SVG_PATHSEG_MOVETO_ABS e un SVG_PATHSEG_ARC_ABS.

```csharp
public SVGPathSegList PathSegList { get; set; }
```

### Valore della proprietà

L'elenco dei segmenti di percorso.
