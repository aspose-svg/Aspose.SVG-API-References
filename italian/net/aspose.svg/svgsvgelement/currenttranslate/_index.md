## SVGSVGElement.CurrentTranslate property

Su un elemento svg più esterno, il fattore di traduzione corrispondente che tiene conto dell'"ingrandimento" dell'utente. Quando si accede a un elemento 'svg' che non è un elemento svg più esterno, non è definito il comportamento di questo attributo.

```csharp
public SVGPoint CurrentTranslate { get; }
```

### Valore della proprietà

La traduzione corrente.
