## SVGSVGElement.GetCurrentTime method

Restituisce l'ora corrente in secondi relativa all'ora di inizio del frammento di documento SVG corrente. Se getCurrentTime viene chiamato prima dell'inizio della sequenza temporale del documento (ad esempio, tramite uno script in esecuzione in un elemento 'script' prima dell'invio dell'evento SVGLoad del documento), viene restituito 0.

```csharp
public float GetCurrentTime()
```

### Valore di ritorno

L'ora corrente in secondi o 0 se la sequenza temporale del documento non è ancora iniziata.
