## SVGSVGElement.GetCurrentTime method

Restituisce l'ora corrente in secondi relativa all'ora di inizio per il frammento di documento SVG corrente. Se getCurrentTime viene chiamato prima dell'inizio della sequenza temporale del documento (ad esempio, mediante uno script in esecuzione in un elemento 'script' prima che venga inviato l'evento SVGLoad del documento), viene restituito 0.

```csharp
public float GetCurrentTime()
```

### Valore di ritorno

L'ora corrente in secondi o 0 se la sequenza temporale del documento non è ancora iniziata.
