## Metered.SetMeteredKey method

Imposta la chiave pubblica e privata misurata. Se acquisti una licenza misurata, quando avvii l'applicazione, questa API dovrebbe essere chiamata, normalmente, questo è sufficiente. Tuttavia, se non riesci sempre a caricare i dati di consumo e superi le 24 ore, la licenza verrà impostata sullo stato di valutazione, per evitare tale caso, dovresti controllare regolarmente lo stato della licenza, se è lo stato di valutazione, chiama di nuovo questa API.

```csharp
public void SetMeteredKey(string publicKey, string privateKey)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| publicKey | String | chiave pubblica |
| privateKey | String | chiave privata |
