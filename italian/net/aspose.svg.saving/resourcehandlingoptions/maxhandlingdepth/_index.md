## ResourceHandlingOptions.MaxHandlingDepth property

Ottiene o imposta la profondità massima delle pagine che verranno gestite. Profondità 1 significa che verranno gestite solo le pagine direttamente referenziate dal documento salvato. L'impostazione di questa proprietà su -1 comporterà la gestione di tutte le pagine. Il valore predefinito è 0.

```csharp
public int MaxHandlingDepth { get; set; }
```
