## IUserAgentService.Language property

Il`Language` specifica la lingua primaria per i contenuti dell'elemento e per qualsiasi attributo dell'elemento che contiene testo. Il suo valore deve essere un BCP 47 valido ( tag di lingua o la stringa vuota. L'impostazione dell'attributo sulla stringa vuota indica che la lingua principale è sconosciuta.

```csharp
public string Language { get; set; }
```

### Valore della proprietà

La lingua dell'agente utente.
