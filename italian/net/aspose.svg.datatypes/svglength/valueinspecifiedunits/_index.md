## SVGLength.ValueInSpecifiedUnits property

Il valore come valore in virgola mobile, nelle unità espresse da unitType. L'impostazione di questo attributo farà sì che value e valueAsString vengano aggiornati automaticamente per riflettere questa impostazione.

```csharp
public float ValueInSpecifiedUnits { get; set; }
```

### Valore della proprietà

Il valore nelle unità specificate.
