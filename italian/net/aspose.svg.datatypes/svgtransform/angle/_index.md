## SVGTransform.Angle property

Un attributo utile per SVG_TRANSFORM_ROTATE, SVG_TRANSFORM_SKEWX e SVG_TRANSFORM_SKEWY. Mantiene l'angolo specificato. Per SVG_TRANSFORM_MATRIX, SVG_TRANSFORM_TRANSLATE e SVG_TRANSFORM_SCALE, l'angolo sarà zero.

```csharp
public float Angle { get; }
```

### Valore della proprietà

L'angolo specificato.
