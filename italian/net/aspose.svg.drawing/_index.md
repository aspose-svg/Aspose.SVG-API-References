Il **Aspose.Svg.Drawing** lo spazio dei nomi contiene oggetti e interfacce per specificare misure e unità, nonché oggetti attributi di disegno come pennelli, colori e caratteri.

## Classi

| Classe | Descrizione |
| --- | --- |
| [Angle](./angle/) | Il tipo di dati dell'angolo |
| [Color](./color/) | La classe Color consente di specificare i colori come valori Red-Green-Blue (RGB), valori Hue-Saturation-Luminosity (HSL), valori Hue-Saturation-Value (HSV), Hue-Whiteness-Blackness (HWB ) valori, valori lightness-AB (LAB), valori Luminance-Chroma-Hue (LCH), valori Cyan-Magenta-Yellow-Key (CMYK), valori Natural colors (NCOL), o con un nome di colore . È disponibile anche un canale alfa per indicare la trasparenza. |
| [Dimension](./dimension/) | Fornisce la classe base per le dimensioni. Il termine generale 'dimensione' si riferisce a un numero con un'unità allegata e sono denotati da[`UnitType`](../aspose.svg.drawing/unittype/) . |
| [Frequency](./frequency/) | L'unità di 'frequenza'. |
| [Length](./length/) | Rappresenta un'unità di misura della lunghezza. |
| [LengthOrAuto](./lengthorauto/) | Rappresenta un contenitore per la lunghezza di archiviazione o le unità 'auto'.. |
| [Margin](./margin/) | Rappresenta il margine della pagina. |
| [Numeric](./numeric/) | Fornisce la classe base per i tipi numerici. |
| [Page](./page/) | Rappresenta un oggetto pagina utilizzato per la pagina di output della configurazione. Il valore lacuna per la dimensione della pagina è A4(210x297mm) |
| [Resolution](./resolution/) | Rappresenta un'unità di risoluzione. |
| [Size](./size/) | Memorizza un valore che specifica un'altezza e una larghezza. |
| [Time](./time/) | Rappresenta un'unità di tempo. |
| [Unit](./unit/) | Fornisce la classe base per le unità di misura. |
| [UnitType](./unittype/) | Specifica l'unità di misura. |
## Interfacce

| Interfaccia | Descrizione |
| --- | --- |
| [IBrush](./ibrush/) | Dichiara il metodo per ottenere il tipo di pennello. |
| [IColorComponents](./icolorcomponents/) | Dichiara il metodo e le proprietà per l'elaborazione dei componenti del colore. |
| [IGradientBrush](./igradientbrush/) | Dichiara metodi per ottenere proprietà comuni dei pennelli sfumatura. |
| [IInterpolationColor](./iinterpolationcolor/) | Dichiara i metodi per ottenere il colore dell'interpolazione. |
| [ILinearGradientBrush](./ilineargradientbrush/) | Definisce l'interfaccia del pennello con un gradiente lineare. |
| [ISolidBrush](./isolidbrush/) | Definisce l'interfaccia del pennello di un singolo colore |
| [ITextureBrush](./itexturebrush/) | Definisce l'interfaccia del pennello che utilizza un'immagine per riempire l'interno di una forma. |
| [ITransformableBrush](./itransformablebrush/) | Dichiara i metodi per ottenere la matrice di trasformazione e la modalità wrap. |
| [ITrueTypeFont](./itruetypefont/) | Dichiara i metodi per lavorare con il carattere TrueType. |
