## ITrueTypeFont.FullFontName property

Dovrebbe essere una combinazione di "NomeFamiglia" e "Nomesottofamiglia". Eccezione: se il font è "Regular" come indicato in "SubFamilyName", utilizzare solo il cognome contenuto in "FamilyName". Un'eccezione alla definizione precedente di Nome font completo è per le stringhe della piattaforma Microsoft per i font CFF OpenType: in questo caso, la stringa del nome font completo deve essere identica a FontName PostScript nel nome CFF INDEX.

```csharp
public string FullFontName { get; }
```

### Valore di ritorno

Nome completo del carattere
