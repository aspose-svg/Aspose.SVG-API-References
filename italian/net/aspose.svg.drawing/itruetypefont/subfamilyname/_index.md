## ITrueTypeFont.SubFamilyName property

Il nome della sottofamiglia di font distingue il font in un gruppo con lo stesso nome di famiglia di font. Si presume che si tratti di stile (corsivo, obliquo) e peso (chiaro, grassetto, nero, ecc.). Un font senza particolari differenze di peso o di stile (es. peso medio, non corsivo e fsSelection bit 6 impostato) dovrebbe avere la stringa "Regular" memorizzata in questa posizione.

```csharp
public string SubFamilyName { get; }
```

### Valore di ritorno

Nome della sottofamiglia del carattere
