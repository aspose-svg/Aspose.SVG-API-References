## Color.ToNaturalColorString method

Restituisce un colore specificato dai colori naturali (NCol) utilizzando una lettera di colore con un numero per specificare la distanza (in percentuale) dal colore.

```csharp
public string ToNaturalColorString(int digits)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| digits | Int32 | Imposta la precisione di arrotondamento per i componenti colore. |

### Valore di ritorno

Una stringa di colori naturali (NCol).
