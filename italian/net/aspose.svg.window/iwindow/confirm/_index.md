## IWindow.Confirm method

Visualizza un prompt OK/Annulla modale con il messaggio specificato, attende che l'utente lo ignori e restituisce true se l'utente fa clic su OK e false se l'utente fa clic su Annulla.

```csharp
public bool Confirm(string message)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| message | String | Il messaggio. |

### Valore di ritorno

Restituisce true se l'utente fa clic su OK e false se l'utente fa clic su Annulla
