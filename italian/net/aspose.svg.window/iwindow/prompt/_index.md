## IWindow.Prompt method

Visualizza un prompt di campo di testo modale con il messaggio specificato, attende che l'utente lo concluda e restituisce il valore immesso dall'utente. Se l'utente annulla il prompt, restituisce invece null. Se è presente il secondo argomento, il valore specificato viene utilizzato come predefinito.

```csharp
public string Prompt(string message, string @default)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| message | String | Il messaggio. |
| default | String | Il predefinito. |

### Valore di ritorno

Restituisce il valore immesso dall'utente
