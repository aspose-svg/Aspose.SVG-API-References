## IWindow.Parent property

L'attributo IDL padre sull'oggetto Window di un Documento in un contesto di navigazione b deve restituire l'oggetto WindowProxy del contesto di navigazione padre, se presente (ad esempio se b è un contesto di navigazione figlio), oppure l'oggetto WindowProxy della navigazione contesto b stesso, altrimenti (ad esempio se si tratta di un contesto di navigazione di primo livello o di un contesto di navigazione nidificato distaccato).
