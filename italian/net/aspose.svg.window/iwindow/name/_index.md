L'attributo name dell'oggetto Window deve, in fase di acquisizione, restituire il nome corrente del contesto di navigazione e, in fase di impostazione, impostare il nome del contesto di navigazione al nuovo valore.

```csharp
public string Name { get; set; }
```

### Valore della proprietà
