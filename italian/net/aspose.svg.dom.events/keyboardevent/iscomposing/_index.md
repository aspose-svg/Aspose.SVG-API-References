## KeyboardEvent.IsComposing property

true se l'evento chiave si verifica come parte di una sessione di composizione, ovvero dopo un evento compositionstart e prima del corrispondente evento compositionend. Il valore non inizializzato di questo attributo DEVE essere false.

public bool IsComposing { get; }
