## KeyboardEvent.Code property

Il codice contiene una stringa che identifica il tasto fisico premuto. Il valore non è influenzato dal layout di tastiera corrente o dallo stato del modificatore, quindi un particolare tasto restituirà sempre lo stesso valore.

```csharp
public string Code { get; }
```

### Valore della proprietà

Il codice chiave.
