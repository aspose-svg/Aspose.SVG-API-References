## KeyboardEvent.Key property

Il tasto contiene il valore del tasto premuto. Se il valore ha una rappresentazione stampata, DEVE essere una stringa di caratteri Unicode non vuota, conforme all'algoritmo per determinare il valore della chiave definito in questa specifica. Se il valore è una chiave di controllo che non ha una rappresentazione stampata, DEVE essere uno dei valori chiave definiti nel set di valori chiave, come determinato dall'algoritmo per determinare il valore della chiave. Le implementazioni che non sono in grado di identificare una chiave DEVONO utilizzare il valore della chiave Unidentified.

```csharp
public string Key { get; }
```

### Valore della proprietà

Il valore della chiave.
