## WheelEvent.DeltaY property

Nei programmi utente in cui l'azione predefinita dell'evento ruota è lo scorrimento, il valore DEVE essere la misura lungo l'asse y (in pixel, linee o pagine) da scorrere nel caso in cui l'evento non venga annullato. In caso contrario, si tratta di una misurazione specifica dell'implementazione (in pixel, linee o pagine) del movimento di un dispositivo ruota attorno all'asse y.

```csharp
public double DeltaY { get; }
```

### Valore della proprietà

L'attributo deltaY.
