## ErrorEvent.Error property

L'attributo error deve restituire il valore a cui è stato inizializzato. Quando l'oggetto viene creato, questo attributo deve essere inizializzato su null. Ove appropriato, viene impostato sull'oggetto che rappresenta l'errore (ad es. l'oggetto eccezione nel caso di un'eccezione DOM non rilevata).

```csharp
public object Error { get; }
```

### Valore della proprietà

L'errore.
