Il **Aspose.Svg.Dom.Attributes** lo spazio dei nomi contiene classi di attributi unificati che specifica o qualifica qualsiasi caratteristica o comportamento di qualsiasi oggetto (nodo, elemento, ...).

## Classi

| Classe | Descrizione |
| --- | --- |
| [DOMConstructorAttribute](./domconstructorattribute/) | Specifica un costruttore definito dal W3C. |
| [DOMNameAttribute](./domnameattribute/) | Specifica il nome ufficiale dell'oggetto DOM come definito dal W3C. |
| [DOMNamedPropertyGetterAttribute](./domnamedpropertygetterattribute/) | Specifica che il metodo verrà utilizzato come getter di proprietà denominato. |
| [DOMNoInterfaceObjectAttribute](./domnointerfaceobjectattribute/) | Se l'attributo esteso [NoInterfaceObject] viene visualizzato su un'interfaccia, indica che non esisterà un oggetto interfaccia per l'interfaccia nell'associazione ECMAScript. |
| [DOMNullableAttribute](./domnullableattribute/) | Specifica che a un oggetto DOM può essere assegnato un valore nullo. |
| [DOMObjectAttribute](./domobjectattribute/) | Specifica che l'oggetto è contrassegnato con questo attributo è definito dal W3C. |
| [DOMTreatNullAsAttribute](./domtreatnullasattribute/) | Indica che il valore nullo del membro verrà trattato come valore specificato. |
