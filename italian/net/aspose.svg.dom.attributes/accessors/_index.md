## Accessors enumeration

Rappresenta l'enumerazione delle funzioni di accesso ai membri definita dal W3C.

```csharp
[Flags]
public enum Accessors : byte
```

### I valori

| Nome | Valore | Descrizione |
| --- | --- | --- |
| None | `0` | Specifica che la proprietà non ha alcun significato speciale. |
| Getter | `1` | Specifica che la proprietà o il metodo devono essere gestiti come getter. |
| Setter | `2` | Specifica che la proprietà o il metodo devono essere gestiti come setter. |
| Deleter | `4` | Specifica che la proprietà o il metodo devono essere gestiti da delete. |
