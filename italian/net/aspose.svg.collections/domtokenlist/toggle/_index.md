## Toggle(string, bool?) {#toggle_1}

Rimuove il token dall'elenco se esiste o aggiunge il token all'elenco se non esiste.

```csharp
public bool Toggle(string token, bool? force)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| token | String | Il token che vuoi attivare/disattivare. |
| force | Nullable`1 | Un valore booleano che, se incluso, trasforma l'interruttore in un'operazione unidirezionale. Se impostato su false, il token verrà solo rimosso, ma non aggiunto. Se impostato su true, il token verrà solo aggiunto, ma non rimosso. |

### Valore di ritorno

Valore booleano che indica se il token è nell'elenco dopo la chiamata.
