## DOMTokenList class

La classe DOMTokenList rappresenta un insieme di token separati da spazi. Viene indicizzato a partire da 0 come con gli oggetti JavaScript Array. DOMTokenList fa sempre distinzione tra maiuscole e minuscole.

```csharp
public class DOMTokenList : DOMObject, IEnumerable<string>
```
