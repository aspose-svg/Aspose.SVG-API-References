Il **Aspose.Svg.Collections** namespace è costituito da classi per rappresentare, archiviare e manipolare nodi ed elementi.

## Classi

| Classe | Descrizione |
| --- | --- |
| [DOMTokenList](./domtokenlist/) | La classe DOMTokenList rappresenta un insieme di token separati da spazi. Viene indicizzato a partire da 0 come con gli oggetti JavaScript Array. DOMTokenList fa sempre distinzione tra maiuscole e minuscole. |
| [HTMLCollection](./htmlcollection/) | Il[`HTMLCollection`](../aspose.svg.collections/htmlcollection/) rappresenta una raccolta generica di[`Element`](../aspose.svg.dom/element/) . |
| [NamedNodeMap](./namednodemap/) | Rappresenta raccolte di attributi a cui è possibile accedere per nome. |
| [NodeList](./nodelist/) | NodeList fornisce l'astrazione di una raccolta ordinata di nodi, senza definire o vincolare il modo in cui questa raccolta viene implementata. |
| [SVGListBase&lt;T&gt;](./svglistbase-1/) | Questa interfaccia definisce un elenco di base di tutti gli elenchi SVG. |
