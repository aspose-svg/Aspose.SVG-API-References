## IFile.LastModified property

La data dell'ultima modifica del file. Al momento dell'ottenimento, se i programmi utente possono rendere disponibili queste informazioni, questo deve restituire un long long impostato sull'ora in cui il file è stato modificato l'ultima volta come numero di millisecondi dall'epoca Unix.

```csharp
public long LastModified { get; }
```
