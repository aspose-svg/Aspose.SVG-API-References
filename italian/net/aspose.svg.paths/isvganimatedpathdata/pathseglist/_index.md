## ISVGAnimatedPathData.PathSegList property

Fornisce l'accesso al contenuto di base (cioè statico) dell'attributo 'd' in una forma che corrisponde uno a uno con la sintassi di SVG. Pertanto, se l'attributo 'd' ha un comando "absolute moveto (M)" e un "absolute arcto (A)", pathSegList avrà due voci: un SVG_PATHSEG_MOVETO_ABS e un SVG_PATHSEG_ARC_ABS.

```csharp
public SVGPathSegList PathSegList { get; }
```

### Valore della proprietà

L'elenco dei segmenti di percorso.
