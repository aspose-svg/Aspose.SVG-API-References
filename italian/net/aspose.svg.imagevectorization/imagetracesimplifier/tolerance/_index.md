## ImageTraceSimplifier.Tolerance property

Il valore della tolleranza determina la massima tolleranza di errore consentita per eliminare un punto dalla traccia. Deve essere compreso nell'intervallo da 0 a 4. Qualsiasi valore superiore o inferiore sarà allineato con i valori minimo e massimo di questo intervallo, di conseguenza. Il valore predefinito è 0.3.

```csharp
public float Tolerance { get; set; }
```
