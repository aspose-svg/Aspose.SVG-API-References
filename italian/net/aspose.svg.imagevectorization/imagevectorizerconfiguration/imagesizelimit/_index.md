## ImageVectorizerConfiguration.ImageSizeLimit property

Ottiene o imposta la dimensione massima dell'immagine determinata dalla moltiplicazione larghezza e altezza dell'immagine. La dimensione dell'immagine verrà ridimensionata in base a questa proprietà. Il valore predefinito è 1800000.

```csharp
public int ImageSizeLimit { get; set; }
```
