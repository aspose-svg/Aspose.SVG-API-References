## ImageVectorizerConfiguration class

Il`ImageVectorizerConfiguration` class definisce una configurazione di metodi e opzioni di vettorizzazione delle immagini. La configurazione viene utilizzata per inizializzare un ImageVectorizer e fornisce le opzioni di configurazione per la vettorializzazione delle immagini.
