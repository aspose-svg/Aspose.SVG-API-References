Il **Aspose.Svg.Rendering**lo spazio dei nomi è costituito da numerosi oggetti renderer e da appropriate classi di opzioni di basso livello che sono responsabili del rendering di documenti/file nell'implementazione IDevice.

## Classi

| Classe | Descrizione |
| --- | --- |
| [CssOptions](./cssoptions/) | Rappresenta le opzioni di rendering css. |
| [Device](./device/) | Rappresenta una classe base per l'implementazione di dispositivi di rendering utilizzati per disegnare grafica in vari formati e ambienti. |
| [Device&lt;TGraphicContext,TRenderingOptions&gt;](./device-2/) | Rappresenta la classe base per l'implementazione di particolari dispositivi di rendering. |
| [GraphicContext](./graphiccontext/) | Contiene i parametri di controllo della grafica correnti. Questi parametri definiscono la struttura globale all'interno della quale vengono eseguiti gli operatori grafici. |
| [PageSetup](./pagesetup/) | Rappresenta un oggetto di configurazione della pagina utilizzato per il set di pagine di output della configurazione. |
| [Renderer](./renderer/) | Rappresenta una classe base per tutti i renderer e implementa l'interfaccia IDisposable. |
| [Renderer&lt;TDocument&gt;](./renderer-1/) | Rappresenta una classe astratta per tutti i renderer. |
| [RenderingOptions](./renderingoptions/) | Rappresenta le opzioni di rendering. |
| [SvgRenderer](./svgrenderer/) | Rappresenta il renderer di documenti SVG. |
| [TextInfo](./textinfo/) | Contiene informazioni sul testo renderizzato. |
