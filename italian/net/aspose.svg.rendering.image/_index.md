Il **Aspose.Svg.Rendering.Image** lo spazio dei nomi fornisce una classe di dispositivo specifica e alcune opzioni di rendering classi responsabili del rendering nei formati raster: jpeg, png, bmp, gif, tiff.

## Classi

| Classe | Descrizione |
| --- | --- |
| [ImageDevice](./imagedevice/) | Rappresenta il rendering nei formati raster: jpeg, png, bmp, gif, tiff. |
| [ImageRenderingOptions](./imagerenderingoptions/) | Rappresenta le opzioni di rendering per[`ImageDevice`](../aspose.svg.rendering.image/imagedevice/) . Questa opzione viene utilizzata per specificare il formato dell'immagine di output, la compressione, la risoluzione ecc. |
| [TextOptions](./textoptions/) | Rappresenta le opzioni di rendering del testo per[`ImageDevice`](../aspose.svg.rendering.image/imagedevice/) . |
## Enumerazione

| Enumerazione | Descrizione |
| --- | --- |
| [Compression](./compression/) | Specifica i possibili schemi di compressione per le immagini bitmap TIFF (Tagged Image File Format). |
| [ImageFormat](./imageformat/) | Specifica il formato file dell'immagine. |
