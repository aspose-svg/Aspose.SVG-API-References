## INodeIterator.NextNode method

Restituisce il nodo successivo nel set e fa avanzare la posizione dell'iteratore nel set. Dopo aver creato un NodeIterator, la prima chiamata a nextNode() restituisce il primo nodo in il set.

```csharp
public Node NextNode()
```

### Valore di ritorno

Il nodo successivo nel set su cui viene ripetuta, oppure null se non ci sono più membri in quel set.
