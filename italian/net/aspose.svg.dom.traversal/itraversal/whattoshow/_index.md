## ITraversal.WhatToShow property

Questo attributo determina quali tipi di nodi vengono presentati tramite l'iteratore . Il set di costanti disponibile è definito nell'interfaccia NodeFilter. I nodi non accettati da whatToShow verranno saltati, ma i loro figli possono ancora essere presi in considerazione. Nota che questo salto ha la precedenza sul filtro, se presente.

```csharp
public long WhatToShow { get; }
```

### Valore della proprietà

Cosa mostrare.
