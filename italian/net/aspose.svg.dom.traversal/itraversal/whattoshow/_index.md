## ITraversal.WhatToShow property

Questo attributo determina quali tipi di nodo vengono presentati tramite l' iteratore. Il set di costanti disponibile è definito nell'interfaccia NodeFilter. I nodi non accettati da whatToShow verranno saltati, ma i loro figli potrebbero comunque essere presi in considerazione. Tieni presente che questo salto ha la precedenza sul filtro, se presente.

```csharp
public long WhatToShow { get; }
```

### Valore della proprietà

Cosa mostrare.
