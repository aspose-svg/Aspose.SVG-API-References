## Node.InsertBefore method

Inserisce il nodo prima del nodo figlio esistente figlio. Se child è null, inserisci il nodo alla fine dell'elenco dei child. Se child è un oggetto DocumentFragment, tutti i suoi child vengono inseriti, nello stesso ordine, prima di child. Se il figlio è già nell'albero, viene prima rimosso.

```csharp
public Node InsertBefore(Node node, Node child)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| node | Node | Il nuovo bambino. |
| child | Node | Il bambino ref. |

### Valore di ritorno

Restituisce il nodo inserito
