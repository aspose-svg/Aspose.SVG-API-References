## Element.SetIdAttribute method

Se il parametro isId è vero, questo metodo dichiara che l'attributo specificato è un attributo ID determinato dall'utente.

```csharp
public void SetIdAttribute(string name, bool isId)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| name | String | Il nome dell'attributo. |
| isId | Boolean | se impostato su`VERO` [è id]. |
