## Element.HasAttribute method

Restituisce true quando un attributo con un determinato nome è specificato su questo elemento o ha un valore predefinito, false in caso contrario.

```csharp
public bool HasAttribute(string name)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| name | String | Il nome dell'attributo. |

### Valore di ritorno

`VERO` se il nome specificato ha attributo; Altrimenti,`falso` .
