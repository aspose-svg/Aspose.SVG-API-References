## Document.StrictErrorChecking property

Un attributo che specifica se il controllo degli errori è applicato o meno. Se impostata su false, l'implementazione è libera di non testare tutti i possibili casi di errore normalmente definiti nelle operazioni DOM e di non generare alcuna DOMException sulle operazioni DOM o di segnalare errori durante l'utilizzo di Document.normalizeDocument(). In caso di errore, il comportamento è indefinito. Questo attributo è true per impostazione predefinita.

```csharp
public bool StrictErrorChecking { get; set; }
```

### Valore della proprietà

`VERO` se rigoroso controllo degli errori; altrimenti,`falso` .
