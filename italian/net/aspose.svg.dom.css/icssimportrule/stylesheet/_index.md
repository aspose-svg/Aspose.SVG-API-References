## ICSSImportRule.StyleSheet property

Il foglio di stile a cui fa riferimento questa regola, se è stato caricato. Il valore di questo attributo è null se il foglio di stile non è stato ancora caricato o se non verrà caricato (ad esempio se il foglio di stile è per un tipo di supporto non supportato dall'agente utente).

```csharp
public ICSSStyleSheet StyleSheet { get; }
```
