## ICSSStyleDeclaration interface

L'interfaccia CSSStyleDeclaration rappresenta un singolo blocco di dichiarazione CSS. Questa interfaccia può essere utilizzata per determinare le proprietà di stile attualmente impostate in un blocco o per impostare le proprietà di stile in modo esplicito all'interno del blocco.

```csharp
public interface ICSSStyleDeclaration : ICSS2Properties, IEnumerable<string>
```

## Proprietà

| Nome | Descrizione |
| --- | --- |
| [CSSText](../../aspose.svg.dom.css/icssstyledeclaration/csstext) { get; set; } | La rappresentazione testuale analizzabile del blocco di dichiarazione (escluse le parentesi graffe circostanti). L'impostazione di questo attributo comporterà l'analisi del nuovo valore e il ripristino di tutte le proprietà nel blocco di dichiarazione inclusa la rimozione o l'aggiunta di proprietà. |
| [Item](../../aspose.svg.dom.css/icssstyledeclaration/item) { get; } | Utilizzato per recuperare le proprietà che sono state impostate in modo esplicito in questo blocco di dichiarazione. L'ordine delle proprietà recuperate utilizzando questo metodo non deve essere necessariamente l'ordine in cui sono state impostate. Questo metodo può essere utilizzato per scorrere tutte le proprietà in questo blocco di dichiarazione. |
| [Length](../../aspose.svg.dom.css/icssstyledeclaration/length) { get; } | Il numero di proprietà che sono state impostate in modo esplicito in questo blocco di dichiarazione. L'intervallo di indici validi è compreso tra 0 e lunghezza-1 compreso. |
| [ParentRule](../../aspose.svg.dom.css/icssstyledeclaration/parentrule) { get; } | La regola CSS che contiene questa dichiarazione blocca o null se questa CSSStyleDeclaration non è collegata a una CSSRule. |
