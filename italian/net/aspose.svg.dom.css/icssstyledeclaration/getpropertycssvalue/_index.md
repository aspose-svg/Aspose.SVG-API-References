## ICSSStyleDeclaration.GetPropertyCSSValue method

Utilizzato per recuperare la rappresentazione dell'oggetto del valore di una proprietà CSS se è stata impostata in modo esplicito all'interno di questo blocco di dichiarazione. Questo metodo restituisce null se la proprietà è una proprietà abbreviata. È possibile accedere e modificare i valori delle proprietà abbreviate solo come stringhe, utilizzando i metodi getPropertyValue e setProperty.

```csharp
public CSSValue GetPropertyCSSValue(string propertyName)
```

| --- | --- | --- |
| propertyName | String | Nome della proprietà. |

### Valore di ritorno
