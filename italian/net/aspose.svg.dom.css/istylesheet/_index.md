## IStyleSheet interface

L'interfaccia StyleSheet è l'interfaccia di base astratta per qualsiasi tipo di foglio di stile. Rappresenta un singolo foglio di stile associato a un documento strutturato.

```csharp
public interface IStyleSheet
```

## Proprietà

| Nome | Descrizione |
| --- | --- |
| [Disabled](../../aspose.svg.dom.css/istylesheet/disabled) { get; set; } | false se il foglio di stile è applicato al documento. vero se non lo è. La modifica di questo attributo può causare una nuova risoluzione dello stile per il documento. Un foglio di stile si applica solo se è presente sia una definizione del supporto appropriata sia se l'attributo disabilitato è false.
