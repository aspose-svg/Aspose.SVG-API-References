## IStyleSheet.ParentStyleSheet property

Per i linguaggi di fogli di stile che supportano il concetto di inclusione di fogli di stile, questo attributo rappresenta il foglio di stile incluso, se esistente. Se il foglio di stile è un foglio di stile di primo livello o il linguaggio del foglio di stile non supporta l'inclusione, il valore di questo attributo è null.

public IStyleSheet ParentStyleSheet { get; }
