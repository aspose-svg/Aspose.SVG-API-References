## IStyleSheet.Disabled property

false se il foglio di stile è applicato al documento. vero se non lo è. La modifica di questo attributo può causare una nuova risoluzione dello stile per il documento. Un foglio di stile si applica solo se è presente una definizione media appropriata e l'attributo disabled è falso. Quindi, se il supporto non si applica all'agente utente corrente, l'attributo disabilitato viene ignorato.

```csharp
public bool Disabled { get; set; }
```

### Valore della proprietà

`VERO` se disabilitato; Altrimenti,`falso` .
