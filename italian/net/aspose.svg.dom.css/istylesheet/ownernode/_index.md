## IStyleSheet.OwnerNode property

Il nodo che associa questo foglio di stile al documento. Per HTML, questo può essere l'elemento LINK o STYLE corrispondente. Per XML, potrebbe essere l'istruzione di elaborazione del collegamento. Per i fogli di stile inclusi da altri fogli di stile, il valore di questo attributo è null.

public Node OwnerNode { get; }
