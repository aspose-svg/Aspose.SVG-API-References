## ICSSKeyframesRule.FindRule method

Il metodo findRule restituisce la regola con una chiave che corrisponde alla chiave passata. Se tale regola non esiste, viene restituito un valore null

```csharp
public ICSSKeyframeRule FindRule(string key)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| key | String | La chiave che descriveva la regola da trovare. La chiave deve risolversi in un numero compreso tra 0 e 1, altrimenti la regola viene ignorata. |
