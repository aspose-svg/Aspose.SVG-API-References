## ICSSKeyframesRule interface

L'interfaccia CSSKeyframesRule rappresenta un set completo di fotogrammi chiave per una singola animazione

```csharp
public interface ICSSKeyframesRule : ICSSRule
```

## Proprietà

| Nome | Descrizione |
| --- | --- |
| [CSSRules](../../aspose.svg.dom.css/icsskeyframesrule/cssrules/) { get; } | Questo attributo dà accesso ai fotogrammi chiave nell'elenco |
| [Name](../../aspose.svg.dom.css/icsskeyframesrule/name/) { get; } | Questo attributo è il nome dei fotogrammi chiave, utilizzato dalla proprietà 'animation-name'. |

## Metodi

| Nome | Descrizione |
| --- | --- |
| [AppendRule](../../aspose.svg.dom.css/icsskeyframesrule/appendrule/)(string) | Il metodo appendRule aggiunge la CSSKeyframeRule passata nell'elenco alla chiave passata |
| [DeleteRule](../../aspose.svg.dom.css/icsskeyframesrule/deleterule/)(string) | Il metodo deleteRule elimina CSSKeyframeRule con la chiave passata. Se una regola con questa chiave non esiste, il metodo non fa nulla |
| [FindRule](../../aspose.svg.dom.css/icsskeyframesrule/findrule/)(string) | Il metodo findRule restituisce la regola con una chiave corrispondente alla chiave passata. Se tale regola non esiste, viene restituito un valore null |
