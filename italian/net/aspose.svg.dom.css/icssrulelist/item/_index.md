## ICSSRuleList indexer

Utilizzato per recuperare una regola CSS tramite il metodo item() (http://www.w3.org/TR/DOM-Level-2-Style/css.html#CSS-CSSRuleList). L'ordine in questa raccolta rappresenta l'ordine delle regole nel foglio di stile CSS. Se index è maggiore o uguale al numero di regole nell'elenco, restituisce null.

```csharp
public ICSSRule this[int index] { get; }
```

| Parametro | Descrizione |
| --- | --- |
| index | L'indice delle regole |

### Valore di ritorno

L'istanza della regola CSS.

### Valore della proprietà

Il[`ICSSRule`](../../icssrule/) .
