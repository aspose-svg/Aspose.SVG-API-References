## IElementCSSInlineStyle interface

Le informazioni sullo stile in linea associate agli elementi vengono esposte tramite l'attributo style. Rappresenta il contenuto dell'attributo STYLE per gli elementi HTML (o elementi in altri schemi o DTD che utilizzano l'attributo STYLE allo stesso modo).
