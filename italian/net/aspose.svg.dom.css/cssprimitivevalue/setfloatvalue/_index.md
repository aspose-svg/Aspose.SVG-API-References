## CSSPrimitiveValue.SetFloatValue method

Un metodo per impostare il valore float con un'unità specificata. Se la proprietà associata a questo valore non può accettare l'unità specificata o il valore float, il valore rimarrà invariato e verrà sollevata una DOMException.

```csharp
public abstract void SetFloatValue(ushort unitType, float floatValue)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| unitType | UInt16 | Tipo di unità. |
| floatValue | Single | Il valore float. |
