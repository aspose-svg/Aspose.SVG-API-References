## CSSPrimitiveValue.GetRectValue method

Questo metodo viene utilizzato per ottenere il valore Rect. Se questo valore CSS non contiene un valore rect, viene sollevata un'eccezione DOMException. La modifica della proprietà di stile corrispondente può essere ottenuta utilizzando l'interfaccia Rect.

```csharp
public abstract Rect GetRectValue()
```

### Valore di ritorno

Restituisce il valore Rect
