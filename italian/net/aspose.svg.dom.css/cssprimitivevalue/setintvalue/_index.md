## CSSPrimitiveValue.SetIntValue method

Un metodo per impostare il valore int con un'unità specificata. Se la proprietà associata a questo valore non può accettare l'unità specificata o il valore int, il valore rimarrà invariato e verrà sollevata un'eccezione DOMException.

```csharp
public abstract void SetIntValue(ushort unitType, int intValue)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| unitType | UInt16 | Tipo di unità. |
| intValue | Int32 | Il valore int. |
