## CSSPrimitiveValue.GetCounterValue method

Questo metodo viene utilizzato per ottenere il valore del contatore. Se questo valore CSS non contiene un valore contatore, viene sollevata un'eccezione DOMException. La modifica della proprietà di stile corrispondente può essere ottenuta utilizzando l'interfaccia Counter.

```csharp
public abstract Counter GetCounterValue()
```

### Valore di ritorno

Restituisce il valore del contatore
