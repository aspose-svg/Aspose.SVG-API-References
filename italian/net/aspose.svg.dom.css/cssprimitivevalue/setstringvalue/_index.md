## CSSPrimitiveValue.SetStringValue method

Un metodo per impostare il valore della stringa con l'unità specificata. Se la proprietà associata a questo valore non può accettare l'unità specificata o il valore stringa, il valore rimarrà invariato e verrà sollevata un'eccezione DOMException.

```csharp
public abstract void SetStringValue(ushort stringType, string stringValue)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| stringType | UInt16 | Tipo di stringa. |
| stringValue | String | Il valore della stringa. |
