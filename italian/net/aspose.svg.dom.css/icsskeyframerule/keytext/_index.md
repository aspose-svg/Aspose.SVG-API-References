## ICSSKeyframeRule.KeyText property

Questo attributo rappresenta il selettore di fotogrammi chiave come un elenco di valori percentuali separati da virgole. Le parole chiave "da" e "a" corrispondono rispettivamente a "0%" e "100%".

```csharp
public string KeyText { get; }
```
