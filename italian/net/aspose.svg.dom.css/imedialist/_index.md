## IMediaList interface

L'interfaccia MediaList fornisce l'astrazione di una raccolta ordinata di media, senza definire o vincolare il modo in cui questa raccolta viene implementata. Un elenco vuoto è uguale a un elenco che contiene il supporto "tutto".

```csharp
public interface IMediaList : IEnumerable<string>
```
