## ICSSRule.ParentRule property

Se questa regola è contenuta all'interno di un'altra regola (ad es. una regola di stile all'interno di un blocco @media), questa è la regola che la contiene. Se questa regola non è nidificata all'interno di altre regole, restituisce null.
