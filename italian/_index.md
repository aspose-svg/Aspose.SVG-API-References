## Aspose.SVG per .NET

{{% alert color="primary" %}} 

![Logo del prodotto Aspose.SVG per .NET](../home_1.png)


Aspose.SVG per .NET API è una libreria multipiattaforma che fornisce un'ampia gamma di funzionalità per l'elaborazione e il rendering di documenti SVG. L'API mantiene il Document Object Model (DOM) ed è completamente compatibile con le specifiche SVG ufficiali, consentendoti di avere il controllo totale sui nodi SVG e sulle loro proprietà. Puoi facilmente modificare l'albero del documento, aggiungere e rimuovere nodi, modificare le proprietà del nodo come descritto nelle specifiche ufficiali. Utilizzando l'API Aspose.SVG puoi caricare, salvare e convertire rapidamente SVG, nonché leggere e attraversare gli elementi del file. L'API è indipendente da qualsiasi altro software. Ti consente di lavorare con i file SVG su tutte le piattaforme che supportano .NET (Windows, Linux e macOS).
