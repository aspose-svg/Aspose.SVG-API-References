## Node.LocalName property

Returns the local part of the qualified name of this node. For nodes of any type other than [`ELEMENT_NODE`](../element_node/) and [`ATTRIBUTE_NODE`](../attribute_node/) and nodes created with a DOM Level 1 method, such as [`CreateElement`](../../document/createelement/), this is always null.

```csharp
public virtual string LocalName { get; }
```

### Property Value

A string representing the local part of the element's qualified name.
