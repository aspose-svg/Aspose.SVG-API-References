## IDocumentEvent class

The [`IDocumentEvent`](/svg/python-net/aspose.svg.dom.events/idocumentevent) interface provides a mechanism by which the user can create an [`Event`](/svg/python-net/aspose.svg.dom.events/event) of a type supported by the implementation.



The IDocumentEvent type exposes the following members:

### Methods
| Method | Description |
| :- | :- |
| [create_event](/svg/python-net/aspose.svg.dom.events/idocumentevent/create_event/#str) | Creates an [`Event`](/svg/python-net/aspose.svg.dom.events/event) of a type supported by the implementation. |



### Remarks 


It is expected that the [`IDocumentEvent`](/svg/python-net/aspose.svg.dom.events/idocumentevent) interface will be implemented on the same object which implements the
[`Document`](/svg/python-net/aspose.svg.dom/document)
interface in an implementation which supports the Event model.
