## ICSSPageRule class

The CSSPageRule interface represents a @page rule within a CSS style sheet. The @page rule is used to specify the dimensions, orientation, margins, etc. of a page box for paged media.



The ICSSPageRule type exposes the following members:

### Properties
| Property | Description |
| :- | :- |
| [selector_text](/svg/python-net/aspose.svg.dom.css/icsspagerule/selector_text) | The parsable textual representation of the page selector for the rule. |
| [style](/svg/python-net/aspose.svg.dom.css/icsspagerule/style) | The declaration-block of this rule. |
| [type](/svg/python-net/aspose.svg.dom.css/icsspagerule/type) | The type of the rule, as defined above. The expectation is that binding-specific casting methods can be used to cast down from an instance of the CSSRule interface to the specific derived interface implied by the type. |
