## ICSSStyleRule class

The CSSStyleRule interface represents a single rule set in a CSS style sheet.



The ICSSStyleRule type exposes the following members:

### Properties
| Property | Description |
| :- | :- |
| [selector_text](/svg/python-net/aspose.svg.dom.css/icssstylerule/selector_text) | The textual representation of the selector for the rule set. The implementation may have stripped out insignificant whitespace while parsing the selector. |
| [style](/svg/python-net/aspose.svg.dom.css/icssstylerule/style) | The declaration-block of this rule set. |
| [type](/svg/python-net/aspose.svg.dom.css/icssstylerule/type) | The type of the rule, as defined above. The expectation is that binding-specific casting methods can be used to cast down from an instance of the CSSRule interface to the specific derived interface implied by the type. |
| [css_text](/svg/python-net/aspose.svg.dom.css/icssstylerule/css_text) | The parsable textual representation of the rule. This reflects the current state of the rule and not its initial value. |
| [parent_style_sheet](/svg/python-net/aspose.svg.dom.css/icssstylerule/parent_style_sheet) | The style sheet that contains this rule. |
| [parent_rule](/svg/python-net/aspose.svg.dom.css/icssstylerule/parent_rule) | If this rule is contained inside another rule (e.g. a style rule inside an @media block), this is the containing rule. If this rule is not nested inside any other rules, this returns null. |
