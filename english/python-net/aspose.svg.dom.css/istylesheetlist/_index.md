## IStyleSheetList class

The StyleSheetList interface provides the abstraction of an ordered collection of style sheets.



The IStyleSheetList type exposes the following members:

### Properties
| Property | Description |
| :- | :- |
| [length](/svg/python-net/aspose.svg.dom.css/istylesheetlist/length) | The number of StyleSheets in the list. The range of valid child stylesheet indices is 0 to length-1 inclusive. |



Used to retrieve a style sheet by method item(int index) accordind to http://www.w3.org/TR/2000/REC-DOM-Level-2-Style-20001113/stylesheets.html.
