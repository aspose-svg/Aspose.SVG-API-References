## SVGMaskElementBuilder class

Builder class for constructing an SVG 'mask' element, which is used to define an alpha mask for compositing the current object into the background.
This class enables the building of content within the 'mask' element and provides methods to set various attributes specific to the 'mask' element in SVG.
