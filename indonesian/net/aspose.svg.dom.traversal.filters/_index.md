Namespace yang diwakili oleh fungsi pemfilteran node. Filter adalah objek yang mengetahui cara "menyaring" node. Jika iterator node atau tree walker diberi filter node, ia menerapkan filter sebelum mengembalikan node berikutnya. Jika filter mengatakan untuk menerima node, traversal logic mengembalikannya; jika tidak, traversal mencari node berikutnya dan berpura-pura bahwa node yang ditolak tidak ada.
