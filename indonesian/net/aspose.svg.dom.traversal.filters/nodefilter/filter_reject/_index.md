## NodeFilter.FILTER_REJECT field

Menolak node. Metode navigasi yang ditentukan untuk NodeIterator atau TreeWalker tidak akan mengembalikan node ini. Untuk TreeWalker, anak dari node ini juga akan ditolak. NodeIterator memperlakukan ini sebagai sinonim untuk FILTER_SKIP.

public const short FILTER_REJECT;
