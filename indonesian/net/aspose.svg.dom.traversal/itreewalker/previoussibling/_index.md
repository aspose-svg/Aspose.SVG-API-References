## ITreeWalker.PreviousSibling method

Memindahkan TreeWalker ke saudara sebelumnya dari node saat ini, dan mengembalikan node baru. Jika no no node saat ini terlihat saudara sebelumnya, kembalikan null, dan pertahankan current node.

```csharp
public Node PreviousSibling()
```

### Nilai Pengembalian

Node baru, atau null jika node saat ini tidak memiliki saudara kandung sebelumnya. dalam tampilan logis TreeWalker.
