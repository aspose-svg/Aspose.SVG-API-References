Namespace menyediakan akses ke semua tipe data spesifik SVG.

## Kelas

| Kelas | Keterangan |
| --- | --- |
| [SVGAngle](./svgangle/) | Antarmuka SVGAngle sesuai dengan tipe data dasar sudut. |
| [SVGAnimatedAngle](./svganimatedangle/) | Digunakan untuk atribut sudut tipe data dasar yang dapat dianimasikan. |
| [SVGAnimatedBoolean](./svganimatedboolean/) | Digunakan untuk atribut bertipe boolean yang dapat dianimasikan. |
| [SVGAnimatedEnumeration](./svganimatedenumeration/) | Digunakan untuk atribut yang nilainya harus berupa konstanta dari enumerasi tertentu dan dapat dianimasikan. |
| [SVGAnimatedInteger](./svganimatedinteger/) | Digunakan untuk atribut bilangan bulat tipe dasar yang dapat dianimasikan. |
| [SVGAnimatedLength](./svganimatedlength/) | Digunakan untuk atribut dengan panjang tipe dasar yang dapat dianimasikan. |
| [SVGAnimatedLengthList](./svganimatedlengthlist/) | Digunakan untuk atribut tipe SVGLengthList yang dapat dianimasikan. |
| [SVGAnimatedNumber](./svganimatednumber/) | Digunakan untuk atribut nomor tipe dasar yang dapat dianimasikan. |
| [SVGAnimatedNumberList](./svganimatednumberlist/) | Digunakan untuk atribut yang mengambil daftar angka dan dapat dianimasikan. |
| [SVGAnimatedPreserveAspectRatio](./svganimatedpreserveaspectratio/) | Digunakan untuk atribut tipe SVGPreserveAspectRatio yang dapat dianimasikan. |
| [SVGAnimatedRect](./svganimatedrect/) | Digunakan untuk atribut tipe SVGRect yang dapat dianimasikan. |
| [SVGAnimatedString](./svganimatedstring/) | Digunakan untuk atribut bertipe DOMString yang dapat dianimasikan. |
| [SVGAnimatedTransformList](./svganimatedtransformlist/) | Digunakan untuk berbagai atribut yang menentukan sekumpulan transformasi, seperti atribut 'transform' yang tersedia untuk banyak elemen SVG, dan yang dapat dianimasikan. |
| [SVGAnimatedValue&lt;T&gt;](./svganimatedvalue-1/) | Digunakan untuk atribut tipe yang dapat dianimasikan. |
| [SVGLength](./svglength/) | Antarmuka SVGLength sesuai dengan panjang tipe data dasar. Objek SVGLength dapat ditetapkan sebagai hanya baca, yang berarti bahwa upaya untuk mengubah objek akan menghasilkan pengecualian, seperti dijelaskan di bawah. |
| [SVGLengthList](./svglengthlist/) | Antarmuka ini mendefinisikan daftar objek Panjang SVGL. |
| [SVGMatrix](./svgmatrix/) | Banyak operasi grafik SVG menggunakan matriks 2x3 dalam bentuk: [ace] [bdf] yang, ketika diperluas menjadi matriks 3x3 untuk tujuan aritmatika matriks, menjadi: [ace] [bdf] [0 0 1] |
| [SVGNumber](./svgnumber/) | Digunakan untuk atribut nomor tipe dasar. |
| [SVGNumberList](./svgnumberlist/) | Antarmuka ini mendefinisikan daftar objek SVGNumber. |
| [SVGPoint](./svgpoint/) | Banyak antarmuka DOM SVG mengacu pada objek kelas SVGPoint. SVGPoint adalah pasangan koordinat (x, y). Saat digunakan dalam operasi matriks, SVGPoint diperlakukan sebagai vektor dalam bentuk: [x] [y] [1] Jika objek SVGRect ditetapkan sebagai hanya baca, maka mencoba menetapkan ke salah satu atributnya akan menghasilkan pengecualian yang dilemparkan. |
| [SVGPointList](./svgpointlist/) | Antarmuka ini mendefinisikan daftar objek SVGPoint. |
| [SVGPreserveAspectRatio](./svgpreserveaspectratio/) | Antarmuka SVGPreserveAspectRatio sesuai dengan atribut 'preserveAspectRatio', yang tersedia untuk beberapa elemen SVG. |
| [SVGRect](./svgrect/) | Merupakan geometri persegi panjang. Persegi panjang didefinisikan sebagai terdiri dari pasangan koordinat (x,y) yang mengidentifikasi nilai X minimum, nilai Y minimum, serta lebar dan tinggi, yang biasanya dibatasi menjadi non-negatif. |
| [SVGStringList](./svgstringlist/) | SVGStringList memiliki atribut dan metode yang sama dengan antarmuka SVGxxxList lainnya. Pelaksana dapat mempertimbangkan untuk menggunakan satu kelas dasar untuk mengimplementasikan berbagai antarmuka SVGxxxList. |
| [SVGTransform](./svgtransform/) | SVGTransform adalah antarmuka untuk salah satu transformasi komponen dalam SVGTransformList; dengan demikian, objek SVGTransform sesuai dengan komponen tunggal (misalnya, 'skala(â€¦)' atau 'matriks(...)') dalam spesifikasi atribut 'transformasi'. |
| [SVGTransformList](./svgtransformlist/) | Antarmuka ini mendefinisikan daftar objek SVGTransform. |
| [SVGValueType](./svgvaluetype/) | Tipe SVGValueType digunakan untuk mewakili tipe nilai dasar SVG. |
