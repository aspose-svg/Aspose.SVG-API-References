## SVGAngle.ValueInSpecifiedUnits property

Nilai sudut sebagai nilai floating point, dalam satuan yang dinyatakan oleh unitType. Menyetel atribut ini akan menyebabkan value dan valueAsString diperbarui secara otomatis untuk mencerminkan setelan ini.

```csharp
public float ValueInSpecifiedUnits { get; set; }
```
