**Aspose.Svg.Dom.Views** namespace menjelaskan antarmuka AbstractView dan DocumentView menurut spesifikasi Document Object Model Views.

## Antarmuka

| Antarmuka | Keterangan |
| --- | --- |
| [IAbstractView](./iabstractview/) | Antarmuka dasar tempat semua tampilan berasal. |
| [IDocumentView](./idocumentview/) | Antarmuka DocumentView diimplementasikan oleh objek Dokumen dalam implementasi DOM yang mendukung Tampilan DOM. Ini memberikan atribut untuk mengambil tampilan default dokumen. |
