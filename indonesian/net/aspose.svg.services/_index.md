Itu **Aspose.Svg.Layanan** namespace berisi antarmuka sebagai protokol untuk implementasi layanan.

## Antarmuka

| Antarmuka | Keterangan |
| --- | --- |
| [IDeviceInformationService](./ideviceinformationservice/) | Antarmuka yang menggambarkan lingkungan tempat[`Document`](../aspose.svg.dom/document/) disajikan kepada pengguna. |
| [INetworkService](./inetworkservice/) | Menyediakan antarmuka untuk operasi jaringan. |
| [IRuntimeService](./iruntimeservice/) | Layanan ini digunakan untuk mengonfigurasi properti terkait runtime. |
| [IService](./iservice/) | Menentukan antarmuka dasar untuk implementasi layanan. |
| [IServiceContainer](./iservicecontainer/) | Menyediakan wadah untuk layanan. |
| [IServiceProvider](./iserviceprovider/) | Menyediakan wadah yang tidak dapat diubah untuk layanan. |
| [IUserAgentService](./iuseragentservice/) | Antarmuka yang menggambarkan lingkungan agen pengguna. |
