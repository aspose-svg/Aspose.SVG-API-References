## IRuntimeService.JavaScriptTimeout property

Mendapat atau setTimeSpanyang membatasi waktu eksekusi JavaScript. Jika skrip dijalankan lebih lama dari yang disediakanTimeSpan , itu akan dibatalkan. Anda dapat menentukan batas waktu tak terbatas dengan pengaturanTimeSpan sama dengan -1 milidetik. Nilai default adalah 1 menit.

```csharp
public TimeSpan JavaScriptTimeout { get; set; }
```
