Itu **Aspose.Svg.Net**namespace disajikan oleh kelas dan antarmuka yang bertanggung jawab untuk membantu pemrosesan jaringan yang mudah.

## Kelas

| Kelas | Keterangan |
| --- | --- |
| [ByteArrayContent](./bytearraycontent/) | Mewakili konten berdasarkan array byte. |
| [Content](./content/) | Mewakili kelas dasar untuk badan entitas HTTP dan header konten. |
| [ContentHeaders](./contentheaders/) | Berisi header yang terkait dengan konten. |
| [FormUrlEncodedContent](./formurlencodedcontent/) | Penampung untuk tupel nama/nilai yang dikodekan menggunakan jenis MIME application/x-www-form-urlencoded. |
| [HttpMethod](./httpmethod/) | Merupakan kelas utilitas untuk mengambil dan membandingkan metode HTTP standar. |
| [MessageFilter](./messagefilter/) | Mewakili kelas dasar abstrak untuk berbagai kelas filter yang digunakan untuk mengkueri pesan |
| [MessageHandler](./messagehandler/) | Mewakili tipe dasar untuk penangan pesan. |
| [MessageHandlerCollection](./messagehandlercollection/) | Merupakan koleksi dari[`MessageHandler`](../aspose.svg.net/messagehandler/) . |
| [MultipartContent](./multipartcontent/) | Mewakili konten multipart/*. |
| [MultipartFormDataContent](./multipartformdatacontent/) | Mewakili konten untuk algoritme penyandian data multipart/formulir |
| [RequestHeaders](./requestheaders/) | Berisi header protokol yang terkait dengan permintaan. |
| [RequestMessage](./requestmessage/) | Merupakan pesan permintaan. |
| [ResponseHeaders](./responseheaders/) | Berisi header protokol yang terkait dengan respons. |
| [ResponseMessage](./responsemessage/) | Merupakan pesan tanggapan. |
| [StreamContent](./streamcontent/) | Mewakili konten berdasarkan aliran. |
| [StringContent](./stringcontent/) | Mewakili konten berdasarkan string. |
| [UrlResolver](./urlresolver/) | Merupakan kelas utilitas untuk menyelesaikan URL absolut dengan Uniform Resource Identifier (URI). |
