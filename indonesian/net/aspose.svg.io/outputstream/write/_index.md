## OutputStream.Write method

Menulis urutan byte ke aliran keluaran yang dibungkus dan memajukan posisi saat ini dalam aliran ini dengan jumlah byte yang ditulis.

```csharp
public override void Write(byte[] buffer, int offset, int count)
```

| Parameter | Jenis | Keterangan |
| --- | --- | --- |
| buffer | Byte[] | Array byte. Metode ini menyalin jumlah byte dari buffer ke aliran output yang dibungkus. |
| offset | Int32 | Offset byte berbasis nol dalam buffer untuk mulai menyalin byte ke aliran output yang dibungkus. |
| count | Int32 | Jumlah byte yang akan ditulis ke aliran keluaran yang dibungkus. |
