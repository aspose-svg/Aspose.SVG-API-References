Selain alat standar IO (input-output)  **Aspose.Svg.IO** namespace berisi kelas dan antarmuka pembantu.

## Kelas

| Kelas | Keterangan |
| --- | --- |
| [FileCreateStreamProvider](./filecreatestreamprovider/) | MewakiliFileStream implementasi untuk menyediakan streaming ke perangkat rendering. |
| [LocalFileSystemStorage](./localfilesystemstorage/) | Penyimpanan sistem file. |
| [OutputStream](./outputstream/) | Aliran pengganti membungkus aliran keluaran nyata dan mengontrol akses ke sana. [`OutputStream`](../aspose.svg.io/outputstream/) berisi data URI yang menjelaskan lokasi aliran keluaran. |
| [OutputStreamContext](./outputstreamcontext/) | Konteks untuk inisialisasi aliran keluaran. |
## Antarmuka

| Antarmuka | Keterangan |
| --- | --- |
| [IBlob](./iblob/) | Objek Blob merujuk ke urutan byte, dan memiliki atribut ukuran yang merupakan jumlah total byte dalam urutan byte, dan atribut tipe, yang merupakan string berkode ASCII dalam huruf kecil yang mewakili jenis media dari urutan byte . |
| [ICreateStreamProvider](./icreatestreamprovider/) | Merupakan antarmuka yang dapat diimplementasikan oleh kelas yang menyediakan aliran ke perangkat rendering. |
| [IFile](./ifile/) | Objek File adalah objek Blob dengan atribut nama, yaitu string; itu dapat dibuat dalam aplikasi web melalui konstruktor, atau merupakan referensi ke urutan byte dari file dari sistem file (OS) yang mendasarinya. |
| [IFileList](./ifilelist/) | Mewakili antarmuka untuk daftar file. |
| [IOutputStorage](./ioutputstorage/) | Menyediakan pembuatan dan pengelolaan aliran keluaran[`OutputStream`](../aspose.svg.io/outputstream/) . |
