## Element.SetAttributeNS method

Menambahkan atribut baru. Jika atribut dengan nama lokal dan URI namespace yang sama sudah ada pada elemen, prefiksnya diubah menjadi bagian prefiks dari QualifiedName, dan nilainya diubah menjadi parameter value.

```csharp
public void SetAttributeNS(string namespaceURI, string qualifiedName, string value)
```

| Parameter | Jenis | Keterangan |
| --- | --- | --- |
| namespaceURI | String | URI ruang nama. |
| qualifiedName | String | Nama yang memenuhi syarat. |
| value | String | Nilai. |
