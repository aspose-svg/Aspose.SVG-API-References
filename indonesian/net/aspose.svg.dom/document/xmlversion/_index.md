## Document.XmlVersion property

Atribut yang menentukan, sebagai bagian dari deklarasi XML, nomor versi dokumen ini. Jika tidak ada deklarasi dan jika dokumen ini mendukung fitur "XML", nilainya adalah "1.0". Jika dokumen ini tidak mendukung fitur "XML", nilainya selalu null.

```csharp
public string XmlVersion { get; set; }
```

### Nilai properti

Versi XML.
