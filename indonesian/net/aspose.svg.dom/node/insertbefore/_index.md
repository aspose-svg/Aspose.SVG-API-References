## Node.InsertBefore method

Menyisipkan simpul sebelum anak simpul anak yang ada. Jika anak adalah null, sisipkan simpul di akhir daftar anak. Jika anak adalah objek DocumentFragment, semua anak disisipkan, dalam urutan yang sama, sebelum anak. Jika anak sudah ada di pohon, pertama-tama dihapus.

```csharp
public Node InsertBefore(Node node, Node child)
```

| Parameter | Jenis | Keterangan |
| --- | --- | --- |
| node | Node | Anak baru. |
| child | Node | Anak wasit. |

### Nilai Pengembalian

Mengembalikan simpul yang dimasukkan
