## IWindow.Parent property

Atribut IDL induk pada objek Window dari Dokumen dalam konteks penelusuran b harus mengembalikan objek WindowProxy dari konteks penelusuran induk, jika ada (yaitu jika b adalah konteks penelusuran anak), atau objek WindowProxy dari penelusuran konteks b itu sendiri, jika tidak (yaitu jika itu adalah konteks penelusuran tingkat atas atau konteks penelusuran bersarang terpisah).

```csharp
public IWindow Parent { get; }
```
