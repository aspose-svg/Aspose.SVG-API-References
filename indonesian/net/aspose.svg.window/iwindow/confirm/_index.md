## IWindow.Confirm method

Menampilkan prompt modal OK/Batal dengan pesan yang diberikan, menunggu pengguna menutupnya, dan mengembalikan true jika pengguna mengklik OK dan salah jika pengguna mengklik Batal.

```csharp
public bool Confirm(string message)
```

| Parameter | Jenis | Keterangan |
| --- | --- | --- |
| message | String | Pesan. |

### Nilai Pengembalian

Mengembalikan nilai true jika pengguna mengklik OK dan false jika pengguna mengklik Cancel
