## IXPathNSResolver.LookupNamespaceURI method

Cari URI namespace yang terkait dengan awalan namespace yang diberikan. Penilai XPath tidak boleh menyebut ini dengan a`batal` atau argumen kosong, karena hasil dari melakukan ini tidak terdefinisi.

```csharp
public string LookupNamespaceURI(string prefix)
```

| Parameter | Jenis | Keterangan |
| --- | --- | --- |
| prefix | String | Awalan yang harus dicari. |

### Nilai Pengembalian

Mengembalikan URI namespace terkait atau`batal` jika tidak ada ditemukan.
