## Ruang nama

| Ruang nama | Keterangan |
| --- | --- |
| [Aspose.Svg](./aspose.svg/) | Semua kelas di **Aspose.Svg.Dom.Svg**namespace didasarkan pada rekomendasi w3c SVG2. Dengan menggunakan ruang nama ini, Anda dapat memuat, menavigasi, atau merender file SVG sesuai kebutuhan Anda. |
| [Aspose.Svg.Collections](./aspose.svg.collections/) | Itu **Aspose.Svg.Collections** namespace terdiri dari kelas untuk mewakili, menyimpan dan memanipulasi node dan elemen. |
| [Aspose.Svg.Converters](./aspose.svg.converters/) | Itu **Aspose.Svg.Konverter** tujuan namespace adalah akses mudah ke metode konversi. Ini menyediakan berbagai konversi ke format populer, seperti PDF, XPS, format gambar, dll. Kasus pengguna konversi (rendering, penyimpanan) yang lebih spesifik disajikan oleh fungsi API tingkat rendah yang dikenal dan didokumentasikan dengan baik di ruang nama berorientasi subjek. |
| [Aspose.Svg.DataTypes](./aspose.svg.datatypes/) | Namespace menyediakan akses ke semua tipe data spesifik SVG. |
| [Aspose.Svg.Dom](./aspose.svg.dom/) | Itu **Aspose.Svg.Dom (Model Objek Dokumen)** namespace menyediakan API yang mewakili dan berinteraksi dengan dokumen HTML, XML, atau SVG apa pun. DOM adalah model dokumen yang dimuat di browser dan mewakili dokumen sebagai pohon simpul, di mana setiap node mewakili bagian dari dokumen (misalnya elemen, teks string, atau komentar). |
| [Aspose.Svg.Dom.Attributes](./aspose.svg.dom.attributes/) | Itu **Aspose.Svg.Dom.Attributes** namespace berisi kelas atribut terpadu yang menentukan atau memenuhi syarat fitur atau perilaku objek (simpul, elemen, ...) apa pun. |
| [Aspose.Svg.Dom.Css](./aspose.svg.dom.css/) | Itu **Aspose.Svg.Dom.Css** namespace adalah untuk semua manipulasi terkait CSS. Berkonsentrasi di sekitar nama properti CSS - pasangan nilai ditentukan oleh dokumen resmi CSS. |
| [Aspose.Svg.Dom.Events](./aspose.svg.dom.events/) | Itu **Aspose.Svg.Dom.Events** namespace menyediakan objek untuk acara apa pun terkait pembaruan DOM. Ini mencakup langganan ke pengamatan informasi kontekstual spesifik terkait dengan acara serta konstruksi acara khusus. |
| [Aspose.Svg.Dom.Mutations](./aspose.svg.dom.mutations/) | Itu **Aspose.Svg.Dom.Mutasi** memberikan kesempatan untuk mengamati mutasi DOM dan mendapatkan informasi callback tertentu. |
| [Aspose.Svg.Dom.Traversal](./aspose.svg.dom.traversal/) | Itu **Aspose.Svg.Dom.Traversal**namespace berisi metode yang membuat iterator dan tree-walker untuk bernavigasi di antara elemen dan melintasi node dan turunannya dalam urutan dokumen. |
| [Aspose.Svg.Dom.Traversal.Filters](./aspose.svg.dom.traversal.filters/) | Namespace yang diwakili oleh fungsi pemfilteran node. Filter adalah objek yang mengetahui cara "menyaring" node. Jika iterator node atau tree walker diberi filter node, ia menerapkan filter sebelum mengembalikan node berikutnya. Jika filter mengatakan untuk menerima node, traversal logic mengembalikannya; jika tidak, traversal mencari node berikutnya dan berpura-pura bahwa node yang ditolak tidak ada. |
| [Aspose.Svg.Dom.Views](./aspose.svg.dom.views/) | **Aspose.Svg.Dom.Views** namespace menjelaskan antarmuka AbstractView dan DocumentView menurut spesifikasi Document Object Model Views. |
| [Aspose.Svg.Dom.XPath](./aspose.svg.dom.xpath/) | Ruang nama berisi metode untuk menavigasi elemen dan atribut dalam dokumen XML. |
| [Aspose.Svg.Drawing](./aspose.svg.drawing/) | Itu **Aspose.Svg.Drawing** namespace berisi objek dan antarmuka untuk menentukan ukuran dan satuan serta menggambar objek atribut seperti kuas, warna, dan font. |
| [Aspose.Svg.Events](./aspose.svg.events/) | Ruang nama terdiri dari kelas untuk interaktivitas berbasis peristiwa seperti peristiwa waktu dan pembesaran. |
| [Aspose.Svg.Filters](./aspose.svg.filters/) | Itu **Aspose.Svg.Dom.Svg.Filter** namespace berisi kelas dan antarmuka yang terkait dengan efek filter dalam spesifikasi SVG. |
| [Aspose.Svg.ImageVectorization](./aspose.svg.imagevectorization/) | Itu **Aspose.Svg.ImageVectorization**namespace berisi kelas untuk membuat vektor gambar raster dan mengonversinya menjadi dokumen SVG. Proses ini meliputi pengurangan bitmap menjadi bentuk geometris yang dibuat dari elemen jalur dan menyimpannya sebagai SVG. Ruang nama mencakup kelas untuk membangun segmen jalur, menyederhanakan dan menghaluskan titik jejak, dan mengonfigurasi opsi vektorisasi. |
| [Aspose.Svg.IO](./aspose.svg.io/) | Selain alat standar IO (input-output)  **Aspose.Svg.IO** namespace berisi kelas dan antarmuka pembantu. |
| [Aspose.Svg.Net](./aspose.svg.net/) | Itu **Aspose.Svg.Net**namespace disajikan oleh kelas dan antarmuka yang bertanggung jawab untuk membantu pemrosesan jaringan yang mudah. |
| [Aspose.Svg.Net.Headers](./aspose.svg.net.headers/) | Itu **Aspose.Svg.Net.Headers** menyediakan beberapa kelas data untuk mendeskripsikan konten header di jaringan. |
| [Aspose.Svg.Net.MessageFilters](./aspose.svg.net.messagefilters/) | Itu **Aspose.Svg.Net.MessageFilters** namespace menyediakan kelas yang mengimplementasikan abstraksi filter pesan. |
| [Aspose.Svg.Net.MessageHandlers](./aspose.svg.net.messagehandlers/) | Itu **Aspose.Svg.Net.MessageHandlers** namespace is disajikan oleh kelas - penangan pesan dari protokol yang berbeda. |
| [Aspose.Svg.Paths](./aspose.svg.paths/) | Itu **Aspose.Svg.Dom.Svg.Paths** namespace mendeskripsikan kelas untuk representasi garis bentuk yang dapat diisi atau digores. Jalur juga dapat digunakan sebagai jalur kliping, untuk mendeskripsikan animasi, atau teks posisi. |
| [Aspose.Svg.Rendering](./aspose.svg.rendering/) | Itu **Aspose.Svg.Rendering**namespace terdiri dari banyak objek penyaji serta kelas opsi tingkat rendah yang sesuai yang bertanggung jawab untuk merender dokumen/file ke implementasi IDevice. |
| [Aspose.Svg.Rendering.Fonts](./aspose.svg.rendering.fonts/) | Itu **Aspose.Svg.Rendering.Fonts** namespace berisi kelas dan metode yang memungkinkan Anda mengontrol beberapa bagian dari algoritme pencocokan font. |
| [Aspose.Svg.Rendering.Image](./aspose.svg.rendering.image/) | Itu **Aspose.Svg.Rendering.Image** namespace menyediakan kelas perangkat tertentu serta beberapa opsi rendering kelas yang bertanggung jawab untuk merender ke format raster: jpeg, png, bmp, gif, tiff. |
| [Aspose.Svg.Rendering.Pdf](./aspose.svg.rendering.pdf/) | Itu **Aspose.Svg.Rendering.Pdf**namespace menyediakan kelas perangkat tertentu serta beberapa opsi rendering class bertanggung jawab untuk merender ke dokumen pdf. |
| [Aspose.Svg.Rendering.Pdf.Encryption](./aspose.svg.rendering.pdf.encryption/) | Itu **Aspose.Svg.Rendering.Pdf.Enkripsi** namespace berisi kelas data yang menjelaskan detail enkripsi pdf seperti sandi, algoritme enkripsi, dan sebagainya. |
| [Aspose.Svg.Rendering.Xps](./aspose.svg.rendering.xps/) | Itu **Aspose.Svg.Rendering.Xps** namespace menyediakan kelas perangkat tertentu serta beberapa kelas opsi rendering yang bertanggung jawab untuk merender ke dokumen xps. |
| [Aspose.Svg.Saving](./aspose.svg.saving/) | Itu **Aspose.Svg.Menyimpan** namespace disajikan oleh kelas data untuk deskripsi opsi penyimpanan tertentu di proses konversi/penyimpanan. |
| [Aspose.Svg.Services](./aspose.svg.services/) | Itu **Aspose.Svg.Layanan** namespace berisi antarmuka sebagai protokol untuk implementasi layanan. |
| [Aspose.Svg.Toolkit.Optimizers](./aspose.svg.toolkit.optimizers/) | Itu **Aspose.Svg.Toolkit.Optimizers** namespace berisi kelas untuk mengoptimalkan dokumen SVG. Proses pengoptimalan melibatkan penghapusan elemen yang tidak terpakai atau tidak terlihat beserta atributnya, menggabungkan grup, dan mengurangi ukuran segmen jalur. |
| [Aspose.Svg.Window](./aspose.svg.window/) | Itu **Aspose.Svg.Jendela** namespace terkait untuk objek jendela yang mewakili window yang berisi dokumen DOM. Ini termasuk kelas dan antarmuka terkait dengan alamat dokumen aktif dan riwayat penelusuran. |
