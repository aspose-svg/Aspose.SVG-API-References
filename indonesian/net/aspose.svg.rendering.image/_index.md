Itu **Aspose.Svg.Rendering.Image** namespace menyediakan kelas perangkat tertentu serta beberapa opsi rendering kelas yang bertanggung jawab untuk merender ke format raster: jpeg, png, bmp, gif, tiff.

## Kelas

| Kelas | Keterangan |
| --- | --- |
| [ImageDevice](./imagedevice/) | Mewakili rendering ke format raster: jpeg, png, bmp, gif, tiff. |
| [ImageRenderingOptions](./imagerenderingoptions/) | Mewakili opsi rendering untuk[`ImageDevice`](../aspose.svg.rendering.image/imagedevice/) . Opsi ini digunakan untuk menentukan format gambar keluaran, kompresi, resolusi dll. |
| [TextOptions](./textoptions/) | Mewakili opsi rendering teks untuk[`ImageDevice`](../aspose.svg.rendering.image/imagedevice/) . |
## Pencacahan

| Pencacahan | Keterangan |
| --- | --- |
| [Compression](./compression/) | Menentukan kemungkinan skema kompresi untuk gambar bitmap Tagged Image File Format (TIFF). |
| [ImageFormat](./imageformat/) | Menentukan format file gambar. |
