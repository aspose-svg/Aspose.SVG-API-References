## IUrlSearchParams.Set method

Menetapkan nilai dari pasangan nama-nilai yang ditemukan pertama ke nilai yang ditentukan dan menghapus yang lain. Jika tidak ada pasangan nama-nilai dengan nama yang ditentukan ditemukan, yang baru akan ditambahkan ke daftar.

```csharp
public void Set(string name, string value)
```

| Parameter | Jenis | Keterangan |
| --- | --- | --- |
| name | String | Kunci yang akan digunakan untuk mencari pasangan nama-nilai. |
| value | String | Nilai yang akan ditetapkan. |
