## SVGSVGElement.GetCurrentTime method

Mengembalikan waktu saat ini dalam detik relatif terhadap waktu mulai fragmen dokumen SVG saat ini. Jika getCurrentTime dipanggil sebelum garis waktu dokumen dimulai (misalnya, dengan skrip berjalan di elemen 'skrip' sebelum acara SVGLoad dokumen dikirim), maka 0 dikembalikan.

```csharp
public float GetCurrentTime()
```

### Nilai Pengembalian

Waktu saat ini dalam detik, atau 0 jika garis waktu dokumen belum dimulai.
