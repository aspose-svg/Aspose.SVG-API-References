## SVGSVGElement.GetElementById method

Mencari fragmen dokumen SVG ini (yakni, pencarian dibatasi pada subset pohon dokumen) untuk Elemen yang id-nya diberikan oleh elementId. Jika Elemen ditemukan, Elemen tersebut dikembalikan. Jika tidak ada elemen seperti itu, kembalikan nol. Perilaku tidak ditentukan jika lebih dari satu elemen memiliki id ini.

```csharp
public Element GetElementById(string elementId)
```

| Parameter | Jenis | Keterangan |
| --- | --- | --- |
| elementId | String | Nilai id unik untuk suatu elemen. |
