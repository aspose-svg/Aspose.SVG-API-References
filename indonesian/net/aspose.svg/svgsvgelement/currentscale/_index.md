## SVGSVGElement.CurrentScale property

Pada elemen svg terluar, atribut ini menunjukkan faktor skala saat ini relatif terhadap tampilan awal untuk mempertimbangkan pembesaran pengguna dan operasi panning, seperti yang dijelaskan di bawah Pembesaran dan panning. Atribut DOM currentScale dan currentTranslate setara dengan matriks 2x3 [abcdef] = [currentScale 0 0 currentScale currentTranslate.x currentTranslate.y]. Jika "pembesaran" diaktifkan (yaitu, zoomAndPan="memperbesar"), maka efeknya adalah seolah-olah transformasi ekstra ditempatkan pada tingkat terluar pada fragmen dokumen SVG (yaitu, di luar elemen svg terluar). Saat diakses di elemen 'svg' yang bukan elemen svg terluar, tidak ditentukan perilaku apa yang dimiliki atribut ini.

```csharp
public float CurrentScale { get; set; }
```

Skala saat ini.
