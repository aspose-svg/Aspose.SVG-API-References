## Metered.SetMeteredKey method

Mengatur kunci publik dan pribadi terukur. Jika Anda membeli lisensi terukur, saat memulai aplikasi, API ini harus dipanggil, biasanya, ini sudah cukup. Namun, jika selalu gagal mengunggah data konsumsi dan melebihi 24 jam, lisensi akan disetel ke status evaluasi, untuk menghindari kasus tersebut, Anda harus memeriksa status lisensi secara berkala, jika status evaluasi, hubungi API ini lagi.

```csharp
public void SetMeteredKey(string publicKey, string privateKey)
```

| Parameter | Jenis | Keterangan |
| --- | --- | --- |
| publicKey | String | kunci publik |
| privateKey | String | kunci pribadi |
