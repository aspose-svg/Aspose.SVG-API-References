## Save(Url) {#save_3}

Menyimpan dokumen ke file lokal yang ditentukan oleh`url` . Semua sumber daya yang digunakan dalam dokumen ini akan disimpan dalam ke folder yang berdekatan, yang namanya akan dibuat sebagai: nama_file_output + "_files".

```csharp
public void Save(Url url)
```

| Parameter | Jenis | Keterangan |
| --- | --- | --- |
| url | Url | URL lokal ke file keluaran. |

### Pengecualian

| ArgumentException | Dibesarkan jika ditentukan`url` bukan URL file lokal yang valid. |
