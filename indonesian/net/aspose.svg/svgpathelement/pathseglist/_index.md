## SVGPathElement.PathSegList property

Menyediakan akses ke konten dasar (yaitu, statis) dari atribut 'd' dalam bentuk yang cocok satu-untuk-satu dengan sintaks SVG. Jadi, jika atribut 'd' memiliki perintah "absolute moveto (M)" dan "absolute arcto (A)", maka pathSegList akan memiliki dua entri: SVG_PATHSEG_MOVETO_ABS dan SVG_PATHSEG_ARC_ABS.

```csharp
public SVGPathSegList PathSegList { get; set; }
```

### Nilai properti

Daftar segmen jalur.
