## IStyleSheet.ParentStyleSheet property

Untuk bahasa style sheet yang mendukung konsep penyertaan style sheet, atribut ini mewakili style sheet yang disertakan, jika ada. Jika style sheet adalah style sheet tingkat atas, atau bahasa style sheet tidak mendukung penyertaan, nilai atribut ini adalah null.

```csharp
public IStyleSheet ParentStyleSheet { get; }
```
