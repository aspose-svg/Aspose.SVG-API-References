## ICSSKeyframesRule.FindRule method

Metode findRule mengembalikan aturan dengan kunci yang cocok dengan kunci yang diteruskan. Jika tidak ada aturan seperti itu, nilai nol dikembalikan

```csharp
public ICSSKeyframeRule FindRule(string key)
```

| Parameter | Jenis | Keterangan |
| --- | --- | --- |
| key | String | Kunci yang menggambarkan aturan untuk ditemukan. Kunci harus diselesaikan ke angka antara 0 dan 1, atau aturan akan diabaikan. |
