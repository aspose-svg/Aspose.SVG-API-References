## ICSSStyleDeclaration.GetPropertyCSSValue method

Digunakan untuk mengambil representasi objek dari nilai properti CSS jika telah ditetapkan secara eksplisit dalam blok deklarasi ini. Metode ini mengembalikan nol jika properti adalah properti steno. Nilai properti singkatan hanya dapat diakses dan dimodifikasi sebagai string, menggunakan metode getPropertyValue dan setProperty.

```csharp
public CSSValue GetPropertyCSSValue(string propertyName)
```

| Parameter | Jenis | Keterangan |
| --- | --- | --- |
| propertyName | String | Nama properti. |

### Nilai Pengembalian

Mengembalikan nilai properti
