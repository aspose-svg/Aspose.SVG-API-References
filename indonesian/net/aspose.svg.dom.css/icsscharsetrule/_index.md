## ICSSCharsetRule interface

Antarmuka CSSCharsetRule mewakili aturan @charset dalam lembar gaya CSS. Nilai atribut penyandian tidak memengaruhi penyandian data teks dalam objek DOM; penyandian ini selalu UTF-16. Setelah lembar gaya dimuat, nilai atribut penyandian adalah nilai yang ditemukan dalam aturan @charset. Jika tidak ada @charset di dokumen asli, maka tidak ada CSSCharsetRule yang dibuat. Nilai atribut encoding juga dapat digunakan sebagai petunjuk untuk encoding yang digunakan pada serialisasi style sheet.

```csharp
public interface ICSSCharsetRule : ICSSRule
```
