## IDocumentStyle interface

Antarmuka DocumentStyle menyediakan mekanisme dimana style sheet tertanam dalam dokumen dapat diambil. Harapannya adalah bahwa instance antarmuka DocumentStyle dapat diperoleh dengan menggunakan metode pengecoran khusus yang mengikat pada instance antarmuka Dokumen.

```csharp
public interface IDocumentStyle
```
