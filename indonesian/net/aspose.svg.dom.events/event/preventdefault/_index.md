## Event.PreventDefault method

Jika suatu acara dapat dibatalkan, maka`PreventDefault` metode digunakan untuk menandakan bahwa acara tersebut akan dibatalkan, artinya tindakan default apa pun yang biasanya diambil oleh implementasi sebagai akibat dari acara tersebut tidak akan terjadi.

```csharp
public void PreventDefault()
```

### Perkataan

Jika, selama tahap aliran peristiwa apa pun,`PreventDefault`metode disebut acara dibatalkan. Tindakan default apa pun yang terkait dengan acara tidak akan terjadi. Memanggil metode ini untuk acara yang tidak dapat dibatalkan tidak berpengaruh. Sekali`PreventDefault` telah dipanggil, itu akan tetap berlaku selama sisa propagasi peristiwa. Metode ini dapat digunakan selama tahap aliran peristiwa apa pun.
