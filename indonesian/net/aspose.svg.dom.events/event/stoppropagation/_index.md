## Event.StopPropagation method

Itu`StopPropagation` metode digunakan untuk mencegah penyebaran lebih lanjut dari suatu peristiwa selama aliran peristiwa.

```csharp
public void StopPropagation()
```

### Perkataan

Jika metode ini dipanggil oleh siapa saja[`IEventListener`](../../ieventlistener/) acara akan berhenti menyebar melalui pohon. Acara akan menyelesaikan pengiriman ke semua pendengar saat ini[`IEventTarget`](../../ieventtarget/) sebelum aliran peristiwa berhenti. Metode ini dapat digunakan selama setiap tahapan aliran peristiwa.
