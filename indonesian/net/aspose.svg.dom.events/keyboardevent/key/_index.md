## KeyboardEvent.Key property

Kunci menyimpan nilai kunci dari tombol yang ditekan. Jika nilainya memiliki representasi tercetak, itu HARUS berupa string karakter Unicode yang tidak kosong, sesuai dengan algoritme untuk menentukan nilai kunci yang ditentukan dalam spesifikasi ini. Jika nilainya adalah kunci kontrol yang tidak memiliki representasi tercetak, itu HARUS menjadi salah satu nilai kunci yang ditentukan dalam rangkaian nilai kunci, seperti yang ditentukan oleh algoritme untuk menentukan nilai kunci. Implementasi yang tidak dapat mengidentifikasi kunci HARUS menggunakan nilai kunci Unidentified.

```csharp
public string Key { get; }
```
