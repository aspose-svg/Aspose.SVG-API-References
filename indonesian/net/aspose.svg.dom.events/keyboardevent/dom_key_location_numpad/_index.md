## KeyboardEvent.DOM_KEY_LOCATION_NUMPAD field

Aktivasi tombol berasal dari keypad numerik atau dengan tombol virtual yang sesuai dengan keypad numerik (bila ada lebih dari satu kemungkinan lokasi untuk tombol ini). Perhatikan bahwa kunci NumLock harus selalu dikodekan dengan lokasi DOM_KEY_LOCATION_STANDARD.

```csharp
public const ulong DOM_KEY_LOCATION_NUMPAD;
```
