## KeyboardEvent.Repeat property

benar jika tombol ditekan terus menerus. Menahan tombol HARUS menghasilkan pengulangan peristiwa keydown, sebelum memasukkan, memasukkan dalam urutan ini, dengan kecepatan yang ditentukan oleh konfigurasi sistem. Untuk perangkat seluler yang memiliki perilaku penekanan tombol lama, peristiwa tombol pertama dengan nilai atribut pengulangan true HARUS berfungsi sebagai indikasi penekanan tombol lama. Lamanya waktu tombol HARUS ditekan untuk memulai pengulangan bergantung pada konfigurasi.

public bool Repeat { get; }
