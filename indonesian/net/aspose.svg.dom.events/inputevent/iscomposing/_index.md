## InputEvent.IsComposing property

true jika peristiwa input terjadi sebagai bagian dari sesi komposisi, yaitu setelah peristiwa komposisi mulai dan sebelum peristiwa komposisi akhir yang sesuai. Nilai yang tidak diinisialisasi dari atribut ini HARUS false.

```csharp
public bool IsComposing { get; }
```

### Nilai properti

`BENAR` jika instance ini sedang menulis; jika tidak,`PALSU` .
