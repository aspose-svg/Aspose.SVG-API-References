## InputEvent.Data property

Data menyimpan nilai karakter yang dihasilkan oleh metode input. Ini MUNGKIN berupa karakter Unicode tunggal atau rangkaian karakter Unicode yang tidak kosong [Unicode]. Karakter HARUS dinormalisasi seperti yang didefinisikan oleh bentuk normalisasi Unicode NFC, yang didefinisikan di [UAX15]. Atribut ini MUNGKIN berisi string kosong.

```csharp
public string Data { get; }
```
