Itu **Aspose.Svg.Drawing** namespace berisi objek dan antarmuka untuk menentukan ukuran dan satuan serta menggambar objek atribut seperti kuas, warna, dan font.

## Kelas

| Kelas | Keterangan |
| --- | --- |
| [Angle](./angle/) | Tipe data sudut |
| [Color](./color/) | Kelas Color memungkinkan Anda menentukan warna sebagai nilai Red-Green-Blue (RGB), nilai Hue-Saturation-Luminosity (HSL), nilai Hue-Saturation-Value (HSV), Hue-Whiteness-Blackness (HWB ) nilai, nilai lightness-AB (LAB), nilai Luminance-Chroma-Hue (LCH), nilai Cyan-Magenta-Yellow-Key (CMYK), Nilai warna natural (NCOL), atau dengan nama warna . Saluran Alfa juga tersedia untuk menunjukkan transparansi. |
| [Dimension](./dimension/) | Menyediakan kelas dasar untuk dimensi. Istilah umum 'dimensi' mengacu pada angka dengan satuan yang melekat padanya, dan dilambangkan dengan[`UnitType`](../aspose.svg.drawing/unittype/) . |
| [Frequency](./frequency/) | Satuan 'frekuensi'. |
| [Length](./length/) | Merupakan satuan ukuran panjang. |
| [LengthOrAuto](./lengthorauto/) | Merupakan wadah untuk panjang penyimpanan atau unit 'otomatis'.. |
| [Margin](./margin/) | Mewakili margin halaman. |
| [Numeric](./numeric/) | Menyediakan kelas dasar untuk tipe numerik. |
| [Page](./page/) | Merupakan objek halaman yang digunakan untuk halaman keluaran konfigurasi. Nilai kekosongan untuk ukuran halaman adalah A4(210x297mm) |
| [Resolution](./resolution/) | Merupakan unit resolusi. |
| [Size](./size/) | Menyimpan nilai yang menentukan Tinggi dan Lebar. |
| [Time](./time/) | Merupakan satuan waktu. |
| [Unit](./unit/) | Menyediakan kelas dasar untuk satuan pengukuran. |
| [UnitType](./unittype/) | Menentukan satuan pengukuran. |
## Antarmuka

| Antarmuka | Keterangan |
| --- | --- |
| [IBrush](./ibrush/) | Mendeklarasikan metode untuk mendapatkan jenis kuas. |
| [IColorComponents](./icolorcomponents/) | Mendeklarasikan metode dan properti untuk memproses komponen warna. |
| [IGradientBrush](./igradientbrush/) | Deklarasikan metode untuk mendapatkan properti umum kuas gradien. |
| [IInterpolationColor](./iinterpolationcolor/) | Mendeskripsikan metode untuk mendapatkan warna interpolasi. |
| [ILinearGradientBrush](./ilineargradientbrush/) | Menentukan antarmuka kuas dengan gradien linier. |
| [ISolidBrush](./isolidbrush/) | Mendefinisikan antarmuka kuas dari satu warna |
| [ITextureBrush](./itexturebrush/) | Menentukan antarmuka kuas yang menggunakan gambar untuk mengisi bagian dalam bentuk. |
| [ITransformableBrush](./itransformablebrush/) | Mendeskripsikan metode untuk mendapatkan matriks transformasi dan mode bungkus. |
| [ITrueTypeFont](./itruetypefont/) | Mendeklarasikan metode untuk bekerja dengan font TrueType. |
