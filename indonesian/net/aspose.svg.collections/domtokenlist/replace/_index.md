## DOMTokenList.Replace method

Mengganti token yang ada dengan token baru. Tidak melakukan apa-apa jika token pertama tidak ada.

```csharp
public bool Replace(string token, string newToken)
```

| Parameter | Jenis | Keterangan |
| --- | --- | --- |
| token | String | Token yang ingin Anda ganti. |
| newToken | String | Token yang ingin Anda ganti dengan token lama. |

### Nilai Pengembalian

Boolean`BENAR` jika token ditemukan dan diganti,`PALSU` jika tidak.
