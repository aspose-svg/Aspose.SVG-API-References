## Toggle(string, bool?) {#toggle_1}

Menghapus token dari daftar jika ada, atau menambahkan token ke daftar jika tidak ada.

```csharp
public bool Toggle(string token, bool? force)
```

| Parameter | Jenis | Keterangan |
| --- | --- | --- |
| token | String | Token yang ingin Anda alihkan. |
| force | Nullable`1 | Boolean yang, jika disertakan, mengubah toggle menjadi operasi satu arah saja. Jika disetel ke false, maka token hanya akan dihapus, tetapi tidak ditambahkan. Jika disetel ke true, maka token hanya akan ditambahkan, tetapi tidak dihapus. |

### Nilai Pengembalian

Boolean yang menunjukkan apakah token ada dalam daftar setelah panggilan.
