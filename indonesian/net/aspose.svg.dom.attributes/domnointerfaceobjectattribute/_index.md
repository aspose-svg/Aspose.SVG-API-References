## DOMNoInterfaceObjectAttribute class

Jika atribut yang diperluas [NoInterfaceObject] muncul pada antarmuka, ini menunjukkan bahwa objek antarmuka tidak akan ada untuk antarmuka dalam pengikatan ECMAScript.

public sealed class DOMNoInterfaceObjectAttribute : Attribute
